Template-Driven Validators Using Directives

In template-driven forms, validators are added directly in HTML using directives like required, minlength, maxlength, and pattern.

🔹 HTML

<form #userForm="ngForm">

  <input
    type="text"
    name="username"
    ngModel
    required
    minlength="3"
    maxlength="10"
    pattern="^[A-Za-z]+$"
  >

</form>

Here all validators are applied using HTML directives.
