Multiple Validators on Single Control

Angular allows multiple validators on one form control using an array.

🔹 TypeScript 

username: new FormControl('', [

  Validators.required,
  Validators.minLength(3),
  Validators.maxLength(10),
  Validators.pattern('^[A-Z]+$')

])

🔹 Validation Rules

Validator Purpose
required Field cannot be empty
minLength(3) Minimum 3 characters
maxLength(10) Maximum 10 characters
pattern() Only uppercase letters allowed
