Add Validators Dynamically in Angular

You can add validators after form creation using setValidators().

🔹 TypeScript

userForm = new FormGroup({
  username: new FormControl('')
});

addValidation() {

  this.userForm.get('username')?.setValidators([
    Validators.required,
    Validators.minLength(3)
  ]);

  this.userForm.get('username')?.updateValueAndValidity();

}

updateValueAndValidity() is required to re-check validation immediately.
