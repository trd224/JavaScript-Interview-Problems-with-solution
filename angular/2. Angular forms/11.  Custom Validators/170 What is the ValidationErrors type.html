What is the ValidationErrors type?

ValidationErrors is an Angular type used to represent validation errors.

It is basically an object containing validation error information.

Type Definition


export type ValidationErrors = {

  [key: string]: any;

};

Example Custom Validator


function noSpaceValidator(
  control: AbstractControl
): ValidationErrors | null {

  if (control.value?.includes(' ')) {

    return {

      noSpace: true

    };

  }

  return null;

}

Meaning

If validation fails: 


return {

  noSpace: true

};

If validation passes: 


return null;

Reactive Form Example


form = this.fb.group({

  username: [

    '',

    [

      noSpaceValidator

    ]

  ]

});

HTML View


<input
  type="text"
  formControlName="username"
>

<div
  class="error"
  *ngIf="form.get('username')?.errors?.['noSpace']"
>

  Spaces are not allowed

</div>

Output Example

Spaces are not allowed

Main Point

  • ValidationErrors stores validation error data
  • Validators return ValidationErrors | null
  • Error object keys are used in the template
Angular checks the returned error object to determine whether the control is valid or invalid.
