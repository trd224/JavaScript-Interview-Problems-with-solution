How do you implement a 'strong password' validator?

A strong password validator checks whether the password contains:

  • Uppercase letter
  • Lowercase letter
  • Number
  • Special character
  • Minimum length

Custom Strong Password Validator


function strongPasswordValidator(
  control: AbstractControl
): ValidationErrors | null {

  const value = control.value;

  const pattern =
  /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&]).{8,}$/;

  if (!pattern.test(value)) {

    return {

      strongPassword: true

    };

  }

  return null;

}

Reactive Form


form = this.fb.group({

  password: [

    '',

    [

      Validators.required,

      strongPasswordValidator

    ]

  ]

});

HTML View


<form [formGroup]="form">

  <input
    type="password"
    formControlName="password"
    placeholder="Enter password"
  >

  <div
    class="error"
    *ngIf="form.get('password')?.errors?.['strongPassword']"
  >

    Password is not strong enough

  </div>

</form>

Output Example

Password is not strong enough

Regex Meaning


/^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&]).{8,}$/
  • (?=.*[a-z]) → lowercase letter
  • (?=.*[A-Z]) → uppercase letter
  • (?=.*\d) → number
  • (?=.*[@$!%*?&]) → special character
  • .{8,} → minimum 8 characters
Validators return ValidationErrors | null.
