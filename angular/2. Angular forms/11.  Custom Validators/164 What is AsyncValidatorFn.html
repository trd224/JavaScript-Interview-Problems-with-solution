What is AsyncValidatorFn in Angular?

AsyncValidatorFn in Angular is a function type used to create asynchronous validators.

Async validators are mainly used for:
  • API validation
  • Database checks
  • Server-side validation
  • Username/email availability checking

Syntax 


AsyncValidatorFn

Actual Type Definition 


(control: AbstractControl)
=> Promise<ValidationErrors | null>
| Observable<ValidationErrors | null>

Meaning

Part Meaning
AbstractControl Current form control
null Valid
ValidationErrors Invalid
Observable / Promise Async operation result

Real Example

Suppose we want to check whether username already exists in database.

User Service 


import { Injectable } from '@angular/core';

import {
  Observable,
  of,
  delay
} from 'rxjs';

@Injectable({
  providedIn: 'root'
})

export class UserService {

  checkUsername(username: string): Observable<boolean> {

    const takenUsers = ['admin', 'rizwan', 'john'];

    const isAvailable = !takenUsers.includes(username);

    return of(isAvailable).pipe(delay(2000));

  }

}

Service Logic

Username Result Meaning
admin false Already taken
rizwan false Already taken
sahil true Available

AsyncValidatorFn Example 


import {
  AbstractControl,
  AsyncValidatorFn,
  ValidationErrors
} from '@angular/forms';

import {
  Observable,
  map
} from 'rxjs';

usernameValidator(): AsyncValidatorFn {

  return (

    control: AbstractControl

  ): Observable<ValidationErrors | null> => {

    return this.userService
      .checkUsername(control.value)
      .pipe(

        map(isAvailable => {

          if (isAvailable) {
            return null;
          }

          return { usernameTaken: true };

        })

      );

  };

}

Step-by-Step Understanding

Step 1 → Angular Calls Validator 


validator(control)

Example: 


control.value = 'admin'

Step 2 → Service Checks Username 


checkUsername('admin')

Step 3 → Service Returns Result 


false

Meaning: 


Username already exists

Step 4 → Validator Returns Error 


{ usernameTaken: true }

Angular marks field invalid.

If Username is Available 


checkUsername('sahil')

Service returns: 


true

Validator returns: 


null

Meaning: 


No validation error

Using AsyncValidatorFn in Form 


this.form = this.fb.group({

  username: [
    '',
    [],
    [this.usernameValidator()]
  ]

});

Syntax Meaning

Part Meaning
'' Initial value
[] Sync validators
[this.usernameValidator()] Async validators

Difference Between ValidatorFn and AsyncValidatorFn

Type Used For Return Type
ValidatorFn Synchronous validation ValidationErrors | null
AsyncValidatorFn Asynchronous validation Observable or Promise

Sync Validator Example 


Validators.required

Runs instantly.

Async Validator Example 


Check username availability from server

Takes time because API request happens.

Short Definition

AsyncValidatorFn is a function type in Angular used for asynchronous form validation that returns an Observable or Promise containing validation result.
