How do you combine custom validators with built-in validators?

You combine custom validators and built-in validators by placing them together inside the same validators array.

Custom Validator


function noSpaceValidator(control: AbstractControl) {

  if (control.value?.includes(' ')) {

    return { noSpace: true };

  }

  return null;

}

Reactive Form


form = this.fb.group({

  username: [

    '',

    [

      Validators.required,

      Validators.minLength(5),

      noSpaceValidator

    ]

  ]

});

HTML View


<form [formGroup]="form">

  <input
    type="text"
    formControlName="username"
    placeholder="Enter username"
  >

  <div
    class="error"
    *ngIf="form.get('username')?.errors?.['required']"
  >

    Username is required

  </div>

  <div
    class="error"
    *ngIf="form.get('username')?.errors?.['minlength']"
  >

    Minimum 5 characters required

  </div>

  <div
    class="error"
    *ngIf="form.get('username')?.errors?.['noSpace']"
  >

    Spaces are not allowed

  </div>

</form>

Output Example

Minimum 5 characters required
Spaces are not allowed

Meaning


[
  Validators.required,
  Validators.minLength(5),
  noSpaceValidator
]

Here:

  • Validators.required → built-in validator
  • Validators.minLength(5) → built-in validator
  • noSpaceValidator → custom validator

One form control can use multiple validators together.
