How do you create a cross-field validator (e.g., password match)?

A cross-field validator is a custom validator applied on FormGroup, because it compares multiple form controls together.

Common example:

  • Password
  • Confirm Password

Custom Cross-Field Validator


passwordMatchValidator(form: AbstractControl) {

  const password = form.get('password')?.value;

  const confirmPassword = form.get('confirmPassword')?.value;

  if (password !== confirmPassword) {

    return { passwordMismatch: true };

  }

  return null;

}

Use Validator in Reactive Form


form = this.fb.group(

  {

    password: ['', Validators.required],

    confirmPassword: ['', Validators.required]

  },

  {

    validators: this.passwordMatchValidator

  }

);

Show Validation Error


<div *ngIf="form.errors?.['passwordMismatch']">

  Password and Confirm Password do not match

</div>

How It Works

The validator:

  • Reads password value
  • Reads confirmPassword value
  • Compares both values
  • Returns validation error if values do not match

Main Point


validators: this.passwordMatchValidator
Cross-field validators are added at the FormGroup level, not individual FormControl level.
