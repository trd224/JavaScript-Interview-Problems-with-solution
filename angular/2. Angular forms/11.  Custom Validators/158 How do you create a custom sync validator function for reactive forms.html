How do you create a custom sync validator function for reactive forms?

A custom sync validator in Angular Reactive Forms is a normal function that validates a form control immediately and returns:

  • null → when the value is valid
  • ValidationErrors object → when validation fails
Sync validators run instantly whenever the form value changes.

🔹 Step 1: Create Custom Sync Validator Function


import { AbstractControl, ValidationErrors } from '@angular/forms';

export function noAdminValidator(
  control: AbstractControl
): ValidationErrors | null {

  if (control.value === 'admin') {
    return { noAdmin: true };
  }

  return null;
}

🔹 Step 2: Use Validator in Reactive Form


import { FormGroup, FormControl } from '@angular/forms';

this.userForm = new FormGroup({

  username: new FormControl(
    '',
    [noAdminValidator]
  )

});

🔹 Step 3: HTML Template


<form [formGroup]="userForm">

  <input
    type="text"
    formControlName="username"
    placeholder="Enter username"
  >

  <div *ngIf="userForm.get('username')?.hasError('noAdmin')">
    Username cannot be admin
  </div>

</form>

🔹 How it Works

  • If user enters admin, validation fails.
  • Validator returns:

{ noAdmin: true }
  • If value is valid, validator returns:

null

🔹 Summary

  • Custom sync validators are normal TypeScript functions.
  • They validate values immediately.
  • Return null when valid.
  • Return ValidationErrors object when invalid.
  • Used inside Reactive Forms validators array.
