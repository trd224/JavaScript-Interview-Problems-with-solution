How do you show error messages from a custom validator in the template?

In Angular, custom validator errors are shown using:

  • control.errors
  • control.hasError()
  • *ngIf

Reactive Form Example

This example checks whether the password is strong.

HTML Template


<form [formGroup]="form">

  <!-- Password Input -->
  <input
    type="password"
    formControlName="password"
    placeholder="Enter Password"
  >

  <!-- Error Message -->
  <div
    *ngIf="
      form.get('password')?.hasError('weakPassword')
      &&
      form.get('password')?.touched
    "
    class="error"
  >

    Password must contain:
    uppercase, lowercase, and number

  </div>

</form>
TypeScript


import {
  FormGroup,
  FormControl,
  ValidatorFn,
  AbstractControl,
  ValidationErrors
} from '@angular/forms';

export class AppComponent {

  form = new FormGroup({

    password: new FormControl(
      '',
      [this.weakPasswordValidator()]
    )

  });

  weakPasswordValidator(): ValidatorFn {

    return (
      control: AbstractControl
    ): ValidationErrors | null => {

      const value = control.value;

      if (!value) {
        return null;
      }

      const valid =

        /[A-Z]/.test(value) &&
        /[a-z]/.test(value) &&
        /[0-9]/.test(value);

      return valid
        ? null
        : { weakPassword: true };

    };

  }

}

How It Works

Part Purpose
{ weakPassword: true } Custom validator error object
hasError('weakPassword') Checks whether error exists
*ngIf Shows error conditionally
touched Prevents showing error before user interaction

Custom Error Message Example

You can also return dynamic error messages.

Validator


return {
  weakPassword: {
    message: 'Password is too weak'
  }
};
Template


<div
  *ngIf="
    form.get('password')?.errors?.['weakPassword']
  "
  class="error"
>

  {{
    form.get('password')
    ?.errors?.['weakPassword']
    ?.message
  }}

</div>
