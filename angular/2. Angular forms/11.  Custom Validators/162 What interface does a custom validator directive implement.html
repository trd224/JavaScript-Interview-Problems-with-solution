What Interface Does a Custom Validator Directive Implement?

In Angular, a custom validator directive implements the Validator interface.

Validator Interface Example 


import { Directive } from '@angular/core';

import {
  AbstractControl,
  NG_VALIDATORS,
  ValidationErrors,
  Validator
} from '@angular/forms';

@Directive({
  selector: '[appForbiddenName]',
  providers: [
    {
      provide: NG_VALIDATORS,
      useExisting: ForbiddenNameDirective,
      multi: true
    }
  ]
})

export class ForbiddenNameDirective implements Validator {

  validate(control: AbstractControl): ValidationErrors | null {

    if (control.value === 'admin') {
      return { forbiddenName: true };
    }

    return null;
  }
}

Required Method

The Validator interface requires the validate() method. 


validate(control: AbstractControl): ValidationErrors | null
  • Return null → Valid
  • Return object → Invalid

Invalid Response Example 


return { forbiddenName: true };
