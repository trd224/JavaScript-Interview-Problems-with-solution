How do you implement a 'no whitespace only' validator?

A whitespace validator prevents users from entering:

  • Empty spaces
  • Tabs
  • Only whitespace characters

It is commonly used for:

  • Name fields
  • Username fields
  • Comment boxes

Reactive Form Example

HTML Template


<form [formGroup]="form">

  <input
    type="text"
    formControlName="username"
    placeholder="Enter Username"
  >

  <div
    class="error"
    *ngIf="
      form.get('username')?.hasError('whitespace')
      &&
      form.get('username')?.touched
    "
  >

    Username cannot contain only spaces

  </div>

</form>
TypeScript


import {
  AbstractControl,
  ValidationErrors,
  ValidatorFn,
  FormControl,
  FormGroup
} from '@angular/forms';

export class AppComponent {

  form = new FormGroup({

    username: new FormControl(
      '',
      [this.noWhitespaceValidator()]
    )

  });

  noWhitespaceValidator(): ValidatorFn {

    return (
      control: AbstractControl
    ): ValidationErrors | null => {

      const value = control.value;

      if (value == null) {
        return null;
      }

      const isWhitespace =
        value.trim().length === 0;

      return isWhitespace
        ? { whitespace: true }
        : null;

    };

  }

}

How It Works

Code Purpose
value.trim() Removes spaces from start and end
length === 0 Checks whether only spaces remain
{ whitespace: true } Returns custom error
hasError('whitespace') Displays error message in template
