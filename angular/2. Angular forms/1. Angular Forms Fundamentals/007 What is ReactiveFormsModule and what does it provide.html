ReactiveFormsModule is an Angular module that enables Reactive Forms and provides classes and directives to build forms in TypeScript.

It provides:

  • FormGroup → to manage a group of controls
  • FormControl → to manage individual form fields
  • FormArray → to handle dynamic form controls
  • formControl, formGroup directives → to bind form in HTML

In short:

ReactiveFormsModule provides tools to create and manage reactive forms with full control in TypeScript.

Example

<form [formGroup]="form" (ngSubmit)="onSubmit()">

  <input 
    type="text" 
    formControlName="name" 
    placeholder="Enter Name"
  >

  <input 
    type="email" 
    formControlName="email" 
    placeholder="Enter Email"
  >

  <button type="submit">Submit</button>

</form>

TypeScript

import { FormGroup, FormControl } from '@angular/forms';

export class AppComponent {

  form = new FormGroup({
    name: new FormControl(''),
    email: new FormControl('')
  });

  onSubmit() {
    console.log(this.form.value);
  }

}

Output

{
  "name": "Rizwan",
  "email": "rizwan@gmail.com"
}
