value returns only enabled fields data while rawValue returns all fields data including disabled.

Example

form = new FormGroup({
  name: new FormControl('Rizwan'),
  email: new FormControl({
    value: 'rizwan@gmail.com', 
    disabled: true
  })
});

form.value

{
  "name": "Rizwan"
}

email is not included because it is disabled

form.getRawValue()

{
  "name": "Rizwan",
  "email": "rizwan@gmail.com"
}

email is included

Easy memory

value = enabled only

rawValue = everything
