FormsModule is an Angular module that enables Template-Driven Forms and provides directives to work with forms in HTML.

It provides:

  • ngForm → to track form state
  • ngModelGroup → to group controls
  • ngModel → for two-way data binding

In short:

FormsModule provides tools to create and manage template-driven forms using HTML.

Example

<form #myForm="ngForm" (ngSubmit)="onSubmit(myForm)">

  <div ngModelGroup="user">

    <input 
      type="text" 
      name="name" 
      [(ngModel)]="user.name" 
      placeholder="Enter Name"
      required
    >

    <input 
      type="email" 
      name="email" 
      [(ngModel)]="user.email" 
      placeholder="Enter Email"
      required
    >

  </div>

  <button type="submit">Submit</button>

</form>

TypeScript

export class AppComponent {

  user = {
    name: '',
    email: ''
  };

  onSubmit(form: any) {
    console.log(form.value);
  }

}

Output

{
  "user": {
    "name": "Rizwan",
    "email": "rizwan@gmail.com"
  }
}
