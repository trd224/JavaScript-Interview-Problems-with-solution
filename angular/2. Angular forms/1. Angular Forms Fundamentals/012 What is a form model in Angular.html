A form model in Angular is the object that stores and updates form values as the user types.

Simple Explanation with Example

Step 1: You have an input field

<input name="name" [(ngModel)]="user.name">

Step 2: You have an object in TypeScript

user = {
  name: ''
};

What happens now?

Initially

{
  "name": ""
}

User types: Rizwan

Angular automatically updates the object:

{
  "name": "Rizwan"
}

Important Point

  • The input field and object are connected
  • When user types → object updates
  • When object changes → UI updates

This object user = Form Model

Same idea in Reactive Form

TypeScript

form = new FormGroup({
  name: new FormControl('')
});

HTML

<input formControlName="name">

User types: Rizwan

{
  "name": "Rizwan"
}

form.value holds this data → this is the form model

Final Understanding

  • Form model = where form data is stored
  • Template form → normal object user
  • Reactive form → FormGroup
