What is the structure of the errors object on a control?

The errors object contains error names as keys and error details as values.

Example Errors Object


{
  required: true,

  minlength: {
    requiredLength: 5,
    actualLength: 2
  },

  email: true
}

Access Example


form.get('username')?.errors?.['required']

form.get('username')?.errors?.['minlength']?.requiredLength

form.get('username')?.errors?.['minlength']?.actualLength

If No Error


errors = null
Main Point: errors is null when control is valid.
