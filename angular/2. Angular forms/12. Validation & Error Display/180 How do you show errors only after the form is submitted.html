How do you show errors only after the form is submitted?

Create a boolean variable like submitted and show errors only when it becomes true.

TypeScript


submitted = false;

onSubmit() {

  this.submitted = true;

  if (this.form.invalid) {
    return;
  }

  console.log(this.form.value);

}

HTML View


<form [formGroup]="form" (ngSubmit)="onSubmit()">

  <input
    type="text"
    formControlName="username"
  >

  <div
    *ngIf="
      submitted &&
      form.get('username')?.errors?.['required']
    "
  >
    Username is required
  </div>

  <button type="submit">
    Submit
  </button>

</form>
Main Point: submitted controls when error messages appear.
