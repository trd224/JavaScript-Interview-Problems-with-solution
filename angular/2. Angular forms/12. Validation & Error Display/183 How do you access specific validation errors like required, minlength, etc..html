How do you access specific validation errors?

Use errors object with error name.

HTML View


<div
  *ngIf="form.get('username')?.errors?.['required']"
>
  Username is required
</div>

<div
  *ngIf="form.get('username')?.errors?.['minlength']"
>
  Minimum length error
</div>

TypeScript Access


const errors = this.form.get('username')?.errors;

console.log(errors?.['required']);

console.log(errors?.['minlength']);
Main Point: required, minlength, maxlength, email are keys inside control.errors.
