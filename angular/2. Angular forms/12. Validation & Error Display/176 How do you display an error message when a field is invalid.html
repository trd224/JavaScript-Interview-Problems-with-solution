How do you display an error message when a field is invalid?

In Angular, error messages are usually shown using: 


*ngIf

with: 


control.invalid
or 

control.errors

Reactive Form Example


form = this.fb.group({

  username: [

    '',

    [

      Validators.required,

      Validators.minLength(5)

    ]

  ]

});

HTML View


<form [formGroup]="form">

  <input
    type="text"
    formControlName="username"
    placeholder="Enter username"
  >

  <div
    class="error"
    *ngIf="
      form.get('username')?.invalid
      &&
      form.get('username')?.touched
    "
  >

    Field is invalid

  </div>

</form>

Specific Error Messages


<div
  class="error"
  *ngIf="form.get('username')?.errors?.['required']"
>

  Username is required

</div>

<div
  class="error"
  *ngIf="form.get('username')?.errors?.['minlength']"
>

  Minimum 5 characters required

</div>

Output Example

Minimum 5 characters required

Why use touched?


control.touched

It prevents error messages from showing immediately when page loads.

Error appears only after user touches the field.

Main Point

  • invalid → checks invalid state
  • errors → checks specific error
  • touched → shows error after interaction
Common Pattern: 

*ngIf="control.invalid && control.touched"
