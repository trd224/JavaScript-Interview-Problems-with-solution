What are touched, untouched, dirty, and pristine states?

Angular form controls have interaction states that tell how the user interacted with the field.

Reactive Form Example


form = this.fb.group({

  username: ['']

});

HTML View


<input
  type="text"
  formControlName="username"
>

<p>Touched: {{ form.get('username')?.touched }}</p>
<p>Untouched: {{ form.get('username')?.untouched }}</p>
<p>Dirty: {{ form.get('username')?.dirty }}</p>
<p>Pristine: {{ form.get('username')?.pristine }}</p>

Meaning

State Meaning
touched User focused and then left the field
untouched User has not left the field yet
dirty User changed the field value
pristine User has not changed the field value

Output Example

Touched: true

Untouched: false

Dirty: true

Pristine: false

Main Point

  • touched / untouched depends on focus and blur
  • dirty / pristine depends on value change
Use these states to show validation messages at the correct time.
