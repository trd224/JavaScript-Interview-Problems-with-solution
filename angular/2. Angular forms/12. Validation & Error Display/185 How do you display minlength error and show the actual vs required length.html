How do you display minlength error and show actual vs required length?

Reactive Form


form = this.fb.group({

  username: [

    '',

    [

      Validators.required,

      Validators.minLength(5)

    ]

  ]

});

HTML View


<div
  *ngIf="form.get('username')?.errors?.['minlength'] as error"
>

  Minimum {{ error.requiredLength }} characters required.
  You entered {{ error.actualLength }} characters.

</div>

Example Output


Minimum 5 characters required.
You entered 2 characters.
Main Point: minlength error gives requiredLength and actualLength.
