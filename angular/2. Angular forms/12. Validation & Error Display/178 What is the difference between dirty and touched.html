What is the difference between dirty and touched?

dirty is related to value change. touched is related to focus and blur.

Simple Difference

State When true?
dirty When user changes the input value
touched When user enters and leaves the input field

Example


<input
  type="text"
  formControlName="username"
>

<p>Dirty: {{ form.get('username')?.dirty }}</p>
<p>Touched: {{ form.get('username')?.touched }}</p>

Case 1: User clicks input and leaves without typing


dirty = false
touched = true

Case 2: User types something in input


dirty = true
touched = true

Output Example

Dirty: true

Touched: true

Main Point

  • dirty → value changed
  • touched → field visited and left
A field can be touched but not dirty. Example: user clicks the field and leaves without typing.
