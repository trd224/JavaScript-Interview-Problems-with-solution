What is the difference between form.errors and control.errors?

Property Meaning
form.errors Errors applied on the whole FormGroup
control.errors Errors applied on a specific FormControl

control.errors Example


username: ['', [Validators.required]]


<div *ngIf="form.get('username')?.errors?.['required']">
  Username is required
</div>

form.errors Example


form = this.fb.group(
  {
    password: [''],
    confirmPassword: ['']
  },
  {
    validators: passwordMatchValidator
  }
);


<div *ngIf="form.errors?.['passwordMismatch']">
  Passwords do not match
</div>
Main Point: control.errors is for one field. form.errors is for group-level validation.
