How do you mark all controls as touched programmatically?

Use markAllAsTouched() on the FormGroup.

TypeScript


onSubmit() {

  if (this.form.invalid) {

    this.form.markAllAsTouched();

    return;

  }

  console.log(this.form.value);

}

HTML View


<div
  *ngIf="
    form.get('username')?.invalid &&
    form.get('username')?.touched
  "
>
  Username is required
</div>
Main Point: markAllAsTouched() makes all hidden errors visible after submit.
