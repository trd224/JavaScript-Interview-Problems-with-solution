What is hasError() method on a form control?

hasError() checks whether a control has a specific validation error.

HTML View


<div
  *ngIf="form.get('username')?.hasError('required')"
>
  Username is required
</div>

<div
  *ngIf="form.get('username')?.hasError('minlength')"
>
  Minimum length error
</div>

TypeScript


this.form.get('username')?.hasError('required')

Same As


this.form.get('username')?.errors?.['required']
Main Point: hasError() is a cleaner way to check specific validation errors.
