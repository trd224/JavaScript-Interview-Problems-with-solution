Programmatically Trigger Form Submission From Outside Form in Angular

In Angular, you can programmatically trigger form submission from a button outside the <form> using:

  • @ViewChild() to access the form
  • Call ngSubmit.emit() manually

✅ Real Example (Template-Driven Form)

🔹 HTML 
<form #userForm="ngForm" (ngSubmit)="onSubmit(userForm)">

  <input
    type="text"
    name="username"
    ngModel
    placeholder="Enter username"
    required
  >

  <!-- Normal Submit Button -->
  <button type="submit">
    Inside Submit
  </button>

</form>

<!-- Outside Form -->
<button (click)="submitForm()">
  Outside Submit Button
</button>
🔹 TypeScript 
import { Component, ViewChild } from '@angular/core';
import { NgForm } from '@angular/forms';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {

  @ViewChild('userForm')
  userForm!: NgForm;

  submitForm() {

    // Trigger form submission programmatically
    this.userForm.ngSubmit.emit();

    //or
     //this.onSubmit();

  }

  onSubmit(form: NgForm) {

    console.log('Form Submitted');
    console.log(form.value);

  }

}

✅ How It Works

When you click the outside button: 

this.userForm.ngSubmit.emit();

Angular manually fires: 

(ngSubmit)="onSubmit(userForm)"

So the form submits exactly like a normal submit button.

✅ Real Output

If user enters: 

Rizwan

Console: 

Form Submitted
{username: 'Rizwan'}

✅ Real Example (Reactive Form)

🔹 HTML 
<form [formGroup]="userForm" (ngSubmit)="onSubmit()">

  <input
    type="text"
    formControlName="username"
    placeholder="Enter username"
  >

  <button type="submit">
    Inside Submit
  </button>

</form>

<!-- Outside Form -->
<button (click)="submitForm()">
  Outside Submit Button
</button>
🔹 TypeScript 
import { Component, ViewChild } from '@angular/core';
import {
  FormGroup,
  FormControl,
  FormGroupDirective
} from '@angular/forms';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {

  userForm = new FormGroup({
    username: new FormControl('')
  });

  @ViewChild(FormGroupDirective)
  formDirective!: FormGroupDirective;

  submitForm() {

    // Trigger Reactive Form submit
    this.formDirective.ngSubmit.emit();

  }

  onSubmit() {

    console.log('Reactive Form Submitted');
    console.log(this.userForm.value);

  }

}

✅ Reactive Form Output

If user enters: 

Rizwan

Console: 

Reactive Form Submitted
{username: 'Rizwan'}

✅ Reactive Form Explanation

Concept Purpose
FormGroup Creates Reactive Form
FormControl Controls input field value
FormGroupDirective Access Angular form directive
ngSubmit.emit() Triggers Angular form submit manually

✅ Another Method (Native HTML submit)

🔹 HTML 
<form #myForm>

</form>
🔹 TypeScript 
import { ElementRef, ViewChild } from '@angular/core';

@ViewChild('myForm')
formElement!: ElementRef;

submitForm() {

  this.formElement.nativeElement.submit();

}

But this bypasses Angular validation and ngSubmit.

So in Angular, preferred way is: 

this.userForm.ngSubmit.emit();

✅ Best Practice

Method Recommended Reason
ngSubmit.emit() ✅ Yes Uses Angular validation + lifecycle
native submit() ❌ Usually No Bypasses Angular form handling
