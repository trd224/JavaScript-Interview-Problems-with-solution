How Do You Prevent a Button From Submitting the Form Accidentally?

Inside a form, a <button> without a type is automatically treated as: 

type="submit"

So clicking the button submits the form automatically.

Important: To prevent accidental form submission, use: type="button"

❌ Wrong Example

This button accidentally submits the form because no type is specified. 

<form (ngSubmit)="onSubmit()">

  <button (click)="openPopup()">
    Open Popup
  </button>

</form>

Problem:

  • openPopup() runs
  • Form also submits accidentally

✅ Correct Example

Use: type="button" for normal buttons. 

<form (ngSubmit)="onSubmit()">

  <button
    type="button"
    (click)="openPopup()"
  >
    Open Popup
  </button>

</form>

Result:

  • Only openPopup() runs
  • Form does NOT submit

✅ Common Use Cases

Always use type="button" for:

  • Popup buttons
  • Add row buttons
  • Remove item buttons
  • Show/hide password
  • Custom click actions

✅ Submit Button Example

Use: type="submit" only when you want to submit the form. 

<button type="submit">
  Submit
</button>
