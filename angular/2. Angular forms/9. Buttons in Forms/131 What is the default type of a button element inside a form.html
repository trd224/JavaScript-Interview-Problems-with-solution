What is the Default Type of a Button Element Inside a Form?

In HTML and Angular, the default type of a <button> inside a <form> is: 

type="submit"

So these two buttons are the same:

<button>Save</button>
<button type="submit">Save</button>

When clicked, the form submits automatically.

✅ Template-Driven Form Example

<form #myForm="ngForm" (ngSubmit)="onSubmit()">

  <input type="text" name="username" ngModel>

  <!-- Default type = submit -->
  <button>Submit</button>

</form>

✅ Reactive Form Example

<form [formGroup]="myForm" (ngSubmit)="onSubmit()">

  <input type="text" formControlName="username">

  <!-- Default type = submit -->
  <button>Submit</button>

</form>

⚠ Important

If you use a button for another purpose like reset, open popup, add row, etc., always set: 

<button type="button">

Otherwise form submission will happen accidentally.

✅ Example

<button type="button" (click)="openPopup()">
  Open Popup
</button>
