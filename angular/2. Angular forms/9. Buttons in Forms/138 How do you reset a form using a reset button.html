How do you reset a form using a reset button in Angular?

A reset button clears the form fields and restores their initial values.

✅ Example Using type="reset"



<form #myForm="ngForm">

  <input
    type="text"
    name="username"
    [(ngModel)]="user.name"
    placeholder="Enter name"
  >

  <input
    type="email"
    name="email"
    [(ngModel)]="user.email"
    placeholder="Enter email"
  >

  <button type="submit">
    Submit
  </button>

  <button type="reset">
    Reset
  </button>

</form>

✅ TypeScript



user = {
  name: '',
  email: ''
};
Note:
The type="reset" button resets the form controls to their initial values automatically.

✅ Reset Form Completely Using Angular

To reset validation state, touched state, dirty state, and values together, use form.resetForm(). 



<form #myForm="ngForm">

  <input
    type="text"
    name="username"
    ngModel
  >

  <button
    type="button"
    (click)="myForm.resetForm()"
  >
    Reset Form
  </button>

</form>
Difference:
  • type="reset" → Resets only form values
  • resetForm() → Resets values + validation states

✅ resetForm() vs reset()

Method Used In Purpose
resetForm() Template-driven forms Reset values + form states
reset() Reactive forms Reset FormGroup values + states
