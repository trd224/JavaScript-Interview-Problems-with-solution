Dynamically Bind Button Label Text in Angular

In Angular, you can dynamically bind button text using:

  • Interpolation → {{ }}
  • Property binding
  • Variables from TypeScript
  • Conditions
  • Functions

✅ Real Example (Interpolation)

🔹 HTML 
<button>
  {{ buttonText }}
</button>
🔹 TypeScript 
export class AppComponent {

  buttonText = 'Save';

}

✅ Output 

Save

Angular automatically updates the button label whenever the variable changes.

✅ Dynamic Change Example

🔹 HTML 
<button (click)="changeText()">
  {{ buttonText }}
</button>
🔹 TypeScript 
export class AppComponent {

  buttonText = 'Submit';

  changeText() {
    this.buttonText = 'Submitted';
  }

}

✅ Output

Initially: 

Submit

After click: 

Submitted

✅ Conditional Button Text

🔹 HTML 
<button>
  {{ isLoading ? 'Loading...' : 'Submit' }}
</button>
🔹 TypeScript 
isLoading = false;

✅ Real Usage

Condition Button Text
isLoading = false Submit
isLoading = true Loading...

✅ Example Using API Loading

🔹 HTML 
<button [disabled]="isLoading">

  {{ isLoading ? 'Saving...' : 'Save' }}

</button>
🔹 TypeScript 
isLoading = false;

saveData() {

  this.isLoading = true;

  setTimeout(() => {
    this.isLoading = false;
  }, 3000);

}

✅ Using Function

🔹 HTML 
<button>
  {{ getButtonText() }}
</button>
🔹 TypeScript 
getButtonText() {
  return 'Login';
}

✅ Best Practice

Usually Angular developers use: 

{{ buttonText }}

or 

{{ condition ? 'Text1' : 'Text2' }}

for dynamic button labels.
