Preserve Newlines from a Textarea in Angular

Newlines from a textarea are preserved in Angular by using white-space: pre-line or pre-wrap in CSS.

✅ Problem

When text from a <textarea> is displayed in normal HTML, line breaks (Enter) are removed because HTML ignores newline characters (\n).

So this:

Hello World

becomes:

Hello World

To preserve line breaks, use: 



white-space: pre-line;

This tells HTML to display newline characters as actual new lines.

✅ Template-driven Form Example

HTML


<textarea
  [(ngModel)]="message"
  rows="5"
></textarea>

<div class="output">
  {{ message }}
</div>
CSS


.output{
  white-space: pre-line;
}
TS


message = '';

✅ How it works

  • Textarea stores line breaks using \n
  • Normal HTML ignores line breaks
  • white-space: pre-line preserves newlines
  • Each Enter key appears on a new line

✅ Output Example

Line 1 Line 2 Line 3

✅ Reactive Form Example

HTML


<form [formGroup]="form">

  <textarea
    formControlName="message"
    rows="5"
  ></textarea>

</form>

<div class="output">
  {{ form.value.message }}
</div>
CSS


.output{
  white-space: pre-line;
}
TS


import {
  FormGroup,
  FormControl
} from '@angular/forms';

form = new FormGroup({
  message: new FormControl('')
});
