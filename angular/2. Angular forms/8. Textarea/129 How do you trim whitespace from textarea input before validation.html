Trim Whitespace from Textarea Input Before Validation

Whitespace is trimmed from a textarea in Angular by using the JavaScript trim() method before validation or form submission.

✅ Template-driven Form Example

HTML


<form
  #form="ngForm"
  (ngSubmit)="onSubmit(form)"
>

  <textarea
    name="message"
    [(ngModel)]="message"
    required
    #msg="ngModel"
  ></textarea>

  <div
    *ngIf="form.submitted && msg.invalid"
  >
    Message is required
  </div>

  <button type="submit">
    Submit
  </button>

</form>
TS


message = '';

onSubmit(form: any) {

  // Trim whitespace
  this.message = this.message.trim();

  // Check after trimming
  if (!this.message) {
    return;
  }

  console.log(this.message);

}

✅ How it works

  • trim() removes spaces from start and end
  • " " becomes empty string ""
  • Prevents users from submitting only spaces

✅ Output

• Only spaces → validation fails

• Proper text → form submits successfully

✅ Reactive Form Example

HTML


<form
  [formGroup]="form"
  (ngSubmit)="onSubmit()"
>

  <textarea
    formControlName="message"
  ></textarea>

  <div
    *ngIf="
      submitted &&
      form.controls['message'].invalid
    "
  >
    Message is required
  </div>

  <button type="submit">
    Submit
  </button>

</form>
TS


import {
  FormGroup,
  FormControl,
  Validators
} from '@angular/forms';

submitted = false;

form = new FormGroup({
  message: new FormControl('', Validators.required)
});

onSubmit() {

  this.submitted = true;

  // Trim value
  const trimmedValue =
    this.form.value.message?.trim();

  // Validation after trim
  if (!trimmedValue) {
    return;
  }

  console.log(trimmedValue);

}
