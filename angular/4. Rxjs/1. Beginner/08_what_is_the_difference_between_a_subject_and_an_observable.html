RxJS Beginner · Question 08

What is the difference between a Subject and an Observable?

Observable

An Observable is generally a read-only stream from the subscriber's point of view. Subscribers can listen to values but cannot push new values into it.

Subject

A Subject is both Observable and Observer. It can emit values using next() and can also be subscribed to.

Main Difference

  • Observable only exposes subscription behavior.
  • Subject exposes subscription and emission behavior.
  • Subject is multicast; many subscribers receive the same emitted value.

Key Points

  • Observable is read-only to subscribers.
  • Subject can push values with next().
  • Services should expose Observable, not Subject.
Remember: Observable is for listening; Subject is for listening and pushing.

Full Understandable Example / Examples

observable-example.ts
import { of } from 'rxjs';

const data$ = of('A', 'B');

data$.subscribe(value => {
  console.log(value);
});

// data$.next('C') is not possible
prints A and B.
subject-example.ts
import { Subject } from 'rxjs';

const subject$ = new Subject<string>();

subject$.subscribe(value => console.log(value));

subject$.next('A');
subject$.next('B');
prints A and B because next() pushes values.
