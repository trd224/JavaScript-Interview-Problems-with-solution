What is ReplaySubject?

Beginner-friendly interview-level answer with headings, lists, full examples for every concept, colorful code, and expected behavior.

Interview-Level Answer

Definition

ReplaySubject is a Subject that stores previous emitted values and replays them to new subscribers.

Behavior

  • ReplaySubject(2) stores last two values.
  • New subscribers receive stored values immediately.
  • Then they receive future values normally.

Use Case

Use ReplaySubject when late subscribers need some history, such as last messages, last events, or cached stream values.

Key Points

  • ReplaySubject replays old values.
  • Buffer size controls how many values are stored.
  • Useful for late subscribers needing history.
Remember: ReplaySubject remembers previous emissions.

Full Understandable Example / Examples

replay-subject-example.ts
import { ReplaySubject } from 'rxjs';

const replay$ = new ReplaySubject<string>(2);

replay$.next('A');
replay$.next('B');
replay$.next('C');

replay$.subscribe(value => {
  console.log(value);
});

replay$.next('D');
