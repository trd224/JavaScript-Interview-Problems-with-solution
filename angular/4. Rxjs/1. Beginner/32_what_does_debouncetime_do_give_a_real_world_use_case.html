RxJS Beginner · Question 32

What does debounceTime() do? Give a real-world use case.

Definition

debounceTime waits for a pause in emissions before emitting the latest value.

How It Works

  • If values keep coming, timer resets.
  • When emissions stop for the given time, latest value is emitted.

Real-World Use Case

Search input/typeahead is the most common use case. It prevents API calls on every keystroke.

Key Points

  • debounceTime waits for silence.
  • Emits latest value after pause.
  • Great for search boxes.
Remember: debounceTime reduces unnecessary rapid emissions.

Full Understandable Example / Examples

debounce-search-example.ts
import { FormControl } from '@angular/forms';
import { debounceTime, distinctUntilChanged } from 'rxjs/operators';

const searchControl = new FormControl('');

searchControl.valueChanges.pipe(
  debounceTime(500),
  distinctUntilChanged()
).subscribe(searchText => {
  console.log('Call API with:', searchText);
});
User types a, an, ang quickly.
debounce-click-example.ts
import { fromEvent } from 'rxjs';
import { debounceTime } from 'rxjs/operators';

fromEvent(document, 'click').pipe(
  debounceTime(300)
).subscribe(() => {
  console.log('User stopped clicking quickly');
});
Call API with: ang
