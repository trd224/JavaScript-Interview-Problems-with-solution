What does last() operator do?

Beginner-friendly interview-level answer with headings, lists, full examples for every concept, colorful code, and expected behavior.

Interview-Level Answer

Definition

last() emits the last value from a source Observable when the source completes.

Behavior

  • It waits for completion.
  • It can accept a condition.
  • If source never completes, last() never emits.

Use Case

Use last() when you need the final emitted value from a finite stream.

Key Points

  • last needs source completion.
  • It emits final value.
  • It can filter final matching value.
Remember: last waits until the stream completes.

Full Understandable Example / Examples

last-example.ts
import { of } from 'rxjs';
import { last } from 'rxjs/operators';

of(10, 20, 30).pipe(
  last()
).subscribe(value => {
  console.log(value);
});
last-with-condition.ts
import { of } from 'rxjs';
import { last } from 'rxjs/operators';

of(1, 2, 3, 4, 5).pipe(
  last(value => value % 2 === 0)
).subscribe(value => {
  console.log(value);
});

Expected Output / Behavior

last(): 30
last even value: 4
