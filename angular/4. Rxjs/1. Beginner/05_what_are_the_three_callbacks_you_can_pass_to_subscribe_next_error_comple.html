RxJS Beginner · Question 05

What are the three callbacks you can pass to subscribe()? (next, error, complete)

Definition

The three callbacks are next, error, and complete. They represent the three types of notifications an Observable can send.

Callbacks

  • next: called whenever a value is emitted.
  • error: called when the stream fails. After error, stream stops.
  • complete: called when the stream finishes successfully.

Key Points

  • next handles values.
  • error handles failure.
  • complete handles successful finish.
Remember: An Observable can complete or error, but after that it emits no more values.

Full Understandable Example / Examples

subscribe-callbacks-success.ts
import { of } from 'rxjs';

of(10, 20, 30).subscribe({
  next: value => console.log('Next:', value),
  error: error => console.error('Error:', error),
  complete: () => console.log('Completed')
});
Next: 10
Next: 20
Next: 30
Completed
subscribe-callbacks-error.ts
import { throwError } from 'rxjs';

throwError(() => new Error('Something failed')).subscribe({
  next: value => console.log(value),
  error: error => console.error('Error:', error.message),
  complete: () => console.log('Completed')
});
Error: Something failed
