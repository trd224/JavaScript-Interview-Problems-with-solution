What is finalize() and when would you use it?

Definition

finalize() runs cleanup logic when an Observable completes, errors, or is unsubscribed.

Use Cases

  • Stop loading spinner.
  • Close resources.
  • Log stream cleanup.
  • Run cleanup regardless of success or failure.

Angular Use Case

Use finalize() with HTTP calls to stop loading indicators even if the API fails.

Key Points

  • finalize runs on complete, error, or unsubscribe.
  • Good for cleanup.
  • Similar to finally block.
Remember: finalize is the finally block of RxJS.

Full Understandable Example / Examples

finalize-success-example.ts
import { of } from 'rxjs';
import { finalize } from 'rxjs/operators';

let loading = true;

of('Data loaded').pipe(
  finalize(() => {
    loading = false;
    console.log('Loading stopped');
  })
).subscribe(value => {
  console.log(value);
});
Data loaded
finalize-http-example.ts
loading = true;

this.http.get('/api/users').pipe(
  catchError(error => of([])),
  finalize(() => {
    this.loading = false;
  })
).subscribe(users => {
  console.log(users);
});
Loading stopped
