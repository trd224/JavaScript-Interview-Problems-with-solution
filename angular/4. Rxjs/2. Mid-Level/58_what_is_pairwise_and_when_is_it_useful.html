What is pairwise() and when is it useful?

Definition

pairwise emits the previous and current values together as an array.

Important Behavior

  • It starts emitting from the second source value.
  • Output format is [previous, current].

Use Cases

Use pairwise for comparing old and new form values, detecting route changes, checking scroll direction, or tracking price changes.

Key Points

  • Emits pairs of previous and current values.
  • Does not emit on the first value.
  • Useful for comparison-based logic.
Remember: pairwise is best when you need to know what changed.

Full Understandable Example / Examples

pairwise-example.ts
import { FormControl } from '@angular/forms';
import { pairwise, startWith } from 'rxjs/operators';

const priceControl = new FormControl(100);

priceControl.valueChanges.pipe(
  startWith(priceControl.value),
  pairwise()
).subscribe(([previous, current]) => {
  console.log('Previous price:', previous);
  console.log('Current price:', current);
});
Previous price: 100
Current price: 120
