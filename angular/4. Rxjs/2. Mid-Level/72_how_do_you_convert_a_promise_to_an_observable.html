How do you convert a Promise to an Observable?

Interview-level answer with full explanation, real Angular-style example, colorful code, and expected output.

Interview-Level Answer

Definition

A Promise can be converted to an Observable using from().

Behavior

  • If the Promise resolves, the Observable emits the resolved value and completes.
  • If the Promise rejects, the Observable errors.

Use Case

Use this when Promise-based APIs need to be used inside an RxJS pipeline.

Key Points

  • Use from(promise).
  • Resolved value becomes next emission.
  • Rejected Promise becomes Observable error.
Remember: from() is the standard way to convert Promise to Observable.

Full Understandable Example / Examples

promise-to-observable.ts
import { from } from 'rxjs';

const fetchUserPromise = fetch('/api/user/1')
  .then(response => response.json());

const user$ = from(fetchUserPromise);

user$.subscribe(user => {
  console.log('User:', user);
});

Expected Output / Behavior

User: { id: 1, name: 'Rizwan' }
