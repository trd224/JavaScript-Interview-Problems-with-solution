RxJS Mid-Level · Question 45

What does scan() do? How is it like reduce() but for streams?

scan gives running accumulated values, while reduce gives only the final accumulated value after completion.

Answer

Definition

scan() accumulates values over time and emits the updated accumulated result after every source emission.

How It Is Like reduce

  • Both scan() and reduce() accumulate values.
  • scan() emits every intermediate accumulated value.
  • reduce() emits only the final value after the Observable completes.

Simple Understanding

  • scan() → running total
  • reduce() → final total

Angular Use Case

Use scan() for live counters, cart totals, local component state, undo/redo state, and event accumulation.

Use reduce() when you need only one final result, like final API data summary, total price after all items load, or final report generation.

Difference Between scan() and reduce()

Feature scan() reduce()
Emission Emits after every value Emits only once at the end
Output Type Running accumulated values Final accumulated value
Works Before Completion Yes No
Similar To Array running total Array reduce final result
Common Use Case Live counters, totals, state updates Final summary/result
Remember: scan is reduce for live streams.

Key Points

  • scan() maintains accumulated state over time.
  • scan() emits intermediate accumulated values.
  • reduce() waits for Observable completion.
  • reduce() emits only one final accumulated value.

Full Understandable Example / Examples

scan-example.ts
import { of } from 'rxjs';
import { scan } from 'rxjs/operators';

// scan emits running accumulated values.

of(100, 200, 300).pipe(
  scan((total, value) => total + value, 0)
).subscribe(total => console.log('scan:', total));
scan: 100
scan: 300
scan: 600

scan() Step-by-step

Initial accumulator value = 0

Current Value Calculation Output
100 0 + 100 100
200 100 + 200 300
300 300 + 300 600

scan() works like a running total accumulator. It emits the updated accumulated value after every emission.

reduce-comparison-example.ts
import { of } from 'rxjs';
import { reduce } from 'rxjs/operators';

// reduce emits only final accumulated value after completion.

of(100, 200, 300).pipe(
  reduce((total, value) => total + value, 0)
).subscribe(total => console.log('reduce:', total));
reduce: 600

reduce() Step-by-step

Initial accumulator value = 0

Current Value Calculation Accumulated
100 0 + 100 100
200 100 + 200 300
300 300 + 300 600

After the Observable completes, reduce() emits only the final accumulated value.
