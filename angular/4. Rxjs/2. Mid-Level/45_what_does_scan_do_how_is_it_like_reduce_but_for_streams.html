What does scan() do? How is it like reduce() but for streams?

Definition

scan accumulates values over time and emits the accumulated result after every source emission.

How It Is Like reduce

  • Both scan and reduce accumulate values.
  • scan emits every intermediate accumulated value.
  • reduce emits only the final value after completion.

Angular Use Case

Use scan for counters, cart totals, local component state, undo/redo state, and event accumulation.

Key Points

  • Maintains accumulated state over time.
  • Emits intermediate accumulated values.
  • Useful for counters, cart totals, and simple state stores.
Remember: scan is reduce for live streams.

Full Understandable Example / Examples

scan-example.ts
import { of } from 'rxjs';
import { scan } from 'rxjs/operators';

// scan emits running accumulated values.

of(100, 200, 300).pipe(
  scan((total, value) => total + value, 0)
).subscribe(total => console.log('scan:', total));
