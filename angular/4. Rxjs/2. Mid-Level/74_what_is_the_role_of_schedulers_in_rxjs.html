What is the role of Schedulers in RxJS?

Definition

Schedulers control when and how RxJS work is executed.

Why They Matter

  • They affect execution timing.
  • They affect emission order.
  • They are useful for testing and advanced performance control.

Common Schedulers

Common examples include asyncScheduler, asapScheduler, queueScheduler, and animationFrameScheduler.

Key Points

  • Schedulers control execution timing.
  • They can make synchronous streams asynchronous.
  • Useful for testing and advanced performance control.
Remember: Schedulers decide when Observable work runs.

Full Understandable Example / Examples

scheduler-example.ts
import { asyncScheduler, of } from 'rxjs';
import { observeOn } from 'rxjs/operators';

console.log('Start');

of('A', 'B').pipe(
  observeOn(asyncScheduler)
).subscribe(value => {
  console.log(value);
});

console.log('End');
