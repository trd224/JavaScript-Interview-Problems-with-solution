What does combineLatest() do? When does it emit?

Interview-level answer with full explanation, real Angular-style example, colorful code, and expected output.

Interview-Level Answer

Definition

combineLatest combines multiple Observables and emits the latest value from each source as an array or object.

When It Emits

  • It waits until every source Observable has emitted at least once.
  • After that, whenever any source emits, it emits again with the latest values from all sources.

Angular Use Case

Use combineLatest for live UI state such as search text, selected category, sort order, pagination, route params, or logged-in user state.

Important Interview Point

combineLatest may not emit initially if one source has not emitted. That is why startWith() is commonly used with Angular form valueChanges.

Key Points

  • Waits for all sources to emit once before first output.
  • After first output, any source emission triggers a new combined output.
  • Often used with startWith for Angular form controls.
Remember: combineLatest is best for live latest-value combinations.

Full Understandable Example / Examples

products.component.ts
import { Component, OnInit } from '@angular/core';
import { FormControl } from '@angular/forms';
import { combineLatest } from 'rxjs';
import { startWith } from 'rxjs/operators';

@Component({
  selector: 'app-products',
  template: `
    <input [formControl]="searchControl" placeholder="Search product" />

    <select [formControl]="categoryControl">
      <option value="all">All</option>
      <option value="mobile">Mobile</option>
      <option value="laptop">Laptop</option>
    </select>
  `
})
export class ProductsComponent implements OnInit {
  searchControl = new FormControl('');
  categoryControl = new FormControl('all');

  ngOnInit() {
    combineLatest([
      this.searchControl.valueChanges.pipe(startWith('')),
      this.categoryControl.valueChanges.pipe(startWith('all'))
    ]).subscribe(([search, category]) => {
      console.log('Search:', search);
      console.log('Category:', category);
    });
  }
}

Expected Output / Behavior

Initial output:
Search: 
Category: all

After user types mobile:
Search: mobile
Category: all

After category changes:
Search: mobile
Category: laptop
