What is race() in RxJS?

Interview-level answer with full explanation, real Angular-style example, colorful code, and expected output.

Interview-Level Answer

Definition

race subscribes to multiple Observables and mirrors the first Observable that emits.

How It Works

  • The first Observable to emit wins.
  • Other Observables are unsubscribed.
  • The winner continues to emit values.

Use Case

Use race for fastest-source logic or API response versus timeout behavior.

Key Points

  • First Observable to emit wins.
  • Other Observables are unsubscribed.
  • Useful for fastest source or timeout behavior.
Remember: race means first emission wins.

Full Understandable Example / Examples

race-example.ts
import { race, timer } from 'rxjs';
import { map } from 'rxjs/operators';

const apiResponse$ = this.http.get('/api/data');

const timeout$ = timer(3000).pipe(
  map(() => 'Request timeout')
);

race(apiResponse$, timeout$).subscribe(result => {
  console.log('Winner:', result);
});

Expected Output / Behavior

If API responds before 3 seconds: API result wins.
If API is slow: Request timeout wins.
