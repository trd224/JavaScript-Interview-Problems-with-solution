What does pluck() do? (deprecated in newer RxJS — what replaces it?)

Interview-level answer with full explanation, real Angular-style example, colorful code, and expected output.

Interview-Level Answer

Definition

pluck was used to extract a property from emitted objects, but it is deprecated in newer RxJS versions.

Replacement

  • Use map instead of pluck.
  • map is more flexible and type-safe.
  • map is clearer for nested properties.

Example

Replace pluck('profile', 'name') with map(user => user.profile.name).

Key Points

  • pluck extracted object properties.
  • It is deprecated in modern RxJS.
  • Use map instead for type safety and readability.
Remember: Replace pluck('profile','name') with map(user => user.profile.name).

Full Understandable Example / Examples

pluck-replacement-example.ts
import { of } from 'rxjs';
import { map } from 'rxjs/operators';

const user$ = of({
  id: 1,
  profile: {
    name: 'Rizwan'
  }
});

// Modern replacement for deprecated pluck('profile', 'name')
user$.pipe(
  map(user => user.profile.name)
).subscribe(name => {
  console.log('User name:', name);
});

Expected Output / Behavior

User name: Rizwan
