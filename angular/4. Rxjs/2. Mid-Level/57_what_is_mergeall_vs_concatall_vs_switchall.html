What is mergeAll() vs concatAll() vs switchAll()?

Interview-level answer with full explanation, real Angular-style example, colorful code, and expected output.

Interview-Level Answer

Definition

mergeAll, concatAll, and switchAll are flattening operators used when the source Observable emits inner Observables.

Operator Difference

  • mergeAll: subscribes to inner Observables concurrently.
  • concatAll: queues inner Observables and runs them one by one.
  • switchAll: switches to the latest inner Observable and cancels the previous one.

Relation With Mapping Operators

mergeMap is map + mergeAll, concatMap is map + concatAll, and switchMap is map + switchAll.

Key Points

  • Used for higher-order Observables.
  • mergeAll = concurrent, concatAll = ordered queue, switchAll = latest only.
  • Mapping operators are often easier in application code.
Remember: mergeMap = map + mergeAll; switchMap = map + switchAll; concatMap = map + concatAll.

Full Understandable Example / Examples

mergeAll-example.ts
import { of } from 'rxjs';
import { delay, map, mergeAll } from 'rxjs/operators';

// mergeAll subscribes to inner Observables concurrently.

of(1, 2, 3).pipe(
  map(id => of(`mergeAll API ${id}`).pipe(delay(1000))),
  mergeAll()
).subscribe(value => console.log(value));
concatAll-example.ts
import { of } from 'rxjs';
import { delay, map, concatAll } from 'rxjs/operators';

// concatAll waits for each inner Observable to complete before next.

of(1, 2, 3).pipe(
  map(id => of(`concatAll API ${id}`).pipe(delay(1000))),
  concatAll()
).subscribe(value => console.log(value));
switchAll-example.ts
import { interval, of } from 'rxjs';
import { delay, map, switchAll, take } from 'rxjs/operators';

// switchAll switches to latest inner Observable.

interval(500).pipe(
  take(3),
  map(id => of(`switchAll latest API ${id}`).pipe(delay(1000))),
  switchAll()
).subscribe(value => console.log(value));

Expected Output / Behavior

Each operator flattens inner Observables differently.
Exact output depends on timing and inner Observable duration.
