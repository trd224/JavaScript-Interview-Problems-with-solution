What is the expand() operator and what recursive patterns can it solve?

Advanced with headings, lists, full examples for every concept, colorful code, and expected behavior.

Definition

expand recursively maps each emitted value to another Observable and merges the result back into the output stream.

Use Cases

  • Recursive API pagination.
  • Tree traversal.
  • Polling with dynamic next request.
  • Repeating work until no next value exists.

Important Point

expand can easily create infinite recursion if there is no stop condition, so use EMPTY or takeWhile/takeUntil to stop.

Key Points

  • expand recursively projects values.
  • Useful for pagination and tree traversal.
  • Always define a stop condition.
Remember: expand is recursion for Observable streams.

Full Understandable Example / Examples

pagination-expand-example.ts
import { EMPTY } from 'rxjs';
import { expand, reduce } from 'rxjs/operators';

function getPage(page: number) {
  return http.get<any>(`/api/products?page=${page}`);
}

getPage(1).pipe(
  expand(response => {
    return response.nextPage
      ? getPage(response.nextPage)
      : EMPTY;
  }),
  reduce((allItems, response) => {
    return [...allItems, ...response.items];
  }, [])
).subscribe(allProducts => {
  console.log('All products:', allProducts);
});
Page 1 loaded.
If nextPage exists, next page is loaded.
Stops when nextPage is missing.
