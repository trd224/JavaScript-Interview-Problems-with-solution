#064 What are RxJS operators, Name common ones

🔹 What are RxJS Operators?

👉 RxJS operators are functions used to transform, filter, and manipulate data streams (Observables) in Angular.

🔹 Why Use Operators?

  • Modify data
  • Filter values
  • Handle async flows easily
  • Chain multiple operations

🔹 Common RxJS Operators 🔥

  • 🔸 Transformation
ItemValue
map()Transform data
map(value => value * 2)
  • 🔸 Filtering
ItemValue
filter()Allow specific values
filter(value => value > 10)
  • 🔸 Switching
ItemValue
switchMap()Cancel previous request, use latest
  • 🔸 Merging
ItemValue
mergeMap()Run multiple requests together
  • 🔸 Combination
ItemValue
combineLatest()Combine multiple observables
  • 🔸 Utility
ItemValue
tap()Side effects (log, debug)
tap(value => console.log(value))
  • 🔸 Error Handling
ItemValue
catchError()Handle errors

🔹 Example

this.http.get('/api')
  .pipe(
    map(data => data),
    catchError(err => throwError(err))
  )
  .subscribe();

🔹 Key Points

Used with .pipe()
  • Chain multiple operations
  • Core part of RxJS
