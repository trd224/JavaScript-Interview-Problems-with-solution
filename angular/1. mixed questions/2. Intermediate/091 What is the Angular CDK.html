#091 What is the Angular CDK

Angular CDK (Component Dev Kit) is a set of low-level tools in Angular that helps us build custom UI components with common behaviors like overlay, drag-drop, accessibility, and scrolling without using full Material Design components.

🔹 What it means (simple)

  • CDK = toolkit for building UI features
  • It gives behavior, not UI
  • We design UI ourselves

🔹 Why it is used

  • Build custom components (like dropdown, modal)
  • Add advanced features easily
  • Reuse common UI behaviors
  • Better control than ready-made components

🔹 Example features in Angular CDK

1. Overlay (popup, modal)

  • Create floating elements

2. Drag & Drop

  • Move items easily
import { CdkDrag } from '@angular/cdk/drag-drop';

3. Portal

  • Dynamically render content

4. Scrolling

  • Virtual scrolling for large lists

🔹 Simple example (Drag & Drop)

<div cdkDrag>
  Drag me
</div>
