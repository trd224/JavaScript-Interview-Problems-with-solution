063. How do you pass data from child to parent component?

One-Line Definition

In React, child components send data to parent components by calling functions passed through props.

Important Concept

React normally follows:

Parent → Child Data Flow

So child components cannot directly modify parent data.

Instead:

Parent Passes Function
↓
Child Calls Function
↓
Parent Receives Data

Basic Example

function Child({ sendData }) {

  return (

    <button onClick={() => sendData('Rizwan')}>
      Send Data
    </button>

  );

}

function App() {

  const receiveData = (name) => {

    console.log(name);

  };

  return (

    <Child sendData={receiveData} />

  );

}

Output

Rizwan

How It Works

Step Description
1 Parent creates function
2 Parent passes function as prop
3 Child calls function
4 Parent receives child data

Real Example with useState

import { useState } from 'react';

function Child({ updateName }) {

  return (

    <button onClick={() => updateName('Aman')}>
      Change Name
    </button>

  );

}

function App() {

  const [name, setName] = useState('Rizwan');

  return (

    <div>

      <h1>{name}</h1>

      <Child updateName={setName} />

    </div>

  );

}

Output Before Click

Rizwan

Output After Click

Aman

Why This Pattern Is Used

  • Maintains one-way data flow
  • Keeps state controlled by parent
  • Improves predictability
  • Better architecture

Common Real-World Use Cases

Feature Child Sends
Search box Search text
Login form User credentials
OTP component OTP value
Popup component Close event
Reusable button Click events

Passing Multiple Values

<button

  onClick={() => sendData({

    name: 'Rizwan',
    age: 25

  })}

>

  Send

</button>

Parent Receives Object

const receiveData = (user) => {

  console.log(user.name);

  console.log(user.age);

};

Important Note

Child components do NOT directly update parent state. They only notify the parent using callback functions.

Important Interview Points

  • Child-to-parent communication uses callback functions.
  • Functions are passed via props.
  • Child calls parent function with data.
  • Maintains React one-way data flow.
  • Very common in forms and reusable components.

Short Interview Answer

In React, child components send data to parent components by calling callback functions passed from the parent through props.

One-Line Interview Answer

Child components pass data to parents by calling functions received through props.
