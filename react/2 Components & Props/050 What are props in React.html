050. What are props in React?

One-Line Definition

Props are read-only inputs used to pass data from a parent component to a child component in React.

What Does Props Mean?

Props stands for:

Properties

Props allow components to receive dynamic data.

Why Props Are Used

  • Pass data between components
  • Make components reusable
  • Create dynamic UI
  • Support one-way data flow

Basic Example

function User(props) {

  return (

    <h1>
      Hello {props.name}
    </h1>

  );

}

function App() {

  return (

    <User name="Rizwan" />

  );

}

Output

Hello Rizwan

How Props Work

Parent Component
↓
Pass Props
↓
Child Component Receives Data

Parent and Child Relationship

Component Role
App Parent component
User Child component

Props Are Read-Only

Props should never be modified inside child components.

Wrong Example

// Wrong

props.name = 'Aman';

Why?

React follows one-way data flow.

Data moves:

Parent → Child

Destructuring Props

Instead of using: 

props.name

We can destructure props.

Example

function User({ name }) {

  return (

    <h1>
      Hello {name}
    </h1>

  );

}

Passing Multiple Props

<User

  name="Rizwan"

  age={25}

  city="Jaipur"

/>

Receiving Multiple Props

function User({ name, age, city }) {

  return (

    <div>

      <h2>{name}</h2>

      <p>{age}</p>

      <p>{city}</p>

    </div>

  );

}

Props Can Pass

Data Type Example
String name="Rizwan"
Number age={25}
Boolean isAdmin={true}
Array users={['A', 'B']}
Object user={{ name: 'Rizwan' }}
Function onClick={handleClick}

Passing Function as Props

function Button({ onPress }) {

  return (

    <button onClick={onPress}>
      Click
    </button>

  );

}

Important Interview Points

  • Props are used to pass data between components.
  • Props are read-only.
  • Data flows parent to child.
  • Props improve component reusability.
  • Functions can also be passed as props.

Short Interview Answer

Props are read-only inputs used to transfer data from parent components to child components in React, enabling reusable and dynamic UI components.

One-Line Interview Answer

Props are read-only data passed from parent to child components in React.
