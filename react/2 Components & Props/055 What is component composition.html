055. What is component composition?

One-Line Definition

Component composition is the technique of combining smaller reusable React components together to build larger and more complex UIs.

Simple Understanding

React applications are built by combining components together.

Small components work like building blocks.

Small Components
↓
Combined Together
↓
Large UI Created

Real-Life Analogy

Like building a car using:

  • Engine
  • Wheels
  • Doors
  • Seats

React builds applications using:

  • Button components
  • Card components
  • Navbar components
  • Form components

Basic Example

function Header() {

  return <h1>Header</h1>;

}

function Sidebar() {

  return <aside>Sidebar</aside>;

}

function Content() {

  return <main>Content</main>;

}

function Dashboard() {

  return (

    <div>

      <Header />

      <Sidebar />

      <Content />

    </div>

  );

}

What Happened Here?

Dashboard is composed using:

  • Header
  • Sidebar
  • Content

This is:

Component Composition

Why Component Composition Is Important

Benefit Description
Reusability Reuse components multiple times
Maintainability Easy to manage code
Scalability Supports large applications
Separation of concerns Each component has one responsibility

Composition Using children Prop

function Card({ children }) {

  return (

    <div className="card">

      {children}

    </div>

  );

}

Using Card Component

<Card>

  <h2>User Profile</h2>

  <p>Welcome Rizwan</p>

</Card>

Why children Helps Composition

It allows components to act as reusable wrappers.

Composition vs Inheritance

Composition Inheritance
Combine components Extend classes
Preferred in React Rarely used in React
Flexible Tightly coupled
Reusable UI blocks Traditional OOP approach

React Recommendation

React officially recommends composition over inheritance.

  • Cleaner architecture

Important Interview Points

  • Composition combines smaller components.
  • React prefers composition over inheritance.
  • children prop is heavily used in composition.
  • Improves code reuse and maintainability.
  • Foundation of React architecture.

Short Interview Answer

Component composition is the practice of building complex React UIs by combining smaller reusable components together instead of using inheritance.
