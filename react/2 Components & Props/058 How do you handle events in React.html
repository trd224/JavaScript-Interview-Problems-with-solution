058. How do you handle events in React?

One-Line Definition

Events in React are handled by attaching event handler functions to JSX elements using camelCase event attributes.

What Is Event Handling?

Event handling means responding to user actions like:

  • Click
  • Typing
  • Submit
  • Mouse movement
  • Keyboard press

Basic Example

function App() {

  const handleClick = () => {

    console.log('Button Clicked');

  };

  return (

    <button onClick={handleClick}>
      Click
    </button>

  );

}

Output

Button Clicked

How Event Handling Works

User Action
↓
React Event Triggered
↓
Event Handler Function Executes

Important Syntax Rules

HTML React
onclick onClick
onchange onChange
onsubmit onSubmit

Important Point

React uses camelCase event names.

Handling Input Events

function App() {

  const handleChange = (event) => {

    console.log(event.target.value);

  };

  return (

    <input onChange={handleChange} />

  );

}

Output

Typing in input prints typed text in console.

Handling Form Submit

function App() {

  const handleSubmit = (event) => {

    event.preventDefault();

    console.log('Form Submitted');

  };

  return (

    <form onSubmit={handleSubmit}>

      <button>Submit</button>

    </form>

  );

}

Why preventDefault() Is Used

Prevents browser page reload during form submission.

Passing Parameters to Event Handlers

function App() {

  const handleClick = (name) => {

    console.log(name);

  };

  return (

    <button onClick={() => handleClick('Rizwan')}>
      Click
    </button>

  );

}

Output

Rizwan

Common React Events

Event Purpose
onClick Mouse click
onChange Input change
onSubmit Form submit
onKeyDown Keyboard press
onMouseEnter Mouse hover
onFocus Input focus

React Uses Synthetic Events

React wraps browser events inside:

SyntheticEvent

This provides consistent cross-browser behavior.

Inline Event Handling

<button onClick={() => console.log('Clicked')}>
  Click
</button>

Recommended Approach

For large logic:

  • Create separate handler functions
  • Avoid large inline functions

Event Propagation

React events support:

  • Bubbling
  • Capturing

Can stop propagation using: 

event.stopPropagation()

Real Project Use Cases

Feature Event Used
OTP form onChange, onSubmit
Popup close button onClick

Important Interview Points

  • React uses camelCase event names.
  • Event handlers receive SyntheticEvents.
  • Functions are passed, not executed directly.
  • preventDefault() stops browser behavior.
  • stopPropagation() stops bubbling.

Short Interview Answer

Events in React are handled by attaching handler functions to JSX elements using camelCase event attributes like onClick, onChange, and onSubmit.
