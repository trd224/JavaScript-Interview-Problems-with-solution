060. How do you render a list of items in React?

One-Line Definition

In React, lists are rendered by using JavaScript array methods like map() to convert data into JSX elements.

Why List Rendering Is Needed

Real applications often display collections of data like:

  • Users
  • Products
  • Notifications
  • Messages
  • Orders

React renders these collections dynamically using list rendering.

Most Common Method - map()

React commonly uses: 

array.map()

to generate JSX for each item.

Basic Example

function App() {

  const users = [

    'Rizwan',
    'Aman',
    'Rahul'

  ];

  return (

    <ul>
      {users.map((user) => <li>{user}</li>)}
    </ul>

  );

}

Output


• Rizwan
• Aman
• Rahul

How map() Works

Array Items
↓
map() Iterates Each Item
↓
React Creates JSX Elements
↓
UI Renders List

Important Problem

React needs a unique identifier for each rendered item.

This identifier is called:

key

Correct Example with key

function App() {

  const users = [

    { id: 1, name: 'Rizwan' },
    { id: 2, name: 'Aman' },
    { id: 3, name: 'Rahul' }

  ];

  return (

    <ul>
      {users.map((user) => <li key={user.id}>{user.name}</li>)}
    </ul>

  );

}

Why key Is Important

  • Helps React identify items
  • Improves rendering performance
  • Helps React track updates
  • Prevents rendering bugs

React Internal Process

Old List
↓
New List
↓
React Compares Keys
↓
Updates Only Changed Items

Wrong Example

// Warning: Missing key

users.map((user) => <li>{user.name}</li>)

React Warning


Each child in a list should have a unique "key" prop

Can We Use Index as key?

Yes, but only when list order never changes. 

key={index}

Why Index Is Not Recommended

  • Can cause incorrect UI updates
  • Problems during reordering
  • Issues with insertion/deletion

Best Practice

Always use stable unique IDs as keys whenever possible.

Rendering Complex Objects

const products = [

  {
    id: 1,
    name: 'Laptop',
    price: 50000
  },

  {
    id: 2,
    name: 'Mobile',
    price: 20000
  }

];

Render Products

<div>

  {

    products.map((product) => (

      <div key={product.id}>

        <h2>{product.name}</h2>

        <p>₹{product.price}</p>

      </div>

    ))

  }

</div>

Conditional List Rendering

{

  users.length > 0
    ? users.map((user) => <li key={user.id}>{user.name}</li>)
    : <p>No Users</p>

}

Important Interview Points

  • map() is commonly used for list rendering.
  • Each list item should have a unique key prop.
  • Keys help React optimize rendering.
  • Avoid using array index as key when possible.
  • List rendering is essential for dynamic UI.

Short Interview Answer

In React, lists are rendered by using array.map() to transform array data into JSX elements, and each item should include a unique key prop for efficient rendering.

One-Line Interview Answer

React renders lists using map() and unique key props.
