062. How do you pass data from parent to child component?

One-Line Definition

In React, data is passed from parent to child components using props.

Simple Understanding

React follows:

One-Way Data Flow

This means data moves:

Parent Component
↓
Child Component

Basic Example

function Child(props) {

  return (

    <h1>Hello {props.name}</h1>

  );

}

function App() {

  return (

    <Child name="Rizwan" />

  );

}

Output

Hello Rizwan

How It Works

Component Role
App Parent component
Child Child component

Data Passing Flow

Parent Creates Data
↓
Passes Data as Props
↓
Child Receives Props
↓
Child Renders UI

Using Destructuring

Instead of: 

props.name

We can destructure props.

Example

function Child({ name }) {

  return (

    <h1>Hello {name}</h1>

  );

}

Passing Multiple Values

<Child name="Rizwan" age={25} city="Jaipur" />

Receiving Multiple Props

function Child({ name, age, city }) {

  return (

    <div>

      <h2>{name}</h2>

      <p>{age}</p>

      <p>{city}</p>

    </div>

  );

}

Props Can Pass Different Data Types

Type Example
String name="Rizwan"
Number age={25}
Boolean isAdmin={true}
Array users={['A', 'B']}
Object user={{ name: 'Rizwan' }}
Function onClick={handleClick}

Passing Function as Props

function Button({ onPress }) {

  return (

    <button onClick={onPress}>
      Click
    </button>

  );

}

function App() {

  const handleClick = () => {

    console.log('Clicked');

  };

  return (

    <Button onPress={handleClick} />

  );

}

Props Are Read-Only

Child components should never modify props directly.

Wrong Example

// Wrong

props.name = 'Aman';

Why React Uses Parent → Child Flow

  • Predictable data flow
  • Easier debugging
  • Better component architecture
  • Improved maintainability

Real Project Use Cases

Feature Parent → Child Data
OTP popup OTP state and handlers
User profile User information
Theme system Theme settings
Reusable buttons Text and events

Important Interview Points

  • Props are used for parent-to-child communication.
  • React follows one-way data flow.
  • Props are read-only.
  • Functions can also be passed as props.
  • Destructuring improves readability.

Short Interview Answer

In React, data is passed from parent to child components using props, which allow components to receive dynamic values and functions through one-way data flow.

One-Line Interview Answer

React passes data from parent to child components using props.
