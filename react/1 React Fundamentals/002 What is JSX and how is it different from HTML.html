What is JSX?

JSX (JavaScript XML) is a syntax extension for JavaScript that allows developers to write HTML-like code inside React.

JSX Example

const element = <h1>Hello React</h1>;

JSX vs HTML

HTML JSX
Uses class Uses className
Uses for Uses htmlFor
Browser understands directly Converted by Babel

JavaScript Inside JSX

const name = "Rizwan";

<h1>Hello {name}</h1>

Short Interview Answer

JSX is an HTML-like syntax used in React to create UI components.
