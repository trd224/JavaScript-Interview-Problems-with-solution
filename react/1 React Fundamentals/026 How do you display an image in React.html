026. How do you display an image in React?

One-Line Definition

Images in React are displayed using the img tag with imported files or public folder paths.

Ways to Display Images in React

Method Usage
Import Image Best for src folder images
Public Folder Path Best for static public assets
External URL Images from internet/server

Method 1 - Import Image from src

Project Structure

src/

  images/
    logo.png

  App.js

App.js

import logo from './images/logo.png';

function App() {

  return (

    <img
      src={logo}
      alt="Logo"
    />

  );

}

Why This Method Is Recommended

  • Webpack optimizes images
  • Supports hashing and caching
  • Build system validates paths
  • Better performance

Method 2 - Using public Folder

Project Structure

public/

  images/
    logo.png

App.js

function App() {

  return (

    <img
      src="/images/logo.png"
      alt="Logo"
    />

  );

}

Method 3 - External URL

function App() {

  return (

    <img
      src="https://example.com/logo.png"
      alt="Logo"
    />

  );

}

Important img Attributes

Attribute Purpose
src Image path
alt Alternative text
width Image width
height Image height
className Apply CSS styles

Example with Styling

import logo from './images/logo.png';

function App() {

  return (

    <img
      src={logo}
      alt="Company Logo"
      width="200"
      className="logo"
    />

  );

}

Flow Diagram

Dynamic Image Example

function User() {

  const imageUrl =
    'https://example.com/user.jpg';

  return (

    <img
      src={imageUrl}
      alt="User"
    />

  );

}

Real Project Use Cases

Scenario Image Usage
User Profile Profile picture
Dashboard Icons and banners
E-commerce Product images
Authentication Portal Department logos

Common Mistake

In JSX, use className instead of class.

Wrong

<img class="logo" />

Correct

<img className="logo" />

Best Practice

Use src Import Use public Folder
Optimized images Static public assets
Component-related images favicon, logos, manifest
Webpack processing needed Direct browser access needed

Important Interview Points

  • React uses the img tag to display images.
  • Images can be imported from src.
  • Images can also come from public folder.
  • External image URLs are supported.
  • alt attribute is important for accessibility.

Short Interview Answer

Images in React are displayed using the img tag with imported files, public folder paths, or external URLs.

One-Line Interview Answer

React displays images using the img tag with local or external image paths.
