027. How do you apply CSS styles in React?

One-Line Definition

CSS styles in React can be applied using className, inline styles, CSS modules, or CSS libraries.

Most Common Methods

Method Usage
className External CSS files
Inline Styles Styles inside JSX
CSS Modules Component-scoped CSS
CSS Libraries Bootstrap, Tailwind, Material UI

Method 1 - Using className

React uses className instead of class because class is a JavaScript keyword.

App.css

.title {

  color: cyan;
  font-size: 32px;

}

App.js

import './App.css';

function App() {

  return (

    <h1 className="title">
      Hello React
    </h1>

  );

}

Output

Hello React

Important Note

Always use className in JSX, not class.

Method 2 - Inline Styling

React inline styles use JavaScript objects. 

function App() {

  const styleObj = {

    color: 'cyan',
    fontSize: '30px',
    backgroundColor: 'black'

  };

  return (

    <h1 style={styleObj}>
      Hello React
    </h1>

  );

}

Important Rules for Inline Styles

CSS React Inline Style
background-color backgroundColor
font-size fontSize
text-align textAlign

Why?

React inline styles use camelCase property names because they are JavaScript objects.

Method 3 - CSS Modules

CSS Modules prevent global CSS conflicts.

App.module.css

.title {

  color: cyan;

}

App.js

import styles from './App.module.css';

function App() {

  return (

    <h1 className={styles.title}>
      Hello React
    </h1>

  );

}

Benefits of CSS Modules

  • No class name conflicts
  • Component-specific styling
  • Better maintainability

Method 4 - Using CSS Libraries

Bootstrap Example

<button className="btn btn-primary">
  Save
</button>

Flow Diagram

CSS File / Style Object → React JSX → Browser Applies Styles

Dynamic Styling Example

function App() {

  const darkMode = true;

  return (

    <h1

      style={{
        color: darkMode ? 'white' : 'black',
        backgroundColor: darkMode ? 'black' : 'white'
      }}

    >

      Theme Example

    </h1>

  );

}

Important Interview Points

  • React uses className instead of class.
  • Inline styles use JavaScript objects.
  • Inline CSS properties use camelCase.
  • CSS Modules avoid global conflicts.
  • React supports all modern CSS approaches.

Short Interview Answer

CSS in React can be applied using className, inline styles, CSS modules, or external UI libraries like Bootstrap and Tailwind.
