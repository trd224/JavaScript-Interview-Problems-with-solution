001. What is React and why is it used?

One-Line Definition

React is a JavaScript library used to build fast, interactive, and reusable user interfaces, especially for single-page applications.

Why Is It Used?

  • Build reusable UI components
  • Update UI efficiently using the Virtual DOM
  • Create fast and dynamic web applications
  • Manage complex frontend applications easily
  • Improve code maintainability and scalability

Real Project Use Case

Suppose you are building an Aadhaar Authentication Portal. Different parts of the screen like Login Form, OTP Popup, Loader, QR Scanner, Error Popup, and User Dashboard can all be created as separate reusable React components.

Example Code

function App() {
  return (
    <div>
      <h1>Hello React</h1>
      <p>Welcome to React Application</p>
    </div>
  );
}

export default App;

Output

Hello React
Welcome to React Application

Important Interview Points

  • React is a library, not a full framework.
  • React follows component-based architecture.
  • React uses Virtual DOM for fast rendering.
  • React supports one-way data flow.
  • React is mainly used for frontend UI development.

Short Interview Answer

React is a JavaScript library used to create reusable and fast user interfaces using components and Virtual DOM.

One-Line Interview Answer

React is a component-based JavaScript library used for building fast and interactive user interfaces.
