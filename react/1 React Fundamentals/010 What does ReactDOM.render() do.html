010. What does ReactDOM.render() do?

One-Line Definition

ReactDOM.render() renders a React component into the Real DOM inside the browser.

Why Is It Used?

React components exist inside JavaScript memory.

To display those components on the webpage, React must render them into an actual HTML element in the browser.

ReactDOM.render() performs this task.

Basic Syntax

ReactDOM.render(
  component,
  targetElement
);

Example

import ReactDOM from 'react-dom';

function App() {

  return (
    <h1>Hello React</h1>
  );

}

ReactDOM.render(
  <App />,
  document.getElementById('root')
);

Output

Hello React

What Happens Internally?

Step What Happens
1 React creates the component Virtual DOM
2 ReactDOM.render() converts it into Real DOM
3 Content appears inside the browser

Target Element

Usually React applications render inside: 

<div id="root"></div>

This element exists in the main HTML file.

Flow Diagram

React Component → ReactDOM.render() → Browser DOM

Real Project Use Case

Suppose you build:

  • Authentication Portal
  • Dashboard
  • Admin Panel
  • Shopping Website

ReactDOM.render() loads the main React application into the webpage.

Important Note

ReactDOM.render() is mainly used in React 17 and older versions.

In React 18+, createRoot() is used instead.

React 18 Modern Syntax

import { createRoot } from 'react-dom/client';

const root = createRoot(
  document.getElementById('root')
);

root.render(
  <App />
);

ReactDOM.render() vs createRoot()

ReactDOM.render() createRoot()
Used in React 17 and older Used in React 18+
Older rendering API Modern concurrent rendering API
Limited performance features Supports new React 18 features

Important Interview Points

  • ReactDOM.render() renders React into the browser DOM.
  • It connects React with the webpage.
  • Usually renders into the root element.
  • Used mostly in React 17 and older.
  • React 18 uses createRoot().

Short Interview Answer

ReactDOM.render() renders a React component into the browser DOM by attaching it to a target HTML element.

One-Line Interview Answer

ReactDOM.render() displays React components inside the browser DOM.
