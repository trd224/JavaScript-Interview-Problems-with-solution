029. How do you add a class name conditionally in React?

One-Line Definition

Conditional class names in React are added using JavaScript expressions like ternary operators, && operators, or template literals.

Why Is Conditional Styling Needed?

Applications often need different styles based on:

  • User actions
  • State changes
  • Validation errors
  • Dark/light themes
  • Dynamic UI conditions

Method 1 - Using Ternary Operator

function App() {

  const isActive = true;

  return (

    <button

      className={
        isActive
          ? 'active'
          : 'inactive'
      }

    >

      Submit

    </button>

  );

}

Output

class="active"

How It Works

Condition Class Applied
isActive = true active
isActive = false inactive

Method 2 - Using && Operator

Useful when you only want to add a class if condition is true. 

function App() {

  const hasError = true;

  return (

    <input

      className={hasError && 'error'}

    />

  );

}

Output

class="error"

Important Note

If condition becomes false, className may become false or empty.

Method 3 - Using Template Literals

Most common modern approach. 

function App() {

  const darkMode = true;

  return (

    <div

      className={`container ${darkMode ? 'dark' : 'light'}`}

    >

      Theme Content

    </div>

  );

}

Output

class="container dark"

Why Template Literals Are Popular

  • Supports multiple classes
  • Cleaner syntax
  • Easy dynamic combinations
  • Good readability

Method 4 - Using Array Join

const classes = [

  'btn',

  isActive && 'active',

  disabled && 'disabled'

].filter(Boolean).join(' ');

<button className={classes}>
  Save
</button>

Using clsx or classnames Library

Large projects commonly use helper libraries. 

import clsx from 'clsx';

<button

  className={clsx(
    'btn',
    isActive && 'active',
    disabled && 'disabled'
  )}

>

  Save

</button>

Flow Diagram

Real Project Use Cases

Scenario Conditional Class
Form validation error class
Dark mode dark/light class
Selected menu active class
Disabled button disabled class
OTP status success/error class

Best Practice

Scenario Recommended Approach
Simple condition Ternary operator
Single optional class && operator
Multiple classes Template literals
Large applications clsx/classnames library

Important Interview Points

  • Conditional class names use JavaScript expressions.
  • Ternary operators are very common.
  • Template literals are best for multiple classes.
  • clsx/classnames libraries simplify dynamic classes.
  • React updates className dynamically during re-rendering.

Short Interview Answer

Conditional class names in React are added using ternary operators, && operators, template literals, or helper libraries like clsx.

One-Line Interview Answer

React conditionally applies class names using JavaScript expressions inside className.
