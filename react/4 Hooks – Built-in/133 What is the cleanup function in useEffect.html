133. What is the cleanup function in useEffect?

One-Line Definition

A cleanup function is a function returned from useEffect that cleans up side effects before re-running the effect or when the component unmounts.

Simple Understanding

Cleanup removes old effects that are no longer needed.

Examples

  • Clearing timers
  • Removing event listeners
  • Closing WebSocket connections
  • Stopping subscriptions

Main Idea

useEffect Runs
↓
Side Effect Starts
↓
Cleanup Removes Old Effect

Basic Syntax

useEffect(() => {

  // Side effect

  return () => {

    // Cleanup code

  };

}, []);

Basic Cleanup Example

useEffect(() => {

  console.log('Effect Started');

  return () => {

    console.log('Cleanup Called');

  };

}, []);

What Happens?

  • Effect runs after first render
  • Cleanup runs when component unmounts

When Cleanup Function Runs

Situation Cleanup Runs?
Component unmounts ✅ Yes
Dependencies change ✅ Yes
Before next effect runs ✅ Yes
Initial render ❌ No

Cleanup with Dependencies

import { useEffect, useState } from 'react';

function App() {

  const [count, setCount] = useState(0);

  useEffect(() => {

    console.log('Effect Started', count);

    return () => {

      console.log('Cleanup Old Effect', count);

    };

  }, [count]);

}

What Happens?

count Changes
↓
Old Cleanup Runs
↓
New Effect Runs

Timer Cleanup Example

import { useEffect, useState } from 'react';

function App() {

  const [count, setCount] = useState(0);

  useEffect(() => {

    const timer = setInterval(() => {

      setCount((prev) => prev + 1);

    }, 1000);

    return () => clearInterval(timer);

  }, []);

  return <h1>{count}</h1>;

}

Why Cleanup Is Needed?

Without cleanup, the timer continues running even after component removal.

Event Listener Cleanup Example

useEffect(() => {

  const handleResize = () => {

    console.log(window.innerWidth);

  };

  window.addEventListener('resize', handleResize);

  return () => {

    window.removeEventListener('resize', handleResize);

  };

}, []);

Why Cleanup Is Important?

  • Prevents duplicate listeners
  • Prevents memory leaks
  • Improves performance

Subscription Cleanup Example

useEffect(() => {

  socket.connect();

  return () => {

    socket.disconnect();

  };

}, []);

Use Case

WebSocket, SignalR, RxJS subscriptions, chat systems, live updates.

What Happens Without Cleanup?

  • Memory leaks
  • Duplicate timers
  • Duplicate listeners
  • Unexpected behavior
  • Performance problems

Main Cleanup Flow

Old Effect
↓
Cleanup Removes Old Resources
↓
New Effect Starts

Dependency Behavior with Cleanup

Code Behavior
useEffect(fn, []) Cleanup on unmount only
useEffect(fn, [count]) Cleanup before re-run and on unmount
useEffect(fn) Cleanup before every re-render
