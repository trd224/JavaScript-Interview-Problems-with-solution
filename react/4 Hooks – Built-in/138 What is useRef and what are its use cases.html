138. What is useRef and what are its use cases?

One-Line Definition

useRef is a React hook used to store mutable values and access DOM elements without causing re-renders.

Simple Understanding

useRef creates a persistent object that stays the same between renders.

Main Features

  • Stores mutable values
  • Accesses DOM elements
  • Does not trigger re-render
  • Value persists across renders

Main Idea

useRef()
↓
Creates Ref Object
↓
Stored in ref.current
↓
Value Persists Between Renders

Basic Syntax

const ref = useRef(initialValue);

Important Structure

const ref = useRef(0);

console.log(ref);

// Output

{
  current: 0
}

Important Point

useRef stores data inside the current property.

DOM Access Example

components/InputFocus.jsx

import { useRef } from 'react';

function InputFocus() {

  const inputRef = useRef(null);

  const focusInput = () => {

    inputRef.current.focus();

  };

  return (

    <>

      <input ref={inputRef} />

      <button onClick={focusInput}>

        Focus Input

      </button>

    </>

  );

}

export default InputFocus;

App Component

App.jsx

import InputFocus from './components/InputFocus';

function App() {

  return <InputFocus />;

}

export default App;

What Happens?

  • Button click triggers focusInput()
  • inputRef.current points to input DOM element
  • Input gets focused

Store Mutable Value Example

import { useRef } from 'react';

function Counter() {

  const countRef = useRef(0);

  const increase = () => {

    countRef.current++;

    console.log(countRef.current);

  };

  return (

    <button onClick={increase}>

      Increase

    </button>

  );

}

export default Counter;

Output

1
2
3

Important Difference Between useRef and useState

useRef useState
Does not re-render component Triggers re-render
Stores mutable value Stores reactive state
Uses current property Uses setter function

Why useRef Does Not Re-Render?

React does not track changes inside ref.current for rendering updates.

Persist Value Between Renders

import { useRef, useState } from 'react';

function App() {

  const renderCount = useRef(0);

  const [count, setCount] = useState(0);

  renderCount.current++;

  return (

    <>

      <h1>Render Count: {renderCount.current}</h1>

      <button onClick={() => setCount(count + 1)}>

        {count}

      </button>

    </>

  );

}

export default App;

Timer Example

const timerRef = useRef(null);

timerRef.current = setInterval(() => {

  console.log('Running');

}, 1000);

clearInterval(timerRef.current);

Previous State Tracking Example

import {

  useEffect,
  useRef,
  useState

} from 'react';

function App() {

  const [count, setCount] = useState(0);

  const previousCount = useRef();

  useEffect(() => {

    previousCount.current = count;

  }, [count]);

  return (

    <>

      <h2>Current: {count}</h2>

      <h2>Previous: {previousCount.current}</h2>

      <button onClick={() => setCount(count + 1)}>

        Increase

      </button>

    </>

  );

}

export default App;
