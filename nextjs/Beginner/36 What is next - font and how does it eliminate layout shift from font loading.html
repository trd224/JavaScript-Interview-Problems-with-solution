What is next/font and how does it eliminate layout shift from font loading?

next/font is a built-in Next.js feature that optimizes font loading by downloading, self-hosting, and serving fonts directly from our application.

next/font helps eliminate layout shift caused by font loading and improves Core Web Vitals such as CLS (Cumulative Layout Shift).

Why Do We Need next/font?

Traditionally, web fonts are loaded from external servers such as Google Fonts.

Traditional Method
<link
  href="https://fonts.googleapis.com/css2?family=Roboto"
  rel="stylesheet"
/>

This causes an extra network request before the font becomes available.

The Layout Shift Problem

Suppose our page contains:

HTML
<h1>Welcome To Next.js</h1>

When the page loads:

Browser Uses
Fallback Font

Welcome To Next.js

After the font downloads:

Step 2

The text width and height may change.

This causes content movement on the screen.

How next/font Solves This

Next.js downloads the font during the build process and serves it directly from our application.

Using Google Fonts

app/layout.js
import { Inter } from 'next/font/google';

const inter = Inter({
  subsets: ['latin']
});

export default function RootLayout({
  children
}) {

  return (
    <html lang="en">
      <body className={inter.className}>
        {children}
      </body>
    </html>
  );

}

Using Local Fonts

app/layout.js
import localFont from 'next/font/local';

const myFont = localFont({
  src: './fonts/MyFont.woff2'
});

export default function RootLayout({
  children
}) {

  return (
    <html>
      <body className={myFont.className}>
        {children}
      </body>
    </html>
  );

}

Font Display Optimization

Next.js automatically applies font loading optimizations.

Example
const inter = Inter({
  subsets: ['latin'],
  display: 'swap'
});

This helps improve perceived loading performance.

Benefits of next/font

  • Eliminates layout shift from font loading.
  • Self-hosts fonts automatically.
  • Removes extra font requests.
  • Improves CLS score.
  • Improves page performance.
  • Works with Google Fonts and local fonts.

const inter = Inter({
  subsets: ['latin']
});

  Component,
  pageProps
}) {

  return (
    <main className={inter.className}>
      <Component {...pageProps} />
    </main>
  );

}

Interview Tip

A common misconception is that next/font only downloads fonts. In reality, it downloads, self-hosts, optimizes, and serves fonts in a way that minimizes layout shifts and improves performance.
One-Line Definition:
next/font is a built-in Next.js font optimization feature that self-hosts fonts and eliminates layout shift by ensuring optimized fonts are available immediately when the page renders.
