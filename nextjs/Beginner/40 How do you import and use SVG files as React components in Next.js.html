40. How do you import and use SVG files as React components in Next.js?

SVG files can be used in Next.js in multiple ways, but to use an SVG as a React component we typically use SVGR.

Using SVG as a React component allows us to pass props, apply CSS classes, change colors, control width and height, and reuse SVGs like normal React components.

Method 1: Use SVG from the public Folder

The simplest method is storing SVG files in the public folder.

Project Structure
public/
└── logo.svg
Usage
export default function Home() {

  return (
    <img
      src="/logo.svg"
      alt="Logo"
    />
  );

}

This works, but the SVG behaves like a normal image.

We cannot easily change SVG properties using React props.

Method 2: Use next/image

Example
import Image from 'next/image';

export default function Home() {

  return (
    <Image
      src="/logo.svg"
      alt="Logo"
      width={100}
      height={100}
    />
  );

}

This is useful when SVG behaves like a static image.

Method 3: Import SVG as a React Component (Recommended)

To use SVG files as React components, we need SVGR.

Step 1: Install SVGR

Terminal
npm install -D @svgr/webpack

Step 2: Configure next.config.js

next.config.js
const nextConfig = {

  webpack(config) {

    config.module.rules.push({
      test: /\.svg$/i,
      issuer: /\.[jt]sx?$/,
      use: ['@svgr/webpack']
    });

    return config;

  }

};

module.exports = nextConfig;

Step 3: Create SVG File

assets/logo.svg
<svg
  xmlns="http://www.w3.org/2000/svg"
  viewBox="0 0 100 100"
>
  <circle
    cx="50"
    cy="50"
    r="40"
    fill="green"
  />
</svg>

Step 4: Import SVG as Component

app/page.js
import Logo from '../assets/logo.svg';

export default function Home() {

  return (
    <Logo />
  );

}

Passing Props to SVG Components

Example
import Logo from '../assets/logo.svg';

export default function Home() {

  return (
    <Logo
      width={200}
      height={200}
    />
  );

}

Applying CSS Classes

Example
import Logo from '../assets/logo.svg';

export default function Home() {

  return (
    <Logo className="logo" />
  );

}
CSS
.logo {
  width: 200px;
  height: 200px;
}

Changing SVG Color

If the SVG uses currentColor, we can control its color through CSS.

SVG
<svg>
  <path
    fill="currentColor"
  />
</svg>
Usage
<Logo
  style={{
    color: 'red'
  }}
/>

Important Note

A common misconception is that SVG files can be imported as React components by default. In reality, Next.js requires SVGR configuration before SVG files can be imported and used as React components.

Interview Tip

If an SVG only needs to be displayed, storing it in the public folder is usually enough. If we need dynamic colors, props, event handlers, or reusable icons, importing it as a React component using SVGR is the better approach.
One-Line Definition:
SVG files can be used as React components in Next.js by configuring SVGR, allowing us to import SVG files and use them like normal React components with props, styles, and event handlers.
