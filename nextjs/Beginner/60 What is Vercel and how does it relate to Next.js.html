60. What is Vercel and how does it relate to Next.js?

Vercel is a cloud hosting platform created by the company that also created Next.js. It provides the easiest and most optimized way to deploy, host, and scale Next.js applications.

Vercel is the company behind Next.js, and Next.js is designed to work seamlessly with the Vercel platform.

What is Vercel?

Vercel is a cloud platform that automatically builds, deploys, and hosts web applications.

It supports:

  • Next.js
  • React
  • Vue
  • Angular
  • Svelte
  • Static Websites

Why Is Vercel Recommended for Next.js?

  • Zero configuration deployment.
  • Automatic builds.
  • Automatic scaling.
  • Global CDN.
  • Built-in SSR support.
  • Built-in ISR support.
  • Built-in Edge Functions.
  • Automatic HTTPS.

Example

ISR Page
export default async function Page() {

  const products = await fetch(
    'https://api.example.com/products',
    {
      next: {
        revalidate: 60
      }
    }
  ).then(res => res.json());

  return (
    <div>
      {products.length}
    </div>
  );

}

When deployed on Vercel, ISR caching and regeneration work automatically without additional configuration.

Can Next.js Run Without Vercel?

Yes.

Next.js can be deployed on:

  • Windows Server
  • Linux Server
  • Docker
  • AWS
  • Azure
  • Google Cloud
  • DigitalOcean
  • Nginx + PM2
Vercel is not required to use Next.js. It is simply the easiest and most optimized hosting platform for Next.js applications.
