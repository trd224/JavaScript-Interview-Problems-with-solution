How do you generate a sitemap.xml using the App Router file convention?

In the App Router, we create a special sitemap.js (or sitemap.ts) file. Next.js automatically generates a sitemap.xml file from the data returned by this file.

The sitemap.js file is a built-in App Router file convention used to generate SEO-friendly sitemap.xml files.

What is a Sitemap?

A sitemap helps search engines discover and index pages on our website.

Common search engines that use sitemaps:

  • Google
  • Bing
  • Yahoo
  • DuckDuckGo

Basic Sitemap Example

app/sitemap.js
export default function sitemap() {
  return [
    {
      url: 'https://example.com',
      lastModified: new Date()
    },
    {
      url: 'https://example.com/about',
      lastModified: new Date()
    },
    {
      url: 'https://example.com/contact',
      lastModified: new Date()
    }
  ];
}

Generated sitemap.xml

Output
<urlset>

  <url>
    <loc>https://example.com</loc>
  </url>

  <url>
    <loc>https://example.com/about</loc>
  </url>

  <url>
    <loc>https://example.com/contact</loc>
  </url>

</urlset>

Dynamic Sitemap Example

For dynamic routes such as blog posts or products, we can fetch data and generate sitemap entries dynamically.

app/sitemap.js
async function getProducts() {
  const res = await fetch(
    'https://dummyjson.com/products'
  );

  const data = await res.json();

  return data.products;
}

export default async function sitemap() {

  const products = await getProducts();

  const productUrls = products.map(product => ({
    url: `https://example.com/products/${product.id}`,
    lastModified: new Date()
  }));

  return [
    {
      url: 'https://example.com',
      lastModified: new Date()
    },

    ...productUrls
  ];

}

Additional Properties

Each sitemap entry can include additional SEO information.

Example
export default function sitemap() {
  return [
    {
      url: 'https://example.com',
      lastModified: new Date(),
      changeFrequency: 'daily',
      priority: 1
    }
  ];
}

Generated XML

Output
<url>
  <loc>https://example.com</loc>
  <lastmod>2026-06-02</lastmod>
  <changefreq>daily</changefreq>
  <priority>1.0</priority>
</url>

Why Use sitemap.js?

  • Better SEO.
  • Automatic XML generation.
  • Supports dynamic routes.
  • Built into Next.js.
  • No custom API route required.

Interview Tip

In the App Router, creating a sitemap.js file automatically generates a sitemap.xml endpoint. The function returns an array of URLs, and Next.js converts it into valid XML.
One-Line Definition:
The App Router generates sitemap.xml by using a special sitemap.js file that returns an array of sitemap entries, which Next.js automatically converts into XML.
