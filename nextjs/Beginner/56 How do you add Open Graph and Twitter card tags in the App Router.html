How do you add Open Graph and Twitter card tags in the App Router?

In the App Router, Open Graph and Twitter Card tags are added using the metadata export or generateMetadata().

Next.js automatically generates the required meta tags from the openGraph and twitter properties inside the Metadata API.

Why Use Open Graph and Twitter Cards?

These tags control how links appear when shared on social media platforms.

  • Facebook
  • LinkedIn
  • WhatsApp
  • X (Twitter)
  • Telegram
  • Discord

Basic Open Graph Example

app/about/page.js
export const metadata = {
  title: 'About Us',
  description: 'Learn more about our company.',

  openGraph: {
    title: 'About Us',
    description: 'Learn more about our company.',
    url: 'https://example.com/about',
    siteName: 'My Website',
    images: [
      {
        url: 'https://example.com/og-image.jpg',
        width: 1200,
        height: 630
      }
    ],
    locale: 'en_US',
    type: 'website'
  }
};

export default function About() {
  return <h1>About Page</h1>;
}

Dynamic Open Graph Example

When metadata depends on fetched data, use generateMetadata().

app/products/[id]/page.js
async function getProduct(id) {
  const res = await fetch(`https://dummyjson.com/products/${id}`);
  return res.json();
}

export async function generateMetadata({ params }) {

  const { id } = await params;

  const product = await getProduct(id);

  return {
    title: product.title,
    description: product.description,

    openGraph: {
      title: product.title,
      description: product.description,
      images: [product.thumbnail]
    },

    twitter: {
      card: 'summary_large_image',
      title: product.title,
      description: product.description,
      images: [product.thumbnail]
    }
  };

}

export default async function Page({ params }) {

  const { id } = await params;

  const product = await getProduct(id);

  return (
    <div>
      <h1>{product.title}</h1>
      <p>{product.description}</p>
    </div>
  );

}

Important Note

Open Graph tags are primarily used by Facebook, LinkedIn, WhatsApp, Telegram, and Discord, while Twitter Card tags are used by X (Twitter).

Interview Tip

In the App Router, Open Graph and Twitter Card tags are configured using the openGraph and twitter properties inside the Metadata API. Next.js automatically converts them into the appropriate meta tags.
One-Line Definition:
Open Graph and Twitter Card tags are added in the App Router through the Metadata API using the openGraph and twitter properties, allowing social platforms to display rich link previews.
