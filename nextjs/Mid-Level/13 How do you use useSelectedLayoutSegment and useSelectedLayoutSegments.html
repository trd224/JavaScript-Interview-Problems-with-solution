How do you use useSelectedLayoutSegment and useSelectedLayoutSegments?

useSelectedLayoutSegment() and useSelectedLayoutSegments() are Next.js App Router hooks that allow us to read the active route segment below the current layout.

These hooks are mainly used for active sidebar links, tabs, breadcrumbs, and dashboard navigation.

1. useSelectedLayoutSegment()

useSelectedLayoutSegment() returns only the first active child segment below the current layout.

Folder Structure
app/
└── dashboard/
    ├── layout.tsx
    ├── users/
    │   └── page.tsx
    ├── reports/
    │   └── page.tsx
    └── settings/
        └── page.tsx
dashboard/layout.tsx
import Sidebar from './Sidebar';

export default function DashboardLayout({ children }) {
  return (
    <div>
      <Sidebar />
      <main>{children}</main>
    </div>
  );
}
dashboard/Sidebar.tsx
'use client';

import Link from 'next/link';
import { useSelectedLayoutSegment } from 'next/navigation';

export default function Sidebar() {
  const segment = useSelectedLayoutSegment();

  return (
    <nav>
      <Link className={segment === 'users' ? 'active' : ''} href="/dashboard/users">Users</Link>
      <Link className={segment === 'reports' ? 'active' : ''} href="/dashboard/reports">Reports</Link>
      <Link className={segment === 'settings' ? 'active' : ''} href="/dashboard/settings">Settings</Link>
    </nav>
  );
}

Current URL

URL
/dashboard/reports

Returned Value

Result
segment = 'reports'

Final Sidebar UI

Output
Users
► Reports
Settings

Because the active child route below dashboard/layout.tsx is reports, the Reports link gets the active class.

2. useSelectedLayoutSegments()

useSelectedLayoutSegments() returns all active segments below the current layout as an array.

Folder Structure
app/
└── dashboard/
    ├── layout.tsx
    └── reports/
        └── monthly/
            └── sales/
                └── page.tsx
dashboard/Breadcrumbs.tsx
'use client';

import { useSelectedLayoutSegments } from 'next/navigation';

export default function Breadcrumbs() {
  const segments = useSelectedLayoutSegments();

  return (
    <div>
      {segments.map(segment => (
        <span key={segment}>{segment} / </span>
      ))}
    </div>
  );
}

Current URL

URL
/dashboard/reports/monthly/sales

Returned Value

Result
segments = ['reports', 'monthly', 'sales']

Final Breadcrumb UI

Output
reports / monthly / sales /

Main Difference

Hook Returns Example URL Result
useSelectedLayoutSegment() First active child segment /dashboard/reports/monthly/sales reports
useSelectedLayoutSegments() All active child segments /dashboard/reports/monthly/sales ['reports', 'monthly', 'sales']

Important Point

Both hooks work relative to the layout where they are used. If they are used inside dashboard/layout.tsx, they read only the route segments below /dashboard.

Easy Memory Trick

Remember
useSelectedLayoutSegment()
→ one segment

useSelectedLayoutSegments()
→ all segments
One-Line Definition:
useSelectedLayoutSegment() returns the first active child route segment below a layout, while useSelectedLayoutSegments() returns all active child route segments below that layout.
