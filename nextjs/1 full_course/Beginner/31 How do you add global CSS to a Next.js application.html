31. How do you add global CSS to a Next.js application?

Global CSS is CSS that applies to the entire application instead of a single component.

Global CSS allows us to define styles such as fonts, resets, colors, layouts, and utility classes that can be used throughout our application.

Why Do We Need Global CSS?

Some styles should be available everywhere.

Examples:
  • Font Family
  • CSS Reset
  • Body Background
  • Text Colors
  • Global Utility Classes

Pages Router

In the Pages Router, global CSS is imported inside pages/_app.js.

Step 1: Create Global CSS File

styles/globals.css
body {
  font-family: Arial, sans-serif;
  background: #f5f5f5;
}

h1 {
  color: blue;
}

Step 2: Import Global CSS

pages/_app.js
import '../styles/globals.css';

export default function MyApp({ Component, pageProps }) {

  return (
    <Component {...pageProps} />
  );

}

Project Structure

Pages Router
pages/
├── _app.js
├── index.js

styles/
└── globals.css

App Router

In the App Router, global CSS is imported inside app/layout.js.

Step 1: Create Global CSS File

app/globals.css
body {
  font-family: Arial, sans-serif;
  background: #f5f5f5;
}

h1 {
  color: blue;
}

Step 2: Import Global CSS

app/layout.js
import './globals.css';

export default function RootLayout({ children }) {

  return (
    <html lang="en">
      <body>
        {children}
      </body>
    </html>
  );

}

Project Structure

App Router
app/
├── layout.js
├── page.js
└── globals.css

What Happens Internally?

Pages Router
globals.css
      │
      ▼
Imported In _app.js
      │
      ▼
Available To All Pages
App Router
globals.css
      │
      ▼
Imported In layout.js
      │
      ▼
Available To Entire App

Can We Import Global CSS Anywhere?

No.

Global CSS can only be imported in special root files.

Router Import Location
Pages Router pages/_app.js
App Router app/layout.js

Incorrect Example

components/Header.js
import './globals.css';

This will cause a build error.

Global CSS vs CSS Modules

Feature Global CSS CSS Modules
Scope Entire Application Single Component
Class Name Collision Possible Prevented
Use Case Global Styles Component Styles

Real Example

app/globals.css
body {
  margin: 0;
  font-family: Arial, sans-serif;
}

.container {
  max-width: 1200px;
  margin: auto;
}
app/page.js
export default function Home() {

  return (
    <div className="container">
      <h1>Home Page</h1>
    </div>
  );

}

The container class is available because it was defined globally.

Interview Tip

A common misconception is that global CSS can be imported into any component. In reality, Next.js only allows global CSS imports in pages/_app.js (Pages Router) or app/layout.js (App Router).
One-Line Definition:
Global CSS in Next.js is added by creating a global stylesheet and importing it in pages/_app.js (Pages Router) or app/layout.js (App Router), making the styles available throughout our application.
