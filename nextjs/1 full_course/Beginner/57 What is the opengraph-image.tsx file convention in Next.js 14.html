57. What is the opengraph-image.tsx File Convention in Next.js 14?

The opengraph-image.tsx file convention allows us to dynamically generate Open Graph images for pages in the App Router.

Instead of creating image files manually, Next.js can generate Open Graph images on demand using React and the ImageResponse API.

Why Do We Need Open Graph Images?

When a page is shared on social media, platforms display a preview card.

  • Facebook
  • LinkedIn
  • WhatsApp
  • Telegram
  • Discord
  • X (Twitter)

The Open Graph image is the large preview image shown in those cards.

Without opengraph-image.tsx

metadata
export const metadata = {
  openGraph: {
    images: [
      'https://example.com/og-image.jpg'
    ]
  }
};

We must manually create and manage image files.

With opengraph-image.tsx

app/opengraph-image.tsx
import { ImageResponse } from 'next/og';

export const size = {
  width: 1200,
  height: 630
};

export const contentType = 'image/png';

export default function Image() {

  return new ImageResponse(
    (
      <div
        style={{
          fontSize: 80,
          background: 'black',
          color: 'white',
          width: '100%',
          height: '100%',
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'center'
        }}
      >
        My Website
      </div>
    )
  );

}

File Location

Project Structure
app
├── layout.js
├── page.js
├── opengraph-image.tsx
└── products
    ├── [id]
    │   ├── page.js
    │   └── opengraph-image.tsx

Each route can have its own Open Graph image.

Route-Specific Open Graph Images

For example:

app/products/[id]/opengraph-image.tsx
import { ImageResponse } from 'next/og';

export default async function Image({
  params
}) {

  const { id } = await params;

  const product = await fetch(
    `https://dummyjson.com/products/${id}`
  ).then(res => res.json());

  return new ImageResponse(
    (
      <div
        style={{
          width: '100%',
          height: '100%',
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'center',
          fontSize: 70
        }}
      >
        {product.title}
      </div>
    )
  );

}

Each product page can generate a unique social sharing image.

Important Note

The opengraph-image.tsx file automatically becomes the Open Graph image for its route. No manual image creation is required.

Interview Tip

The opengraph-image.tsx convention uses the ImageResponse API from next/og to dynamically generate Open Graph images using React components.

The opengraph-image.tsx file convention in Next.js 14 dynamically generates Open Graph images for a route using React and the ImageResponse API.
