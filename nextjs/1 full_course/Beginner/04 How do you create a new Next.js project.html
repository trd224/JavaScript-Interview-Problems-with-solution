4. How do you create a new Next.js project?

A new Next.js project can be created using the official create-next-app command-line tool.

The create-next-app tool automatically sets up a Next.js project with all required dependencies and configuration.

Step 1: Verify Node.js Installation

Before creating a Next.js project, ensure Node.js is installed.

Check Node.js Version
node -v

Example Output:

Output
v22.15.0

Step 2: Create a New Next.js Project

Create Project
npx create-next-app@latest my-app

Replace my-app with your project name.

Step 3: Select Project Options

The installer will ask several questions.

Interactive Setup
✔ Would you like to use TypeScript? › Yes
✔ Would you like to use ESLint? › Yes
✔ Would you like to use Tailwind CSS? › Yes
✔ Would you like your code inside a src/ directory? › Yes
✔ Would you like to use App Router? › Yes
✔ Would you like to use Turbopack? › Yes
✔ Would you like to customize the import alias? › No

Step 4: Move Into Project Folder

Navigate to Project
cd my-app

Step 5: Start Development Server

Run Development Server
npm run dev

Example Output:

Output
▲ Next.js

Local:   http://localhost:3000
Ready in 2.1s

Step 6: Open the Application

Open the following URL in your browser:

Application URL
http://localhost:3000

Default Project Structure

Next.js Project Structure
my-app/
├── app/
│   ├── page.js
│   ├── layout.js
│
├── public/
│
├── node_modules/
│
├── package.json
├── next.config.js
└── README.md

Important Files

File / Folder Purpose
app/ Contains application routes and pages
page.js Represents a route/page
layout.js Shared layout for pages
public/ Stores static assets
package.json Project dependencies and scripts
next.config.js Next.js configuration

Create a Project with TypeScript

TypeScript Project
npx create-next-app@latest my-app --typescript

Create a Project Without Prompts

Automatic Setup
npx create-next-app@latest my-app \
  --typescript \
  --tailwind \
  --eslint \
  --src-dir \
  --app

Development Workflow

  1. Create the project using create-next-app.
  2. Navigate to the project folder.
  3. Run npm run dev.
  4. Open http://localhost:3000.
  5. Edit files inside the app folder.
  6. See changes instantly using Fast Refresh.
One-Line Definition:
A new Next.js project is created using the npx create-next-app@latest project-name command, which automatically sets up a complete Next.js application with routing, configuration, and development tools.
