30. Server Actions became stable in which Next.js version?

Server Actions became stable in Next.js 14.

Interview Answer:
Server Actions were introduced as an experimental feature in Next.js 13 and became stable in Next.js 14.

Timeline

Version Status
Next.js 13 ⚠️ Experimental
Next.js 14 ✅ Stable
Next.js 15 ✅ Improved and Enhanced

Next.js 13

When Server Actions were first introduced in Next.js 13, they were experimental and had to be enabled manually.

next.config.js
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverActions: true
  }
};

module.exports = nextConfig;

Without this flag, Server Actions could not be used.

Next.js 14

In Next.js 14, Server Actions became stable and no experimental flag was required.

Server Action
'use server';

export async function createUser() {

  console.log('Server Action');

}

Everything works out of the box.

Next.js 15

Next.js 15 continues to support Server Actions and improves integration with:

  • React 19
  • Form Actions
  • useActionState()
  • useFormStatus()
  • Caching APIs
  • Revalidation APIs
