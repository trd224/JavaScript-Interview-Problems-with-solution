32. What is useActionState (React 19) and how does it replace useFormState in Next.js 15?

useActionState() is a React 19 hook that allows us to manage the state returned from a Server Action.

Simple Meaning:
useActionState() helps our form receive success messages, error messages, and validation results from a Server Action.

Why Do We Need useActionState?

When a form submits to a Server Action, the Server Action may return some result.

  • Success message
  • Error message
  • Validation error
  • Updated form state

useActionState() stores that returned result in component state.

Flow
User Submits Form
        ↓
Server Action Runs
        ↓
Server Action Returns Message
        ↓
useActionState Stores Message
        ↓
UI Shows Message

Old API: useFormState

Earlier, Next.js used useFormState() from react-dom.

Old Import
import { useFormState } from 'react-dom';

This was mainly focused on form state.

New API: useActionState

In React 19 and Next.js 15, useActionState() is used from react.

New Import
import { useActionState } from 'react';
Main Difference:
useFormState() was the older API, while useActionState() is the newer React 19 API.

Basic Syntax

Syntax
const [state, formAction] = useActionState(actionFunction, initialState);
Part Meaning
state Latest value returned from the Server Action
formAction Function passed to the form's action attribute
actionFunction Server Action function
initialState Default state before form submit

Complete Example

1. Server Action

app/actions.ts
'use server';

export async function createUser(previousState: string, formData: FormData) {
  const name = formData.get('name');

  if (!name) return 'Name is required';

  return 'User created successfully';
}

2. Client Component

app/UserForm.tsx
'use client';

import { useActionState } from 'react';
import { createUser } from './actions';

export default function UserForm() {
  const [message, formAction] = useActionState(createUser, '');

  return (
    <form action={formAction}>
      <input name="name" placeholder="Enter name" />
      <button>Save</button>
      <p>{message}</p>
    </form>
  );
}

How It Works

When the form is submitted, formAction calls the Server Action.

Important Line
const [message, formAction] = useActionState(createUser, '');

Here:

Value Meaning
message Stores the returned message from createUser()
formAction Used inside <form action={formAction}>

Important: Action Function Receives Previous State

When we use useActionState(), React automatically passes the previous state as the first argument.

Server Action Signature
export async function createUser(previousState: string, formData: FormData) {
  return 'Updated message';
}

So the action receives: 

1st argument → previousState
2nd argument → formData

Real Validation Example

app/actions.ts
'use server';

type State = {
  success: boolean;
  message: string;
};

export async function register(previousState: State, formData: FormData): Promise<State> {
  const email = formData.get('email');

  if (!email) return { success: false, message: 'Email is required' };

  return { success: true, message: 'Registration successful' };
}
app/RegisterForm.tsx
'use client';

import { useActionState } from 'react';
import { register } from './actions';

const initialState = { success: false, message: '' };

export default function RegisterForm() {
  const [state, formAction] = useActionState(register, initialState);

  return (
    <form action={formAction}>
      <input name="email" placeholder="Enter email" />
      <button>Register</button>
      <p>{state.message}</p>
    </form>
  );
}

useFormState vs useActionState

Feature useFormState useActionState
Import From react-dom react
Used In Older Next.js versions Next.js 15 / React 19
Purpose Manage form action state Manage action result state
Status Older API Recommended API

Easy Memory Trick

Remember
Server Action
      ↓
Returns Message / Error
      ↓
useActionState()
      ↓

One-Line Definition:
useActionState() is a React 19 hook that stores the result returned from a Server Action, and it replaces the older useFormState() API used in earlier Next.js versions.
