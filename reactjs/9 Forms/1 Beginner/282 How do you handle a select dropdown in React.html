282. How Do You Handle a Select Dropdown in React?

One-Line Definition

A select dropdown in React is handled using useState and the onChange event to store the selected option in React state.

Why Use State for a Dropdown?

React typically treats dropdowns as controlled components.

The selected option is stored in state, and the dropdown displays the value from that state.

Basic Example

import { useState } from 'react';

function CitySelector() {

  const [city, setCity] = useState('');

  return (

    <select value={city} onChange={e => setCity(e.target.value)}>

      <option value="">Select City</option>

      <option value="Jaipur">Jaipur</option>

      <option value="Delhi">Delhi</option>

      <option value="Kota">Kota</option>

    </select>

  );

}

How It Works

const [city, setCity] = useState('');

State stores the currently selected option. 

<select value={city} onChange={e => setCity(e.target.value)}>
Property Purpose
value Binds state to the dropdown
onChange Updates state when selection changes

Displaying the Selected Value

import { useState } from 'react';

function CitySelector() {

  const [city, setCity] = useState('');

  return (

    <div>

      <select value={city} onChange={e => setCity(e.target.value)}>

        <option value="">Select City</option>

        <option value="Jaipur">Jaipur</option>

        <option value="Delhi">Delhi</option>

      </select>

      <p>Selected City: {city}</p>

    </div>

  );

}

Using a Separate Handler Function

const [city, setCity] = useState('');

function handleChange(e) {

  setCity(e.target.value);

}

<select value={city} onChange={handleChange}>

  ...

</select>

Dropdown Inside Form Data Object

const [formData, setFormData] = useState({
  city: ''
});

function handleChange(e) {

  setFormData({
    ...formData,
    city: e.target.value
  });

}

Complete Form Example

import { useState } from 'react';

function CityForm() {

  const [city, setCity] = useState('');

  function handleSubmit(e) {

    e.preventDefault();

    console.log(city);

  }

  return (

    <form onSubmit={handleSubmit}>

      <select value={city} onChange={e => setCity(e.target.value)}>

        <option value="">Select City</option>

        <option value="Jaipur">Jaipur</option>

        <option value="Delhi">Delhi</option>

      </select>

      <button type="submit">Submit</button>

    </form>

  );

}

export default CityForm;

Important Interview Points

  • Select dropdowns are usually controlled components.
  • useState stores the selected value.
  • e.target.value returns the selected option.
  • value binds state to the dropdown.
  • onChange updates state when selection changes.

Short Interview Answer

A select dropdown in React is handled using state and the onChange event, where the selected option is stored in React state and updated whenever the user changes the selection.

One-Line Interview Answer

React handles a select dropdown by storing the selected value in state and updating it through the onChange event.
