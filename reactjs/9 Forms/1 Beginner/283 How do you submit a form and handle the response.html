283. How Do You Submit a Form and Handle the Response?

One-Line Definition

A form is submitted using the onSubmit event, and the API response is handled using async/await or Promise methods after the request completes.

Form Submission Flow

User Fills Form
↓
Click Submit
↓
onSubmit Event Fires
↓
API Request Sent
↓
Server Response Received
↓
Update UI / Show Message

Basic Form Submission Example

function handleSubmit(e) {

  e.preventDefault();

  console.log('Form Submitted');

}

<form onSubmit={handleSubmit}>

  ...

</form>
e.preventDefault() prevents the browser from refreshing the page.

Complete Example

import { useState } from 'react';

function UserForm() {

  const [name, setName] = useState('');

  async function handleSubmit(e) {

    e.preventDefault();

    try {

      const response = await fetch('/api/users', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({ name })
      });

      const data = await response.json();

      alert(data.message);

    } catch (error) {

      alert(error.message);

    }

  }

  return (

    <form onSubmit={handleSubmit}>

      <input value={name} onChange={e => setName(e.target.value)} placeholder="Enter Name" />

      <button type="submit">Submit</button>

    </form>

  );

}

export default UserForm;

Real Project Example

In our Aadhaar Authentication application, when users submit Aadhaar Number, OTP, Consent, and Authentication Type, the form sends a POST request to the authentication API. After receiving the response, React displays success messages, error messages, authentication results, or validation errors.

Important Interview Points

  • Use onSubmit to handle form submission.
  • Use e.preventDefault() to stop page refresh.
  • Use fetch or Axios to call APIs.
  • Handle responses using async/await.
  • Use try/catch for error handling.
  • Show loading indicators during API calls.

Short Interview Answer

Form submission in React is handled using the onSubmit event. The API response is processed using async/await, and the UI is updated based on success or failure.

One-Line Interview Answer

React submits forms using onSubmit and handles responses using async/await or Promises.
