047. What are lanes in React Fiber and how do they manage priority?

One-Line Definition

Lanes are React Fiber's priority system. They help React decide which updates should run first and which updates can wait until later.

Why Do We Need Lanes?

Not all updates are equally important.

If React treated every update the same, typing could become slow whenever large UI updates happen.

Interview Answer

Lanes are React Fiber's priority mechanism used to categorize updates based on importance. Each update is assigned to a lane, and React processes higher-priority lanes before lower-priority ones. This allows urgent updates such as typing and user interactions to remain responsive while less important work, such as transitions and background updates, can be delayed. Lanes work together with Fiber and the Scheduler to support efficient concurrent rendering in modern React.

One-Line Interview Answer
