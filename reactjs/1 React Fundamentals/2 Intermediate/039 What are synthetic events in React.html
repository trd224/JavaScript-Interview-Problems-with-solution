047. What are synthetic events in React?

One-Line Definition

Synthetic events are React’s cross-browser wrapper objects around native browser events that provide consistent event behavior across all browsers.

What Problem Do Synthetic Events Solve?

Different browsers sometimes handle events differently.

Example:

  • Different event properties
  • Different event behavior
  • Browser inconsistencies

React solves this problem using:

SyntheticEvent

Simple Understanding

When an event happens in React:

  • Browser creates native event
  • React wraps it inside SyntheticEvent
  • Developers receive consistent event object

Basic Example

function App() {

  const handleClick = (event) => {

    console.log(event);

  };

  return (

    <button onClick={handleClick}>
      Click
    </button>

  );

}

Important Point

The event parameter here is NOT the raw browser event. It is a React SyntheticEvent object.

Common Synthetic Events

React Event Browser Event

Example Accessing Event Data

function App() {

  const handleChange = (event) => {

    console.log(event.target.value);

  };

  return (

    <input onChange={handleChange} />

  );

}

Important Methods

Method Purpose
preventDefault() Prevent default browser behavior
stopPropagation() Stop event bubbling
target Access event target element

preventDefault Example

function App() {

  const handleSubmit = (event) => {

    event.preventDefault();

    console.log('Form Submitted');

  };

  return (

    <form onSubmit={handleSubmit}>

      <button>
        Submit
      </button>

    </form>

  );

}

SyntheticEvent vs Native Event

SyntheticEvent Native Event
React wrapper object Original browser event
Cross-browser consistent Browser-specific behavior
Managed by React Managed by browser
Normalized API May differ across browsers

Accessing Native Browser Event

event.nativeEvent

This gives access to the original browser event.

Old Event Pooling Concept

Older React versions reused SyntheticEvent objects for performance.

This was called:

Event Pooling

React 17 removed event pooling.

Synthetic events are React’s normalized wrapper objects around native browser events that provide consistent cross-browser event handling and improved performance.
