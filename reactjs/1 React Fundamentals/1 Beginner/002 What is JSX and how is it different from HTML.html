002. What is JSX and how is it different from HTML?

One-Line Definition

JSX (JavaScript XML) is a syntax extension for JavaScript that allows developers to write HTML-like code inside React.

Why Is It Used?

  • Easier to read
  • Easier to write
  • Easier to understand
  • Makes UI code cleaner
  • Helps build React components quickly

Real Project Use Case

Suppose you are building a Login Form, OTP Popup, Dashboard, or User Profile Card in React.

JSX allows you to directly design UI inside JavaScript files instead of manually creating DOM elements.

Without JSX

React.createElement(
  'h1',
  null,
  'Hello'
);

With JSX

<h1>Hello</h1>

JSX is much cleaner and easier to maintain.

Example Code

function App() {

  const name = 'Rizwan';

  return (
    <div>
      <h1>Hello {name}</h1>

      <p>
        Welcome to React
      </p>

    </div>
  );
}

Output

Hello Rizwan
Welcome to React

How JSX Is Different from HTML

JSX HTML
Used inside React Used in normal webpages
Written inside JavaScript Written directly in HTML files
Uses className Uses class
Uses camelCase attributes Uses normal HTML attributes
Can use JavaScript expressions Cannot directly use JavaScript expressions
Must return one parent element No such rule

Important JSX Rules

1. Use className Instead of class

<div className="box"></div>

HTML Version:

<div class="box"></div>

2. JavaScript Can Be Used Inside { }

<h1>{2 + 2}</h1>

Output

4

3. JSX Must Have One Parent Element

Correct:

return (
  <div>

    <h1>Hello</h1>

    <p>Welcome</p>

  </div>
);

Wrong:

return (
  <h1>Hello</h1>

  <p>Welcome</p>
);

Real-World Example

  • Facebook
  • Instagram
  • Netflix

Large React applications use JSX to build reusable UI components in a clean and maintainable way.

Important Interview Points

  • JSX is not HTML.
  • JSX gets converted into JavaScript by Babel.
  • Browsers cannot understand JSX directly.
  • JSX improves readability.
  • JSX allows JavaScript expressions inside UI.

Short Interview Answer

JSX is a syntax extension for JavaScript used in React that allows developers to write HTML-like UI code inside JavaScript.

One-Line Interview Answer

JSX lets developers write HTML-like code inside JavaScript for React applications.
