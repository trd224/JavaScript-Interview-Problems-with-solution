022. What is a fragment in React and why use it?

One-Line Definition

A Fragment in React allows grouping multiple elements without adding extra HTML elements to the DOM.

Why Is Fragment Needed?

React components must return only one parent element.

Without Fragment, developers often add unnecessary wrapper div elements.

Fragment solves this problem.

Problem Without Fragment

function App() {

  return (

    <div>

      <h1>Title</h1>

      <p>Description</p>

    </div>

  );

}

Problem Here

The extra div may:

  • Create unnecessary DOM nodes
  • Break CSS layouts
  • Increase DOM size

Using Fragment

import React from 'react';

function App() {

  return (

    <React.Fragment>

      <h1>Title</h1>

      <p>Description</p>

    </React.Fragment>

  );

}

Short Syntax

function App() {

  return (

    <>

      <h1>Title</h1>

      <p>Description</p>

    </>

  );

}

Output

Title
Description

What Happens Internally?

With div With Fragment
Extra DOM node created No extra DOM node
Larger DOM tree Cleaner DOM tree
May affect CSS No layout issues

Flow Diagram

Multiple JSX Elements → Fragment → No Extra DOM Element

Fragment in Table Example

Fragments are very useful in tables because extra div elements break table structure.

Correct Example

function Columns() {

  return (

    <>

      <td>Name</td>

      <td>Rizwan</td>

    </>

  );

}

Why Not div Here?

A div inside tr would create invalid HTML.

Key Prop with Fragment

Short syntax cannot accept props like key.

Use full Fragment syntax when key is required.

Example

import React from 'react';

items.map(item => (

  <React.Fragment key={item.id} >

    <h1>{item.name}</h1>

  </React.Fragment>

));

Real Project Use Cases

Scenario Benefit
Dashboard cards Cleaner DOM
Table rows Valid HTML structure
Popup layouts Avoid extra wrappers
Large applications Better DOM performance

Important Interview Points

  • Fragments group multiple JSX elements.
  • Fragments do not create extra DOM nodes.
  • Useful for clean HTML structure.
  • Short syntax uses <></>.
  • Full Fragment syntax supports key prop.

Short Interview Answer

A Fragment in React groups multiple elements without adding unnecessary wrapper elements to the DOM.

One-Line Interview Answer

React Fragment allows multiple JSX elements to be returned without creating extra DOM nodes.
