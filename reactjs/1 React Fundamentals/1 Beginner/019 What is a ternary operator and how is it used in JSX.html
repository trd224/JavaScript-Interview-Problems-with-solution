019. What is a ternary operator and how is it used in JSX?

One-Line Definition

A ternary operator is a short form of if...else used to conditionally return values inside JSX.

Why Is It Used in JSX?

JSX does not allow if statements directly inside HTML-like code because JSX only accepts expressions.

The ternary operator is an expression, so it works perfectly inside JSX.

Ternary Operator Syntax

condition ? trueValue : falseValue

Simple JavaScript Example

const age = 18;

const result =
  age >= 18
    ? 'Adult'
    : 'Minor';

console.log(result);

Output

Adult

JSX Example

function App() {

  const isLoggedIn = true;

  return (

    <div>

      {
        isLoggedIn
          ? <h1>Welcome User</h1>
          : <h1>Please Login</h1>
      }

    </div>

  );

}

Output

Welcome User

How It Works

Condition Result
isLoggedIn = true Welcome User
isLoggedIn = false Please Login

Flow Diagram

Condition → True ? First Value : Second Value

Real Project Use Cases

Scenario Ternary Usage
Login System User ? Dashboard : Login
Loader loading ? Loader : Data
Error Handling error ? ErrorMessage : Content
Theme Switch dark ? DarkTheme : LightTheme

Example with Button

function Theme() {

  const darkMode = true;

  return (

    <button>

      {
        darkMode
          ? 'Dark Mode'
          : 'Light Mode'
      }

    </button>

  );

}

Output

Dark Mode

Nested Ternary Example

{
  score >= 80
    ? 'Excellent'
    : score >= 50
      ? 'Good'
      : 'Fail'
}

Important Note

Nested ternary operators can make code hard to read.

For complex conditions, use separate variables or functions.

Ternary vs if Statement

Ternary Operator if Statement
Expression Statement
Works inside JSX Cannot work directly inside JSX
Short syntax Long syntax
Used for conditional rendering Used outside JSX

Important Interview Points

  • Ternary operator is a shorthand for if...else.
  • It works inside JSX because it is an expression.
  • Used heavily for conditional rendering.
  • Syntax uses ? and : operators.
  • Avoid deeply nested ternary operators.

Short Interview Answer

A ternary operator is a short form of if...else that allows conditional rendering inside JSX using the syntax condition ? trueValue : falseValue.

One-Line Interview Answer

A ternary operator is a JSX-friendly conditional expression used for rendering different UI based on conditions.
