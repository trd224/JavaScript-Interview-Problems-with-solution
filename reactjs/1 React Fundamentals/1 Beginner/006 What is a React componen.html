006. What is a React Component?

One-Line Definition

A React component is a reusable piece of UI that contains its own structure, logic, and behavior.

Why Is It Used?

Components help developers break large applications into smaller, reusable, and manageable pieces.

Instead of writing the same UI code multiple times, React components allow code reuse.

Real Project Use Case

Suppose you are building an Aadhaar Authentication Portal.

Different parts of the application can be separate components:

  • Login Component
  • OTP Popup Component
  • Loader Component
  • Navbar Component
  • Dashboard Component
  • Error Popup Component

These components can be reused throughout the application.

Think of Components Like LEGO Blocks

Small components combine together to create a complete application UI.

Example Component

function Welcome() {

  return (

    <h1>
      Welcome to React
    </h1>

  );

}

Using the Component

function App() {

  return (

    <div>

      <Welcome />

    </div>

  );

}

Output

Welcome to React

Types of Components

Type Description
Functional Component Simple JavaScript function that returns JSX
Class Component Older class-based component syntax

Functional Component Example

function Header() {

  return (
    <h1>
      Header Component
    </h1>
  );

}

Class Component Example

class Header extends React.Component {

  render() {

    return (

      <h1>
        Header Component
      </h1>

    );

  }

}

How Components Work Together

<App>

  <Navbar />

  <Sidebar />

  <Dashboard />

  <Footer />

</App>

Multiple components combine to build a full application.

Benefits of Components

  • Reusable code
  • Cleaner project structure
  • Easy maintenance
  • Independent logic handling
  • Better scalability

Important Interview Points

  • Components are reusable UI blocks.
  • React applications are component-based.
  • Components return JSX.
  • Functional components are most commonly used today.
  • Components can have props and state.

Short Interview Answer

A React component is a reusable UI block that contains its own structure and logic. React applications are built by combining multiple components together.

One-Line Interview Answer

A React component is a reusable piece of UI used to build React applications.
