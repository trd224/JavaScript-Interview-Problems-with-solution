030. What is event propagation and how do you stop it in React?

One-Line Definition

Event propagation is the process where an event travels through parent and child elements, and it can be stopped in React using event.stopPropagation().

What Is Event Propagation?

When an event occurs on an element:

  • The event does not stay only on that element.
  • The event travels through the DOM tree.

This traveling process is called:

Event Propagation

Types of Event Propagation

Type Direction
Capturing Phase Parent → Child
Bubbling Phase Child → Parent

Most Common in React

React mainly uses event bubbling by default.

Simple Example

function App() {

  const parentClick = () => {
    console.log('Parent Clicked');
  };

  const buttonClick = () => {
    console.log('Button Clicked');
  };

  return (

    <div onClick={parentClick}>

      <button onClick={buttonClick}>
        Click
      </button>

    </div>

  );

}

What Happens?

When button is clicked: 

Button Clicked
Parent Clicked

Why Parent Clicked?

Because the click event bubbled from:

button
↓
parent div

How to Stop Event Propagation

React provides: 

event.stopPropagation()

Example

function App() {

  const parentClick = () => {
    console.log('Parent Clicked');
  };

  const buttonClick = (event) => {

    event.stopPropagation();

    console.log('Button Clicked');

  };

  return (

    <div onClick={parentClick}>

      <button onClick={buttonClick}>
        Click
      </button>

    </div>

  );

}

Output

Button Clicked

What stopPropagation() Does

It prevents the event from moving to parent elements.

Flow Without stopPropagation()

Button Click
↓
Button Handler Executes
↓
Parent Handler Executes

Flow With stopPropagation()

Button Click
↓
Button Handler Executes
↓
Event Stops Here

Capturing Phase in React

React also supports capturing events using: 

onClickCapture

Example

<div onClickCapture={handleClick}>
  <button>Click</button>
</div>

Capturing runs before bubbling.

Event Propagation Order

Capturing Phase
Parent → Child

Target Phase
Actual Element

Bubbling Phase
Child → Parent

Real Project Use Cases

Scenario Why stopPropagation() Is Used
Popup modal Prevent background click
Dropdown menu Prevent parent close event
Table row buttons Prevent row click handler
Authentication popup Prevent overlay click events

Difference Between preventDefault and stopPropagation

preventDefault() stopPropagation()
Stops default browser behavior Stops event bubbling/capturing
Example: stop form reload Example: stop parent click

Important Interview Points

  • Event propagation means event traveling through DOM.
  • React mainly uses bubbling phase.
  • stopPropagation() stops event bubbling.
  • onClickCapture handles capturing phase.
  • preventDefault() and stopPropagation() are different.

Short Interview Answer

Event propagation is the movement of events through parent and child elements. In React, it can be stopped using event.stopPropagation() to prevent bubbling or capturing behavior.

One-Line Interview Answer

Event propagation is event movement through the DOM tree, and React stops it using event.stopPropagation().
