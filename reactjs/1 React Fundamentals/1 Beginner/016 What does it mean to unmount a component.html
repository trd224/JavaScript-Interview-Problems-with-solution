016. What does it mean to unmount a component?

One-Line Definition

Unmounting means a React component is removed from the DOM and destroyed.

Why Does Unmounting Happen?

React removes components when they are no longer needed in the UI.

This helps:

  • Free memory
  • Stop unnecessary processes
  • Improve performance
  • Prevent memory leaks

Simple Real Example

Action What Happens
Open OTP Popup Component mounts
Close OTP Popup Component unmounts

Example

function App() {

  const [show, setShow] = useState(true);

  return (

    <div>

      <button onClick={() => setShow(!show)} >
        Toggle
      </button>

      {show && <User />}

    </div>

  );

}

User Component

function User() {

  return (
    <h1>User Component</h1>
  );

}

What Happens Here?

Condition Result
show = true User component mounts
show = false User component unmounts

Unmount Cleanup

During unmounting, React allows cleanup operations.

This is important for:

  • Removing event listeners
  • Stopping timers
  • Closing WebSocket connections
  • Canceling API requests

useEffect Cleanup Example

import { useEffect } from 'react';

function User() {

  useEffect(() => {

    console.log('Mounted');

    return () => {

      console.log('Unmounted');

    };

  }, []);

  return (
    <h1>User Component</h1>
  );

}

Output

Mounted
Unmounted

Important Point

The cleanup function inside useEffect runs during unmounting.

Lifecycle Flow

Mounting → Updating → Unmounting

Real Project Use Cases

Scenario Unmounting Purpose
Closing popup Remove popup component
Navigating pages Remove old page component
Leaving chat screen Close WebSocket connection
Stopping timers Prevent memory leaks

Why Cleanup Is Important

Without cleanup:

  • Timers continue running
  • API calls may continue
  • Memory leaks can occur
  • Performance issues may happen

Important Interview Points

  • Unmounting removes a component from the DOM.
  • Cleanup happens during unmounting.
  • useEffect cleanup function handles unmount logic.
  • Unmounting helps prevent memory leaks.
  • Unmounting occurs when components disappear from UI.

Short Interview Answer

Unmounting means React removes a component from the DOM. Cleanup operations usually happen during unmounting using useEffect cleanup functions.

One-Line Interview Answer

Unmounting is the process where React removes a component from the DOM.
