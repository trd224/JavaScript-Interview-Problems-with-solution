020. What is short-circuit evaluation in JSX?

One-Line Definition

Short-circuit evaluation in JSX uses the && operator to conditionally render elements when a condition is true.

Why Is It Used?

It provides a short and clean way to render UI conditionally without using a ternary operator.

Mostly used when you want to show something only if a condition is true.

Basic Syntax

condition && JSX

How It Works

Condition Result
true Right side renders
false Nothing renders

Simple Example

function App() {

  const isLoggedIn = true;

  return (

    <div>

      {isLoggedIn && <h1>Welcome User</h1>}

    </div>

  );

}

Output

Welcome User

When Condition Is False

const isLoggedIn = false;

{isLoggedIn && <h1>Welcome User</h1>}

Output

Nothing Rendered

Why It Is Called Short-Circuit?

JavaScript stops checking further when the first condition becomes false.

So the JSX part never executes if the condition is false.

Flow Diagram

Condition True → Render JSX

Condition False → Stop Execution → Render Nothing

Real Project Use Cases

Scenario Usage
Loader isLoading && <Loader />
Error Popup hasError && <ErrorPopup />
OTP Box showOtp && <OtpForm />
User Profile user && <Profile />

Example with Loader

function App() {

  const loading = true;

  return (

    <div>

      {loading && <h1>Loading...</h1>}

    </div>

  );

}

Output

Loading...

Important Problem with 0

Be careful when using numbers with && because 0 can appear on screen.

Example

const count = 0;

{count && <h1>Items Available</h1>}

Output

0

Why?

JavaScript returns the first falsy value.

Since 0 is falsy, React renders 0.

Correct Solution

{count > 0 && <h1>Items Available</h1>}

Short-Circuit vs Ternary Operator

Short-Circuit Ternary Operator
Used for showing only true condition Used for true and false both
Uses && Uses ? :
Short syntax Slightly larger syntax
No else part Has else part

Important Interview Points

  • Short-circuit evaluation uses the && operator.
  • Used for conditional rendering in JSX.
  • If condition is false, React renders nothing.
  • Commonly used for loaders, popups, and optional UI.
  • Be careful with numeric values like 0.

Short Interview Answer

Short-circuit evaluation in JSX uses the && operator to render elements only when a condition is true.
