007. How do you create a functional component in React?

One-Line Definition

A functional component is a simple JavaScript function that returns JSX.

Why Is It Used?

Functional components are used because they are:

  • Simple
  • Readable
  • Reusable
  • Easy to maintain
  • Faster to write

Modern React applications mostly use functional components with Hooks.

Basic Syntax

function ComponentName() {

  return (
    JSX
  );

}

Simple Functional Component Example

function Welcome() {

  return (

    <h1>
      Welcome to React
    </h1>

  );

}

Using the Component

function App() {

  return (

    <div>

      <Welcome />

    </div>

  );

}

Output

Welcome to React

Arrow Function Component

Functional components can also be created using arrow functions. 

const Welcome = () => {

  return (

    <h1>
      Welcome to React
    </h1>

  );

};

Real Project Use Case

Suppose you are building:

  • Navbar
  • Login Form
  • OTP Popup
  • Loader
  • Dashboard

Each UI part can be created as a separate functional component.

Component with Props

Functional components can receive data using props. 

function User(props) {

  return (

    <h1>
      Hello {props.name}
    </h1>

  );

}

Using Props

<User name="Rizwan" />

Output

Hello Rizwan

Component with useState Hook

Functional components can manage state using Hooks. 

import { useState } from 'react';

function Counter() {

  const [count, setCount] = useState(0);

  return (

    <div>

      <h1>{count}</h1>

      <button
        onClick={() => setCount(count + 1)}
      >

        Increment

      </button>

    </div>

  );

}

What Happens Here?

  • useState creates state
  • count stores the current value
  • setCount updates the state
  • UI re-renders automatically

Why Functional Components Are Preferred

Functional Component Class Component
Simple syntax Complex syntax
Uses Hooks Uses lifecycle methods
Less boilerplate More boilerplate
Modern React standard Older approach

Important Interview Points

  • Functional components are plain JavaScript functions.
  • They return JSX.
  • Hooks work only inside functional components.
  • Functional components are preferred in modern React.
  • Props are used to pass data into components.

Short Interview Answer

A functional component is a JavaScript function that returns JSX. Modern React applications mainly use functional components with Hooks.

One-Line Interview Answer

A functional component is a JavaScript function used to create reusable UI in React.
