012. What is a React root and how is it created?

One-Line Definition

A React root is the main entry point where a React application gets attached to the browser DOM.

Why Is It Used?

React components exist inside JavaScript memory.

To display the application in the browser, React needs a root container where the entire application will render.

Root Element in HTML

Usually React applications render inside: 

<div id="root"></div>

This element exists in the main HTML file.

How React Root Is Created

In React 18+, createRoot() is used. 

import { createRoot } from 'react-dom/client';

const root = createRoot(
  document.getElementById('root')
);

Rendering the App

root.render(
  <App />
);

Complete Example

import React from 'react';
import { createRoot } from 'react-dom/client';

function App() {

  return (
    <h1>Hello React</h1>
  );

}

const root = createRoot(
  document.getElementById('root')
);

root.render(
  <App />
);

Output

Hello React

Flow Diagram

HTML Root Element → createRoot() → root.render() → React UI Appears

What Happens Internally?

Step What Happens
1 Browser loads HTML page
2 React finds the root div
3 createRoot() creates React root
4 root.render() renders React components
5 UI appears in browser

React 17 Older Syntax

Before React 18, applications used: 

ReactDOM.render(
  <App />,
  document.getElementById('root')
);

React 18 Modern Syntax

const root = createRoot(
  document.getElementById('root')
);

root.render(
  <App />
);

Why React 18 Uses createRoot()

  • Supports concurrent rendering
  • Better performance optimization
  • Supports modern React features
  • Improved rendering engine

Real Project Use Case

In applications like:

  • Authentication Portal
  • Dashboard
  • Admin Panel
  • E-commerce Website

The entire React application starts from a single React root.

Important Interview Points

  • A React root is the mounting point of the React app.
  • React 18 uses createRoot().
  • root.render() displays components in browser.
  • The root is usually attached to a div with id="root".
  • Older React versions used ReactDOM.render().

Short Interview Answer

A React root is the main DOM container where a React application renders. It is created using createRoot() in React 18.

One-Line Interview Answer

A React root is the entry point where React renders the application into the browser DOM.
