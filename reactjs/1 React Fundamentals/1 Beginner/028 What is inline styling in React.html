028. What is inline styling in React?

One-Line Definition

Inline styling in React means applying CSS directly inside JSX using JavaScript style objects.

Why Is Inline Styling Used?

Inline styles are useful when:

  • Styles are dynamic
  • Styles depend on state or props
  • Quick component-level styling is needed
  • Conditional styling is required

Basic Syntax

<h1 style={{ color: 'cyan' }}>
  Hello React
</h1>

Important Syntax Rule

Inline styles use double curly braces.
Part Meaning
{ } JavaScript expression in JSX
{{ }} JavaScript object

Example

function App() {

  return (

    <h1

      style={{
        color: 'cyan',
        backgroundColor: 'black',
        fontSize: '32px'
      }}

    >

      Hello React

    </h1>

  );

}

Output

Hello React

Why CSS Properties Use camelCase?

Inline styles are JavaScript objects, so property names follow JavaScript naming rules.

CSS vs React Inline Style

CSS React Inline Style
background-color backgroundColor
font-size fontSize
text-align textAlign
border-radius borderRadius

Using Style Object Variable

function App() {

  const headingStyle = {

    color: 'white',
    backgroundColor: 'black',
    padding: '20px'

  };

  return (

    <h1 style={headingStyle}>
      Hello React
    </h1>

  );

}

Dynamic Inline Styling

function App() {

  const darkMode = true;

  return (

    <h1

      style={{
        color: darkMode ? 'white' : 'black',
        backgroundColor: darkMode ? 'black' : 'white'
      }}

    >

      Theme Example

    </h1>

  );

}

Flow Diagram

JavaScript Style Object → style Attribute → Browser Applies CSS

Advantages of Inline Styling

  • Dynamic styling support
  • No separate CSS file needed
  • Easy conditional styling
  • Component-level styling

Disadvantages of Inline Styling

  • Large styles become hard to manage
  • No pseudo selectors like :hover
  • No media queries directly
  • Less reusable compared to CSS classes

Real Project Use Cases

Scenario Usage
Dark mode Dynamic theme colors
OTP validation Error color changes
Button states Enable/disable styling
Loader visibility Conditional display styles

Important Interview Points

  • Inline styling uses JavaScript objects.
  • Styles are applied using the style attribute.
  • CSS properties use camelCase.
  • Double curly braces are required.
  • Best for dynamic and conditional styles.

Short Interview Answer

Inline styling in React applies CSS directly inside JSX using JavaScript style objects and camelCase property names.

One-Line Interview Answer

Inline styling in React uses JavaScript objects inside the style attribute to apply CSS directly in JSX.
