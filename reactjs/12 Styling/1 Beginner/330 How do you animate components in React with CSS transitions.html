330. How Do You Animate Components in React with CSS Transitions?

One-Line Definition

React components can be animated using CSS transitions by changing CSS classes or styles when state changes.

What Is a CSS Transition?

A CSS transition creates a smooth animation when a CSS property changes from one value to another.

Commonly animated properties include:

  • opacity
  • transform
  • width
  • height
  • background-color
  • border-color

Transition Flow

User Action
↓
State Changes
↓
Class Name Changes
↓
CSS Transition Starts
↓
Smooth Animation

Basic CSS Transition

App.css

.box {

  width: 100px;

  height: 100px;

  background: blue;

  transition: all 0.3s ease;

}

.box.active {

  transform: scale(1.2);

}

Component

import { useState } from 'react';

function Box() {

  const [active, setActive] = useState(false);

  return (

    <div

      className={active ? 'box active' : 'box'}

      onClick={() => setActive(!active)}

    ></div>

  );

}

Fade In / Fade Out Animation

CSS

.content {

  opacity: 0;

  transition: opacity 0.5s ease;

}

.content.show {

  opacity: 1;

}

React

const [show, setShow] = useState(false);

<div className={show ? 'content show' : 'content'}>

  Hello React

</div>

Slide Animation Example

CSS

.panel {

  transform: translateX(-100%);

  transition: transform 0.4s ease;

}

.panel.open {

  transform: translateX(0);

}

React

const [open, setOpen] = useState(false);

<div className={open ? 'panel open' : 'panel'}>

  Sidebar

</div>

Using Inline Styles for Animation

const [active, setActive] = useState(false);

<div

  style={{
    opacity: active ? 1 : 0,
    transition: 'opacity 0.5s ease'
  }}

>

  Animated Content

</div>

Hover Animation Example

.button {

  background: blue;

  color: white;

  transition: transform 0.3s ease;

}

.button:hover {

  transform: scale(1.05);

}

Common Transition Properties

Property Purpose
opacity Fade In / Fade Out
transform Scale, Rotate, Slide
background-color Color Animation
width Expand / Collapse
height Accordion Effects

Complete Example

App.css

.card {

  opacity: 0;

  transform: translateY(20px);

  transition: all 0.4s ease;

}

.card.show {

  opacity: 1;

  transform: translateY(0);

}

App.jsx

import { useState } from 'react';

function App() {

  const [show, setShow] = useState(false);

  return (

    <>

      <button onClick={() => setShow(!show)}>

        Toggle

      </button>

      <div className={show ? 'card show' : 'card'}>

        Welcome

      </div>

    </>

  );

}

export default App;

When CSS Transitions Are Enough

  • Fade effects.
  • Hover animations.
  • Sidebar opening.
  • Dropdown menus.
  • Button effects.
  • Modal animations.

When We Need Animation Libraries

  • Complex page transitions.
  • Drag and drop animations.
  • Physics-based animations.
  • Gesture animations.
For advanced React animations, libraries such as Framer Motion are commonly used.

Important Interview Points

  • React itself does not perform the animation.
  • CSS handles the animation using transition.
  • State changes trigger class changes.
  • Class changes trigger CSS transitions.
  • Transitions work best for simple UI animations.

Short Interview Answer

Components can be animated in React by changing CSS classes or styles based on state and using the CSS transition property to animate those changes.

One-Line Interview Answer

React animations with CSS transitions work by changing styles or class names when state changes and letting CSS animate the transition.
