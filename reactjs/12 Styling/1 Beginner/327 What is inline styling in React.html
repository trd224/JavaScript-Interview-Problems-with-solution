327. What is Inline Styling in React?

One-Line Definition

Inline styling in React is a way to apply CSS directly to an element using a JavaScript object passed to the style prop.

Why Use Inline Styling?

Sometimes we need styles that are dynamic and depend on component state, props, or calculations.

Instead of creating a separate CSS file, we can define styles directly inside the component.

Basic Syntax

<h1 style={{ color: 'blue' }}>

  Hello React

</h1>
The style prop expects a JavaScript object, not a CSS string.

Using a Style Object

const headingStyle = {
  color: 'blue',
  fontSize: '30px',
  fontWeight: 'bold'
};

<h1 style={headingStyle}>

  Hello React

</h1>

How Inline Styling Works

↓
style Prop
↓

CSS vs React Style Syntax

CSS React Inline Style
background-color backgroundColor
font-size fontSize
margin-top marginTop
text-align textAlign

Why CamelCase?

JavaScript object properties cannot contain hyphens without quotes. 

// Wrong

{
  font-size: '20px'
}

// Correct

{
  fontSize: '20px'
}

Multiple Styles Example

const buttonStyle = {
  backgroundColor: 'blue',
  color: 'white',
  padding: '10px 20px',
  border: 'none',
  borderRadius: '5px'
};

<button style={buttonStyle}>

  Submit

</button>

Dynamic Inline Styling

const [isActive, setIsActive] = useState(false);

<button
  style={{
    backgroundColor: isActive ? 'green' : 'red',
    color: 'white'
  }}
>

  Status

</button>
Inline styles are very useful when styles depend on state or props.

Conditional Styling Example

<p
  style={{
    color: age >= 18 ? 'green' : 'red'
  }}
>

  User Status

</p>

  const cardStyle = {
    backgroundColor: '#1e293b',
    color: 'white',
    padding: '20px',

  };

  return (

    <div style={cardStyle}>

export default Profile;

Advantages of Inline Styling

  • Easy to use.
  • Works well with dynamic values.
  • No separate CSS file required.
  • Styles can use state and props directly.

Limitations of Inline Styling

  • No CSS pseudo-classes (:hover, :focus).
  • No media queries.
  • Can become difficult to maintain in large applications.
  • Not ideal for large reusable style systems.

Important Interview Points

  • Inline styling uses the style prop.
  • Styles are written as JavaScript objects.
  • CSS properties use camelCase.
  • Supports dynamic and conditional styling.
  • Useful for small and dynamic styles.

Short Interview Answer

Inline styling in React applies CSS directly through the style prop using JavaScript objects, making it useful for dynamic styling based on state or props.
