324. How Do You Apply CSS in a React Component?

One-Line Definition

CSS can be applied in React using CSS files, inline styles, CSS Modules, or CSS-in-JS libraries.

Most Common Ways to Apply CSS

Method Description
CSS File Import a normal CSS file
Inline Style Apply styles using a JavaScript object
CSS Modules Component-scoped CSS
Styled Components CSS-in-JS approach

1. Using a CSS File (Most Common)

Button.css

.button {

  background: blue;

  color: white;

  padding: 10px 20px;

}

Button.jsx

import './Button.css';

function Button() {

  return (

    <button className="button">

      Click Me

    </button>

  );

}
React uses className instead of class because class is a JavaScript keyword.

2. Using Inline Styles

function Button() {

  const buttonStyle = {
    backgroundColor: 'blue',
    color: 'white',
    padding: '10px 20px'
  };

  return (

    <button style={buttonStyle}>

      Click Me

    </button>

  );

}

Inline Style Rules

CSS React Style
background-color backgroundColor
font-size fontSize
margin-top marginTop

3. Using CSS Modules

Button.module.css

.button {

  background: green;

  color: white;

}

Button.jsx

import styles from './Button.module.css';

function Button() {

  return (

    <button className={styles.button}>

      Click Me

    </button>

  );

}
CSS Modules prevent class name conflicts between components.

4. Using Styled Components

import styled from 'styled-components';

const Button = styled.button`

  background: blue;

  color: white;

  padding: 10px 20px;

`;

function App() {

  return <Button>Click Me</Button>;

}

Complete Example

import './App.css';

function App() {

  return (

    <div className="container">

      <h1>Hello React</h1>

      <button className="btn">

        Submit

      </button>

    </div>

  );

}

export default App;

App.css

.container {

  padding: 20px;

}

.btn {

  background: blue;

  color: white;

  padding: 10px 20px;

  border: none;

}

Real Project Example

In our Aadhaar Authentication application, most styles are applied using CSS files and inline style objects. Modal popups, consent screens, authentication forms, and dashboard pages use className to apply reusable CSS styles.

Important Interview Points

  • React uses className instead of class.
  • Inline styles use JavaScript objects.
  • CSS property names use camelCase.
  • CSS Modules prevent style conflicts.
  • Normal CSS files are the most commonly used approach.

Short Interview Answer

CSS can be applied in React using CSS files, inline styles, CSS Modules, or CSS-in-JS libraries such as Styled Components.

One-Line Interview Answer

React components can be styled using CSS files, inline styles, CSS Modules, or CSS-in-JS solutions.
