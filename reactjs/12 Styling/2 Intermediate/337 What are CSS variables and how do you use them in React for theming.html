337. What are CSS Variables and How Do You Use Them in React for Theming?

One-Line Definition

CSS Variables (Custom Properties) are reusable CSS values that can be defined once and used throughout an application. They are commonly used in React for light mode, dark mode, and application-wide theming.

What Problem Do CSS Variables Solve?

Without CSS variables: 

.header {
  background: #2563eb;
}

.button {
  background: #2563eb;
}

.card {
  border-color: #2563eb;
}

If the primary color changes, every occurrence must be updated manually.

Large applications can contain hundreds of repeated colors.

Using CSS Variables

:root {
  --primary-color: #2563eb;
}

.header {
  background: var(--primary-color);
}

.button {
  background: var(--primary-color);
}
Now changing one variable updates the entire application.

Using CSS Variables for Dark Mode

File: src/App.css
:root {
  --bg: #ffffff;
  --text: #111827;
  --card: #f3f4f6;
}

[data-theme="dark"] {
  --bg: #111827;
  --text: #f9fafb;
  --card: #1f2937;
}
Light Theme
↓
--bg = white

Dark Theme
↓
--bg = dark

Step 3 - Toggle Theme in React

File: src/components/ThemeToggle.jsx
function ThemeToggle({ theme, toggleTheme }) {
  return <button onClick={toggleTheme}>{theme === 'dark' ? 'Light Mode' : 'Dark Mode'}</button>;
}

export default ThemeToggle;

Step 4 - Apply Theme Attribute

File: src/App.jsx
import { useEffect, useState } from 'react';
import ThemeToggle from './components/ThemeToggle';
import './App.css';

function App() {
  const [theme, setTheme] = useState(() => localStorage.getItem('theme') || 'light');

  function toggleTheme() {
    setTheme(prev => prev === 'light' ? 'dark' : 'light');
  }

  useEffect(() => {
    document.documentElement.setAttribute('data-theme', theme);
    localStorage.setItem('theme', theme);
  }, [theme]);

  return (
    <div className="app">
      <ThemeToggle theme={theme} toggleTheme={toggleTheme} />

      <div className="card">
        <h2>Dashboard</h2>
        <p>Current Theme: {theme}</p>
      </div>
    </div>
  );
}

export default App;

Advantages of CSS Variables

  • Easy theming
  • Dark mode support
  • Reusable values
  • Less duplicated CSS
  • Easy maintenance
  • Works without extra libraries

CSS Variables vs CSS Classes

Great for theming Great for layouts

Interview Answer

CSS Variables are custom CSS properties defined using the -- syntax. They are accessed using var(). In React applications, CSS Variables are commonly used for theming by defining light and dark theme values and switching them using a class or data-theme attribute. This allows the entire application's colors to change by updating only a few variables.

One-Line Interview Answer

CSS Variables are reusable CSS values that are commonly used in React to implement themes such as light mode and dark mode.
