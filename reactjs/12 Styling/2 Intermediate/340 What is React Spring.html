340. What is React Spring?

One-Line Definition

React Spring is a React animation library that creates smooth, physics-based animations using spring physics instead of traditional duration-based animations.

Why React Spring?

Traditional animations usually use: 

transition: all 0.5s ease;

React Spring uses: 

Mass
+
Tension
+
Friction
=
Natural Motion
Animations feel more realistic because they behave like real-world springs.

Basic Fade Animation

File: src/components/FadeBox.jsx
import { useSpring, animated } from '@react-spring/web';

function FadeBox() {
  const styles = useSpring({
    opacity: 1,
    from: { opacity: 0 }
  });

  return <animated.div style={styles}>Hello React Spring</animated.div>;
}

export default FadeBox;

Understanding The Code

  opacity: 1,

});

↓

Why animated.div?

React Spring cannot animate a normal div.

Instead of: 

<div>Hello</div>

Use: 

<animated.div>Hello</animated.div>
animated.div can receive animated values.

Toggle Animation Example

File: src/components/ToggleBox.jsx
import { useState } from 'react';
import { useSpring, animated } from '@react-spring/web';

function ToggleBox() {
  const [open, setOpen] = useState(false);

  const styles = useSpring({
    opacity: open ? 1 : 0,
    transform: open ? 'translateY(0px)' : 'translateY(-50px)'
  });

  return (
    <>
      <button onClick={() => setOpen(!open)}>Toggle</button>

      <animated.div style={styles}>
        Content
      </animated.div>
    </>
  );
}

export default ToggleBox;

Spring Physics

  • Tension
  • Friction

const styles = useSpring({
  opacity: 1,
  from: { opacity: 0 },
  config: {
    tension: 180,
    friction: 12

});

What Do These Mean?

Property Meaning
tension Spring strength
friction Resistance
mass Object weight

List Animation Example

File: src/components/ListAnimation.jsx
import { useTransition, animated } from '@react-spring/web';

function ListAnimation({ items }) {
  const transitions = useTransition(items, {
    from: { opacity: 0, transform: 'translateY(-20px)' },
    enter: { opacity: 1, transform: 'translateY(0px)' },
    leave: { opacity: 0, transform: 'translateY(-20px)' }
  });

  return transitions((style, item) => (
    <animated.div style={style}>
      {item}
    </animated.div>
  ));
}

export default ListAnimation;

React Spring vs Framer Motion

React Spring Framer Motion
Physics-based Timeline-based
Hook-driven Component-driven
More flexible Easier API
Great for complex motion Great for UI animations
Steeper learning curve Easier to learn

Framer Motion Example

File: src/components/Card.jsx
import { motion } from 'framer-motion';

function Card() {
  return (
    <motion.div
      initial={{ opacity: 0 }}
      animate={{ opacity: 1 }}
    >
      Card
    </motion.div>
  );
}

export default Card;
Framer Motion usually requires less code for common UI animations.

When Should You Use React Spring?

  • Physics-based animations
  • Interactive dashboards
  • Complex motion systems
  • Gesture-driven interfaces
  • Natural spring movement

When Should You Use Framer Motion Instead?

  • Page transitions
  • Modal animations
  • Hover effects
  • Route transitions
  • Typical UI animations

Interview Answer

React Spring is a React animation library that uses spring physics to create smooth and natural animations. It provides hooks such as useSpring(), useTransition(), and useTrail() to animate React components based on state changes rather than fixed-duration timelines.
