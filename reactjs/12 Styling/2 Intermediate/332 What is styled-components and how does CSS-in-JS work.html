332. What is styled-components and How Does CSS-in-JS Work?

One-Line Definition

styled-components is a CSS-in-JS library that allows developers to write CSS inside JavaScript files. CSS-in-JS works by generating CSS dynamically and injecting it into the DOM at runtime.

What is CSS-in-JS?

CSS-in-JS means writing CSS directly inside JavaScript instead of separate .css files.

Traditional Approach

File: src/components/Button.css
.btn {
  background: royalblue;
  color: white;
}
File: src/components/Button.jsx
import './Button.css';

function Button() {
  return <button className="btn">Click Me</button>;
}

export default Button;

CSS-in-JS Approach

File: src/components/Button.jsx
import styled from 'styled-components';

const Button = styled.button`
  background: royalblue;
  color: white;
`;

export default Button;
CSS and component logic live in the same file.

How CSS-in-JS Works Internally

Write CSS In JavaScript
↓
styled-components Parses CSS
↓
Generates Unique Class Name
↓
Injects CSS Into DOM
↓
Component Receives Class Name

What You Write

File: src/components/Button.jsx
import styled from 'styled-components';

const Button = styled.button`
  background: royalblue;
  color: white;
`;

export default Button;

What styled-components Generates

Generated HTML: 

<button class="sc-kd92hf">
  Click Me
</button>

Generated CSS: 

.sc-kd92hf {
  background: royalblue;
  color: white;
}
Class names are automatically unique.

Basic Example

File: src/components/Button.jsx
import styled from 'styled-components';

const Button = styled.button`
  background: royalblue;
  color: white;
  border: none;
  padding: 12px 20px;
  border-radius: 8px;
`;

export default Button;
File: src/App.jsx
import Button from './components/Button';

function App() {
  return <Button>Save</Button>;
}

export default App;

Dynamic Styling With Props

File: src/components/Button.jsx
import styled from 'styled-components';

const Button = styled.button`
  background: ${props => props.primary ? 'royalblue' : 'gray'};
  color: white;
  padding: 12px 20px;
`;

export default Button;
File: src/App.jsx
import Button from './components/Button';

function App() {
  return (
    <>
      <Button primary>Primary</Button>
      <Button>Secondary</Button>
    </>
  );
}

export default App;

Why CSS-in-JS Became Popular

  • No class name conflicts
  • Component-scoped styles
  • Dynamic styling with props
  • Reusable UI components
  • Easier maintenance

Real World Example

File: src/components/UserCard.jsx
import styled from 'styled-components';

const Card = styled.div`
  padding: 20px;
  border-radius: 10px;
  background: #1e293b;
`;

const Name = styled.h3`
  color: #38bdf8;
`;

function UserCard() {
  return (
    <Card>
      <Name>Rizwan</Name>
    </Card>
  );
}

export default UserCard;

Interview Answer

styled-components is a CSS-in-JS library that allows developers to write CSS inside JavaScript using template literals. CSS-in-JS works by generating unique class names, creating CSS dynamically, and injecting those styles into the DOM at runtime. This provides scoped styles, reusable components, and dynamic styling through props.

One-Line Interview Answer

styled-components is a CSS-in-JS library that generates unique CSS classes and injects styles into the DOM at runtime.
