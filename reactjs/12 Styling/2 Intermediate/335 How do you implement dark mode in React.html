335. How do you implement dark mode in React?

One-Line Definition

Dark mode in React is implemented by storing the selected theme in state, applying a CSS class or data attribute, and optionally saving the preference in localStorage.

How It Works

User Clicks Toggle
↓
Update Theme State
↓
Add dark / light Class
↓
CSS Variables Change Colors
↓
Save Theme In localStorage

Project Structure

src/

Step 1 - Create Theme Toggle Component

File: src/components/ThemeToggle.jsx
function ThemeToggle({ theme, toggleTheme }) {
  return <button onClick={toggleTheme}>{theme === 'dark' ? 'Light Mode' : 'Dark Mode'}</button>;
}

export default ThemeToggle;

Step 2 - Use Theme State

File: src/App.jsx
import { useEffect, useState } from 'react';
import ThemeToggle from './components/ThemeToggle';
import './App.css';

function App() {
  const [theme, setTheme] = useState('light');

  function toggleTheme() {
    setTheme(theme === 'light' ? 'dark' : 'light');
  }

  useEffect(() => {
    document.body.className = theme;
    localStorage.setItem('theme', theme);
  }, [theme]);

  return (
    <div className="app">
      <ThemeToggle theme={theme} toggleTheme={toggleTheme} />
      <h1>Dark Mode Example</h1>
      <p>Current theme: {theme}</p>
    </div>
  );
}

export default App;

Problem With This Version

The theme resets to light after page refresh because the initial state is always 'light'.

Better Version With localStorage

File: src/App.jsx
import { useEffect, useState } from 'react';
import ThemeToggle from './components/ThemeToggle';
import './App.css';

function App() {
  const [theme, setTheme] = useState(() => localStorage.getItem('theme') || 'light');

  function toggleTheme() {
    setTheme(prev => prev === 'light' ? 'dark' : 'light');
  }

  useEffect(() => {
    document.body.className = theme;
    localStorage.setItem('theme', theme);
  }, [theme]);

  return (
    <div className="app">
      <ThemeToggle theme={theme} toggleTheme={toggleTheme} />
      <h1>Dark Mode Example</h1>
      <p>Current theme: {theme}</p>
    </div>
  );
}

export default App;
Now the selected theme stays saved even after page refresh.

Recommended Approach - CSS Variables

Instead of writing separate styles everywhere, use CSS variables.

File: src/App.css
body.light {
  --bg: #ffffff;
  --text: #111827;
  --card: #f3f4f6;
  --button: #2563eb;
}

body.dark {
  --bg: #111827;
  --text: #f9fafb;
  --card: #1f2937;
  --button: #38bdf8;
}

body {
  background: var(--bg);
  color: var(--text);
}

.card {
  background: var(--card);
  padding: 20px;
  border-radius: 10px;
}

button {
  background: var(--button);
  color: white;
  border: none;
  padding: 10px 16px;
}

Using CSS Variables In Component

File: src/App.jsx
import { useEffect, useState } from 'react';
import ThemeToggle from './components/ThemeToggle';
import './App.css';

function App() {
  const [theme, setTheme] = useState(() => localStorage.getItem('theme') || 'light');

  function toggleTheme() {
    setTheme(prev => prev === 'light' ? 'dark' : 'light');
  }

  useEffect(() => {
    document.body.className = theme;
    localStorage.setItem('theme', theme);
  }, [theme]);

  return (
    <div className="app">
      <ThemeToggle theme={theme} toggleTheme={toggleTheme} />

      <div className="card">
        <h2>User Card</h2>
        <p>This card changes color based on theme.</p>
      </div>
    </div>
  );
}

export default App;

Dark Mode With data-theme

Many projects prefer data-theme instead of body class.

File: src/App.jsx
useEffect(() => {
  document.documentElement.setAttribute('data-theme', theme);
  localStorage.setItem('theme', theme);
}, [theme]);
File: src/App.css
:root {
  --bg: #ffffff;
  --text: #111827;
}

[data-theme="dark"] {
  --bg: #111827;
  --text: #f9fafb;
}

body {
  background: var(--bg);
  color: var(--text);
}

Why localStorage?

↓
Save "dark" In localStorage
↓
User Refreshes Page
↓

↓

Interview Answer

Dark mode in React can be implemented by keeping the current theme in state, toggling between light and dark themes, applying a class or data-theme attribute to the document, and using CSS variables to change colors. localStorage is commonly used to persist the user's selected theme after refresh.

One-Line Interview Answer
