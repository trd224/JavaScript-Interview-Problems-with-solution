339. What is the AnimatePresence Component in Framer Motion?

One-Line Definition

AnimatePresence is a Framer Motion component that enables exit animations when React components are removed from the DOM.

The Problem It Solves

Normally when React removes a component, it disappears immediately.

File: src/components/Modal.jsx
function Modal({ show }) {
  return (
    <>
      {show && <div>Modal</div>}
    </>
  );
}

export default Modal;
show = true
↓
Modal Visible
↓
show = false
↓
React Removes Component Immediately
No fade out animation is possible because React removes the component instantly.

Solution: AnimatePresence

show = true
↓
Modal Visible
↓
show = false
↓
AnimatePresence Delays Removal
↓
exit Animation Runs
↓
Component Removed

Basic Example

File: src/components/Modal.jsx
import { motion, AnimatePresence } from 'framer-motion';

function Modal({ show }) {
  return (
    <AnimatePresence>
      {show && (
        <motion.div
          initial={{ opacity: 0 }}
          animate={{ opacity: 1 }}
          exit={{ opacity: 0 }}
        >
          Modal Content
        </motion.div>
      )}
    </AnimatePresence>
  );
}

export default Modal;

Use Component

File: src/App.jsx
import { useState } from 'react';
import Modal from './components/Modal';

function App() {
  const [show, setShow] = useState(false);

  return (
    <>
      <button onClick={() => setShow(!show)}>
        Toggle Modal
      </button>

      <Modal show={show} />
    </>
  );
}

export default App;

Understanding The Animation

initial={{ opacity: 0 }}

Start hidden. 

animate={{ opacity: 1 }}

Fade in. 

exit={{ opacity: 0 }}

Fade out before removal.

Without AnimatePresence

File: src/components/Modal.jsx
import { motion } from 'framer-motion';

function Modal({ show }) {
  return (
    <>
      {show && (
        <motion.div
          exit={{ opacity: 0 }}
        >
          Modal
        </motion.div>
      )}
    </>
  );
}

export default Modal;
exit animation will NOT run.

Reason:

show = false
↓
React Removes Component
↓
No Time For exit Animation

With AnimatePresence

show = false
↓
AnimatePresence Detects Removal
↓
Runs exit Animation
↓
Animation Finishes
↓
Component Removed

Important Rule

AnimatePresence must wrap the component that is conditionally rendered.

Correct: 

<AnimatePresence>
  {show && <motion.div exit={{ opacity: 0 }} />}
</AnimatePresence>

Wrong: 

{show && (
  <AnimatePresence>
    <motion.div exit={{ opacity: 0 }} />
  </AnimatePresence>
)}

Most Important Props Used With AnimatePresence

Prop Purpose
initial Start state
animate Visible state
exit Leave state
transition Animation timing

Easy Way To Remember

React Removes Component
↓
AnimatePresence Stops Removal
↓
Runs exit Animation
↓
Then Removes Component

Interview Answer

AnimatePresence is a Framer Motion component that enables exit animations for components being removed from the React tree. It works by temporarily keeping the component mounted, running the exit animation, and only removing the component after the animation completes.

One-Line Interview Answer

AnimatePresence allows Framer Motion exit animations to run before React removes a component from the DOM.
