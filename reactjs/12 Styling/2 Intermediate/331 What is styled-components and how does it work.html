331. What is styled-components and How Does it Work?

One-Line Definition

styled-components is a CSS-in-JS library that allows you to write CSS directly inside JavaScript and create reusable styled React components.

Why Was styled-components Created?

Traditional React styling often uses:

  • CSS files
  • CSS Modules
  • Inline styles

These approaches can cause:

  • Class name conflicts
  • Large CSS files
  • Difficulty finding component styles
styled-components keeps component code and styles together.

Install Package

npm install styled-components

Project Structure

src/
│
├── App.jsx
└── components/
    └── Button.jsx

Basic Example

File: src/components/Button.jsx
import styled from 'styled-components';

const Button = styled.button`
  background: royalblue;
  color: white;
  border: none;
  padding: 12px 20px;
  border-radius: 8px;
`;

export default Button;

Use Component

File: src/App.jsx
import Button from './components/Button';

function App() {
  return <Button>Click Me</Button>;
}

export default App;

How It Works

styled.button
↓
Creates React Component
↓
Generates Unique CSS Class
↓
Injects CSS Into DOM
↓
Applies Style Automatically

What Happens Internally?

You write: 

const Button = styled.button`
  background: royalblue;
`;

styled-components generates something like: 

<button class="sc-a1b2c3">
  Click Me
</button>

And injects: 

.sc-a1b2c3 {
  background: royalblue;
}
Class names are automatically unique, preventing CSS conflicts.

Passing Props To Styles

File: src/components/Button.jsx
import styled from 'styled-components';

const Button = styled.button`
  background: ${props => props.primary ? 'royalblue' : 'gray'};
  color: white;
  padding: 12px 20px;
`;

export default Button;
File: src/App.jsx
import Button from './components/Button';

function App() {
  return (
    <>
      <Button primary>Primary</Button>
      <Button>Secondary</Button>
    </>
  );
}

export default App;

Conditional Styling

File: src/components/Button.jsx
import styled from 'styled-components';

const Button = styled.button`
  background: ${props => props.danger ? 'crimson' : 'green'};
`;

export default Button;

Extending Existing Styles

File: src/components/Button.jsx
import styled from 'styled-components';

export const Button = styled.button`
  padding: 12px;
`;

export const DangerButton = styled(Button)`
  background: crimson;
  color: white;
`;

Styled Input Example

File: src/components/Input.jsx
import styled from 'styled-components';

const Input = styled.input`

Real World Example

File: src/components/LoginForm.jsx
import styled from 'styled-components';

const Wrapper = styled.div`
  max-width: 400px;
`;

const Input = styled.input`
  width: 100%;
  padding: 12px;
  margin-bottom: 10px;
`;

const Button = styled.button`
  background: royalblue;
  color: white;
  padding: 12px;
  border: none;
`;

function LoginForm() {
  return (
    <Wrapper>
      <Input placeholder="Email" />
      <Input type="password" placeholder="Password" />
      <Button>Login</Button>
    </Wrapper>
  );
}

export default LoginForm;

Advantages

  • Component-scoped styles
  • No class name conflicts
  • Dynamic styling with props
  • Reusable components
  • Styles and component together

Disadvantages

  • Extra runtime overhead
  • Larger bundle size
  • Can be slower than plain CSS
  • Less popular today than Tailwind

styled-components vs CSS Modules

styled-components CSS Modules
CSS inside JS Separate CSS file
Supports dynamic props Limited dynamic styling
Runtime CSS generation Build-time CSS
More JavaScript More CSS

styled-components vs Tailwind

styled-components Tailwind
Write CSS Use utility classes
Runtime styling Build-time styling
More flexible Usually faster
Component style files Utility-first approach

Interview Answer

styled-components is a CSS-in-JS library that allows developers to create styled React components using JavaScript template literals. It generates unique class names automatically, supports dynamic styling through props, and keeps styles scoped to individual components.

One-Line Interview Answer

styled-components is a CSS-in-JS library that lets you write CSS directly inside React components and automatically generates unique class names.
