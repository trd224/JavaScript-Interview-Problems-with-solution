338. What is Framer Motion and How Does it Work?

One-Line Definition

Framer Motion is a React animation library that makes it easy to create animations, page transitions, gestures, and interactive UI effects using simple React components.

Why Do We Need Framer Motion?

Creating animations using only CSS can become difficult when animations depend on React state, component mounting, unmounting, or user interactions.

  • Fade In / Fade Out
  • Page Transitions
  • Modal Animations
  • Drag & Drop
  • Hover Effects
  • Scroll Animations
Framer Motion gives React-friendly animations with very little code.

Basic Animation Example

File: src/components/FadeBox.jsx
import { motion } from 'framer-motion';

function FadeBox() {
  return (
    <motion.div
      initial={{ opacity: 0 }}
      animate={{ opacity: 1 }}
      transition={{ duration: 1 }}
    >
      Hello Framer Motion
    </motion.div>
  );
}

export default FadeBox;

Understanding The Props

Prop Purpose
initial Starting animation state
animate Final animation state
transition Animation timing

AnimatePresence (Exit Animations)

File: src/components/Modal.jsx
import { motion, AnimatePresence } from 'framer-motion';

function Modal({ show }) {
  return (
    <AnimatePresence>
      {show && (
        <motion.div
          initial={{ opacity: 0 }}
          animate={{ opacity: 1 }}
          exit={{ opacity: 0 }}
        >
          Modal Content
        </motion.div>
      )}
    </AnimatePresence>
  );
}

export default Modal;
Without AnimatePresence, exit animations do not work because React removes the component immediately.

Interview Answer

Framer Motion is a React animation library that simplifies animations, gestures, drag-and-drop interactions, and page transitions. It works by replacing normal HTML elements with motion components and controlling animations using props such as initial, animate, exit, and transition.
