452. What is the aria-label Attribute?

One-Line Definition

aria-label is an ARIA attribute that provides an accessible name for an element when no visible text label is available.

What is aria-label?

Screen readers need a descriptive name to announce elements to users.

Sometimes an element has no visible text, such as an icon button.

In those cases, we can use aria-label to provide a text description that is read by screen readers.

How aria-label Works

User Uses Screen Reader
↓
Screen Reader Finds Element
↓
Reads aria-label Value
↓
User Understands Purpose

Problem Without aria-label

<button>

  ✖

</button>
A screen reader may only announce:

"button"

The user does not know what the button does.

Using aria-label

<button aria-label="Close Modal">

  ✖

</button>
Now the screen reader announces:

"Close Modal Button"

Common Use Cases

Element Why aria-label Is Needed
Icon Button No visible text
Search Input No associated label
Close Button Contains only an icon
Menu Button Contains only a menu icon
Custom Controls Need accessible names

Example: Search Input

Bad

<input type="search" />

Good

<input type="search" aria-label="Search Products" />

Example: Icon Button

<button aria-label="Open Menu">

  ☰

</button>

React Example

function CloseButton() {

  return (

    <button aria-label="Close Dialog">

      ✖

    </button>

  );

}

export default CloseButton;

aria-label vs label Element

label Element aria-label
Visible to users Usually invisible
Preferred for forms Fallback when no visible label exists
Visible on screen Read mainly by assistive technologies

Preferred Approach

Best

<label htmlFor="email">

  Email Address

</label>

<input id="email" type="email" />

When Visible Label Is Not Possible

<input type="email" aria-label="Email Address" />
Use a real label whenever possible. Use aria-label when there is no visible label.

Common Mistake

<button

  aria-label="Delete User"

>

  Delete User

</button>
This is usually unnecessary because the visible text "Delete User" already provides an accessible name.

Real Project Example

In our Aadhaar Authentication application, aria-label can be used for close icons, menu buttons, help icons, modal popup controls, search fields, and action buttons that contain only icons.

Important Interview Points

  • aria-label provides an accessible name.
  • It is mainly used by screen readers.
  • Useful for icon-only elements.
  • Use visible labels when possible.
  • Part of ARIA accessibility standards.

Short Interview Answer

aria-label provides a text description for elements that do not have visible labels, helping screen readers identify their purpose.

One-Line Interview Answer

aria-label gives an element an accessible name that screen readers can announce to users.
