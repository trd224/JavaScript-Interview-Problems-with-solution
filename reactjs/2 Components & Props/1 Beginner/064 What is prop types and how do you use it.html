064. What is PropTypes and how do you use it?

One-Line Definition

PropTypes is a React library used to validate component props and detect incorrect prop types during development.

Why PropTypes Is Used

Components receive props from parent components.

Sometimes developers may pass:

  • Wrong data types
  • Missing props
  • Invalid values

PropTypes helps detect these problems.

Install PropTypes

npm install prop-types

Basic Example

import PropTypes from 'prop-types';

function User({ name, age }) {

  return (

    <div>

      <h1>{name}</h1>

      <p>{age}</p>

    </div>

  );

}

User.propTypes = {

  name: PropTypes.string,
  age: PropTypes.number

};

What Happens Here?

Prop Expected Type
name string
age number

Wrong Usage Example

<User name={100} age="25" />

Console Warning


Warning:
Invalid prop `name` of type `number`
supplied to `User`, expected `string`

Important Note

PropTypes only show warnings in development mode. They do NOT stop application execution.

Common PropTypes

PropType Description
PropTypes.string String value
PropTypes.number Number value
PropTypes.bool Boolean value
PropTypes.array Array value
PropTypes.object Object value
PropTypes.func Function
PropTypes.node Renderable content
PropTypes.element React element

Required Props

Use: 

.isRequired

to make props mandatory.

Example

User.propTypes = {

  name: PropTypes.string.isRequired,
  age: PropTypes.number.isRequired

};

Missing Prop Warning


Warning:
The prop `name` is marked as required
but its value is undefined

Default Props Example

function User({ name }) {

  return <h1>{name}</h1>;

}

User.defaultProps = {

  name: 'Guest'

};

Advanced Object Validation

User.propTypes = {

  user: PropTypes.shape({

    name: PropTypes.string,
    age: PropTypes.number

  })

};

Array Validation

User.propTypes = {

  users: PropTypes.arrayOf(PropTypes.string)

};

Why PropTypes Is Helpful

  • Detects bugs early
  • Improves maintainability
  • Documents component APIs
  • Helps team collaboration
  • Makes debugging easier

PropTypes vs TypeScript

PropTypes TypeScript
Runtime validation Compile-time validation
Simple setup Full type system
Development warnings Compilation errors
JavaScript-based Type-safe development

Modern React Trend

Many modern React projects prefer TypeScript instead of PropTypes for stronger type safety.

Real Project Use Cases

Component Validation Usage
OTP component Validate callback functions
User card Validate user object
Button component Validate event handlers
Dashboard widgets Validate API response data

Important Interview Points

  • PropTypes validates component props.
  • Used only during development.
  • Shows warnings for invalid props.
  • .isRequired makes props mandatory.
  • TypeScript is now more commonly preferred.

Short Interview Answer

PropTypes is a React validation library used to check component props for correct data types and required values during development.
