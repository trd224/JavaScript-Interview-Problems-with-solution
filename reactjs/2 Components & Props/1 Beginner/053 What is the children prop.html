053. What is the children prop?

One-Line Definition

The children prop is a special React prop used to pass content between the opening and closing tags of a component.

Simple Understanding

Whatever is written inside a component automatically becomes:

props.children

Basic Example

<Card>

  <h1>Hello React</h1>

</Card>

Here: 

<h1>Hello React</h1>

becomes: 

props.children

Receiving children Prop

function Card(props) {

  return (

    <div className="card">

      {props.children}

    </div>

  );

}

Output

Hello React

Flow Diagram

Content Inside Component
↓
React Converts It
↓
props.children
↓
Child Component Renders It

Why children Prop Is Useful

  • Create reusable wrapper components
  • Build layouts
  • Create modals and cards
  • Render dynamic nested content

Using Destructuring

function Card({ children }) {

  return (

    <div className="card">

      {children}

    </div>

  );

}

children Can Contain Multiple Elements

<Card>

  <h1>Title</h1>

  <p>Description</p>

</Card>

What children Contains

children can contain:

Type Example
Text Hello
HTML elements <h1>
Components <User />
Arrays List items
Expressions {count}

children with Components

<Layout>

  <Sidebar />

  <Dashboard />

</Layout>

Layout Component

function Layout({ children }) {

  return (

    <main>

      {children}

    </main>

  );

}

children Is Automatically Passed

Developers do NOT manually write: 

children={...}

React automatically creates it.

Important Difference

Normal Props children Prop
Passed as attributes Passed between tags
<User name="Rizwan" /> <Card>Content</Card>
Explicit props Automatic special prop

Real Project Use Cases

Component children Usage
Modal Dynamic popup content
Card Flexible card body
Layout Page content rendering
Authentication wrapper Nested forms

Important Note

children is one of the most important React composition features.

Important Interview Points

  • children is a special React prop.
  • Contains content between component tags.
  • Used for reusable wrapper components.
  • Automatically passed by React.
  • Can contain elements, text, or components.

Short Interview Answer

The children prop is a special React prop that contains whatever content is placed between a component’s opening and closing tags.

One-Line Interview Answer

children is a special React prop used to render nested content inside components.
