248. How do you use Apollo Client with React?

One-Line Definition

Apollo Client is a GraphQL client library used in React to fetch, update, cache, and manage GraphQL API data.

Why Is Apollo Client Used?

Apollo Client helps our React application work with GraphQL APIs easily.

  • Send GraphQL queries.
  • Send GraphQL mutations.
  • Manage loading state.
  • Manage error state.
  • Cache API responses.
  • Update UI automatically.
Without Apollo Client, we need to manually use fetch, manage state, handle errors, and manage cache.

Install Apollo Client

npm install @apollo/client graphql

Step 1: Create Apollo Client

src/apolloClient.js

import { ApolloClient, InMemoryCache } from '@apollo/client';

const client = new ApolloClient({
  uri: 'https://api.example.com/graphql',
  cache: new InMemoryCache()
});

export default client;

Step 2: Wrap App With ApolloProvider

src/main.jsx

import React from 'react';
import ReactDOM from 'react-dom/client';
import { ApolloProvider } from '@apollo/client';
import client from './apolloClient';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')).render(
  <ApolloProvider client={client}>
    <App />
  </ApolloProvider>
);
ApolloProvider makes Apollo Client available to all React components.

Step 3: Write GraphQL Query

src/graphql/queries.js

import { gql } from '@apollo/client';

export const GET_USERS = gql`
  query GetUsers {
    users {
      id
      name
      email
    }
  }
`;

Step 4: Use Query in React Component

src/components/Users.jsx

import { useQuery } from '@apollo/client';
import { GET_USERS } from '../graphql/queries';

function Users() {
  const { data, loading, error } = useQuery(GET_USERS);

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error: {error.message}</p>;

  return (
    <ul>
      {data.users.map(user => (
        <li key={user.id}>
          {user.name} - {user.email}
        </li>
      ))}
    </ul>
  );
}

export default Users;

Mutation Example

Mutation means changing data on the server. 

Create user
Update user
Delete user
Login user

src/graphql/mutations.js

import { gql } from '@apollo/client';

export const ADD_USER = gql`
  mutation AddUser($name: String!, $email: String!) {
    addUser(name: $name, email: $email) {
      id
      name
      email
    }
  }
`;

Use Mutation in React

import { useMutation } from '@apollo/client';
import { ADD_USER } from '../graphql/mutations';

function AddUser() {
  const [addUser, { loading, error }] = useMutation(ADD_USER);

  function handleAdd() {
    addUser({
      variables: {
        name: 'Rizwan',
        email: 'rizwan@gmail.com'
      }
    });
  }

  return (
    <>
      <button onClick={handleAdd} disabled={loading}>
        Add User
      </button>

      {error && <p>{error.message}</p>}
    </>
  );
}

export default AddUser;

Query With Variables

import { gql } from '@apollo/client';

const GET_USER = gql`
  query GetUser($id: ID!) {
    user(id: $id) {
      id
      name
      email
    }
  }
`;
const { data, loading, error } = useQuery(GET_USER, {
  variables: {
    id: 1
  }
});

Refetch Data

const { data, loading, error, refetch } = useQuery(GET_USERS);

<button onClick={() => refetch()}>
  Refresh
</button>

Add Authorization Header

For protected GraphQL APIs, we can pass token headers. 

import { ApolloClient, InMemoryCache, createHttpLink } from '@apollo/client';
import { setContext } from '@apollo/client/link/context';

const httpLink = createHttpLink({
  uri: 'https://api.example.com/graphql'
});

const authLink = setContext((_, { headers }) => {
  const token = localStorage.getItem('token');

  return {
    headers: {
      ...headers,
      Authorization: token ? `Bearer ${token}` : ''
    }
  };
});

const client = new ApolloClient({
  link: authLink.concat(httpLink),
  cache: new InMemoryCache()
});

export default client;

Important Interview Points

  • Apollo Client connects React with GraphQL APIs.
  • ApolloProvider provides Apollo Client to the React app.
  • useQuery is used to fetch data.
  • useMutation is used to create, update, or delete data.
  • gql is used to write GraphQL queries and mutations.
  • Apollo manages loading, error, data, and cache.
  • Apollo cache helps reduce unnecessary network calls.

Short Interview Answer

Apollo Client is used in React by creating an ApolloClient instance with a GraphQL endpoint and cache, wrapping the app with ApolloProvider, and then using hooks like useQuery and useMutation inside components to fetch and update GraphQL data.

One-Line Interview Answer

Apollo Client connects React to GraphQL APIs using ApolloProvider, useQuery, and useMutation while managing loading, errors, and cache automatically.
