160. What is the use() Hook in React 19?

One-Line Answer

The use() hook allows React to directly read a Promise or Context during rendering. If the Promise is still pending, React automatically pauses (suspends) rendering and waits for the Promise to finish.

Why Was use() Introduced?

Before React 19, loading async data usually required:

  • useState
  • useEffect
  • Loading State
  • Error State
Before React 19
function User() {

  const [user, setUser] = useState(null);

  useEffect(() => {
    fetch('/api/user')
      .then(res => res.json())
      .then(data => setUser(data));
  }, []);

  if (!user) return <h2>Loading...</h2>;

  return <h2>{user.name}</h2>;

}
Render
↓ useEffect Runs
↓ Fetch Starts
↓ Loading State
↓ Data Arrives
↓ setUser()
↓ Re-render
Lots of boilerplate.

React 19 Solution

Using use()
import { use } from 'react';

function User({ userPromise }) {

  const user = use(userPromise);

  return <h2>{user.name}</h2>;

}

How use() Works Internally

When React sees: 

use(userPromise)

React asks:

Promise Resolved?
↓ No
↓ Suspend Rendering
↓ Show Suspense Fallback
↓ Promise Resolves
↓ Continue Rendering
React handles the waiting automatically.

Relationship With Suspense

Just like React.lazy(), use() works together with Suspense.

Complete Example
import { Suspense, use } from 'react';

const userPromise = fetch('/api/user')
                    .then(res => res.json());

function User({ userPromise }) {

  const user = use(userPromise);

  return <h2>{user.name}</h2>;

}

export default function App() {

  return (
    <Suspense fallback={
      <h2>Loading...</h2>
    }>
      <User userPromise={userPromise} />
    </Suspense>
  );

}

App Renders
↓ User Component Renders
↓ use(userPromise)
↓ Promise Pending
↓ Suspend Component
↓ Suspense Shows Loading...
↓ Promise Resolves
↓ Continue Rendering
↓ Show User Data

Context Support

use() can also read Context.

ThemeContext.jsx
import { createContext } from 'react';

export const ThemeContext = createContext('light');
Child.jsx
import { use } from 'react';
import { ThemeContext } from './ThemeContext';

function Child() {

  const theme = use(ThemeContext);

  return <h2>{theme}</h2>;

}

Equivalent to: 

const theme = useContext(ThemeContext);

Interview Answer

The use() hook introduced in React 19 allows components to directly read Promises and Context values during rendering. When used with a Promise, React automatically suspends rendering if the Promise is pending and resumes once it resolves. It works together with Suspense, which shows fallback UI while waiting. This reduces the need for useEffect, loading state, and manual async handling, making asynchronous data fetching much simpler, especially in Server Components and Suspense-based applications.
