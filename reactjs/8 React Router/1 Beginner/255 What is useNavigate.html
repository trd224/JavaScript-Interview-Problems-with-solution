254. What is useNavigate and How is it Used?

One-Line Definition

useNavigate is a React Router hook used to navigate to another route programmatically from JavaScript code.

Why Do We Need useNavigate?

The Link component is used when users manually click a link.

Sometimes we need navigation after an action such as:

  • Login success
  • Form submission
  • Logout
  • Delete operation
  • Payment completion
In these situations, navigation must happen from JavaScript code instead of a Link component.

Importing useNavigate

import { useNavigate } from 'react-router-dom';

Basic Usage

import { useNavigate } from 'react-router-dom';

function Login() {

  const navigate = useNavigate();

}
useNavigate returns a navigate function that can change routes programmatically.

Navigate to Another Page

const navigate = useNavigate();

navigate("/dashboard");
Current Page
↓
navigate("/dashboard")
↓
Dashboard Page Opens

Complete Example

import { useNavigate } from 'react-router-dom';

function Login() {

  const navigate = useNavigate();

  function handleLogin() {

    // Login successful

    navigate("/dashboard");

  }

  return (
    <button onClick={handleLogin}>
      Login
    </button>
  );

}

Login Flow Example

User Clicks Login
↓
API Call

Navigating with Button Click

const navigate = useNavigate();

<button onClick={() => navigate("/about")}>
  About
</button>

Navigate Back

const navigate = useNavigate();

navigate(-1);
Similar to clicking the browser Back button.

Navigate Forward

const navigate = useNavigate();

navigate(1);
Similar to clicking the browser Forward button.

Replace Current History Entry

navigate("/dashboard", {
  replace: true
});
The current page is replaced in browser history, so users cannot return to it using the Back button.

Logout Example

function handleLogout() {

  localStorage.removeItem("token");

  navigate("/login", {
    replace: true
  });

}

Link vs useNavigate

Feature Link useNavigate
Manual User Navigation ✅ Yes ❌ No
Programmatic Navigation ❌ No ✅ Yes
Used in JavaScript Logic ❌ No ✅ Yes
Used After API Calls ❌ No ✅ Yes

Real Project Example

In our Aadhaar Authentication application, after successful login, OTP verification, audit creation, or logout, we can use useNavigate to redirect users to Dashboard, Audit Details, Reports, or Login pages automatically.

Important Interview Points

  • useNavigate is a React Router hook.
  • It returns a navigate function.
  • It is used for programmatic navigation.
  • navigate("/path") navigates to a route.
  • navigate(-1) goes back.
  • navigate(1) goes forward.
  • replace: true replaces the current history entry.

Short Interview Answer

useNavigate is a React Router hook that allows navigation through JavaScript code, making it useful after login, form submission, logout, or API operations.

One-Line Interview Answer

useNavigate is used to navigate between routes programmatically in React Router.
