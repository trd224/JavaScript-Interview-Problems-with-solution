253. What is the Link Component?

One-Line Definition

Link is a React Router component used to navigate between routes without reloading the browser page.

Why Do We Need Link?

In a React Single Page Application (SPA), we want to move between pages without refreshing the browser.

React Router provides the Link component for this purpose.

Importing Link

import { Link } from 'react-router-dom';

Basic Example

import { Link } from 'react-router-dom';

function Navbar() {

  return (

    <nav>

      <Link to="/">Home</Link>

      <Link to="/about">About</Link>

      <Link to="/contact">Contact</Link>

    </nav>

  );

}

What Does the to Prop Mean?

<Link to="/about">

  About

</Link>
The to prop specifies the route where the user should be navigated.
Link Destination
to="/" Home Page
to="/about" About Page
to="/contact" Contact Page

Anchor Tag Example

<a href="/about">

  About

</a>
Clicking an anchor tag causes the browser to request the server again and reload the entire page.

Link Example

<Link to="/about">

  About

</Link>
Clicking Link changes the URL and renders the new route without reloading the browser.

Real Project Example

In our Aadhaar Authentication application, Link can be used to navigate between Dashboard, Audit Management, Reports, Device Registration, User Management, and Settings pages without reloading the browser.

Important Interview Points

  • Link is provided by react-router-dom.
  • It is used for client-side navigation.
  • The to prop specifies the destination route.
  • Link prevents full page reloads.
  • Link is preferred over anchor tags for internal navigation.
  • Link preserves React application state.

Short Interview Answer

Link is a React Router component that navigates between routes by updating the URL and rendering the appropriate component without refreshing the browser page.

One-Line Interview Answer

Link is used to navigate between React Router routes without reloading the browser.
