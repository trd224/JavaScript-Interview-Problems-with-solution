257. What is useParams?

One-Line Definition

useParams is a React Router hook used to read dynamic values from the URL.

Why Do We Need useParams?

Many applications have pages whose content depends on an ID or other value in the URL.

Examples: 

/users/101

/products/25

/orders/500
The values 101, 25, and 500 are dynamic route parameters. useParams allows us to access them.

Real World Example

URL Parameter Value
/users/101 101
/users/102 102
/products/50 50

Step 1 - Define a Dynamic Route

import { Routes, Route } from 'react-router-dom';

<Routes>

  <Route path="/users/:id" element={<UserDetails />} />

</Routes>
The colon (:) indicates a dynamic route parameter.

Step 2 - Read the Parameter

import { useParams } from 'react-router-dom';

function UserDetails() {

  const params = useParams();

  console.log(params);

}

Output

If the URL is: 

/users/101

Then: 

{
  id: "101"
}

Using Destructuring

const { id } = useParams();

console.log(id);

Output: 

101

Complete Example

import { useParams } from 'react-router-dom';

function UserDetails() {

  const { id } = useParams();

  return (

    <h2>

      User ID: {id}

    </h2>

  );

}

Route Matching Flow

URL
/users/101
↓
Route
/users/:id
↓
useParams()
↓

Fetching Data Using the Parameter

import { useEffect, useState } from 'react';
import { useParams } from 'react-router-dom';

function UserDetails() {

  const { id } = useParams();

  const [user, setUser] = useState(null);

  useEffect(() => {

    fetch(`/api/users/${id}`)
      .then(res => res.json())
      .then(data => setUser(data));

  }, [id]);

}
This allows us to load different user records based on the URL.

Multiple Parameters

<Route path="/users/:userId/orders/:orderId" element={<OrderDetails />} />
const { userId, orderId } = useParams();

URL: 

/users/101/orders/500

Output: 

{
  userId: "101",
  orderId: "500"
}

useParams Return Type

useParams always returns an object containing all route parameters as strings. 
const params = useParams();

Example: 

{
  id: "101"
}

Link Example

<Link to="/users/101">

  View User

</Link>

When clicked: 

/users/101

Then: 

const { id } = useParams();

Output: 

101

Real Project Example

In our Aadhaar Authentication application, useParams can be used to open pages such as /audit/1001, /report/50, or /user/25 and load the corresponding audit, report, or user information dynamically.

Important Interview Points

  • useParams is a React Router hook.
  • It reads dynamic values from the URL.
  • Route parameters are defined using a colon (:).
  • useParams returns an object.
  • All parameter values are returned as strings.
  • Commonly used for user, product, order, and detail pages.

Short Interview Answer

useParams is used to access dynamic route parameters from the URL, allowing components to load data based on values such as IDs.

One-Line Interview Answer

useParams reads dynamic route parameters from the URL in React Router.
