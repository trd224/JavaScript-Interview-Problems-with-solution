427. How do you type useRef?

One-Line Definition

In TypeScript, useRef is typed using a generic that specifies the type of value or DOM element the ref will hold.

Basic Syntax

const ref = useRef<Type>(initialValue);

Typing An Input Ref

File: src/components/InputFocus.tsx
import { useRef } from 'react';

function InputFocus() {
  const inputRef = useRef<HTMLInputElement>(null);

  function focusInput() {
    inputRef.current?.focus();
  }

  return (
    <>
      <input ref={inputRef} />
      <button onClick={focusInput}>Focus</button>
    </>
  );
}

export default InputFocus;

Why null?

On the first render, React has not attached the DOM element yet. 

const inputRef = useRef<HTMLInputElement>(null);
Component Render
↓

↓

Typing A Ref For Mutable Values

useRef is not only for DOM elements.

File: src/components/Counter.tsx
import { useRef } from 'react';

function Counter() {
  const renderCount = useRef<number>(0);

  renderCount.current++;

  return <h2>{renderCount.current}</h2>;
}

export default Counter;
No DOM element involved. The ref stores a number.

Typing A Timer ID

File: src/components/Timer.tsx
import { useRef } from 'react';

function Timer() {
  const timerRef = useRef<ReturnType<typeof setInterval> | null>(null);

  function start() {
    timerRef.current = setInterval(() => {
        console.log('Tick');
      }, 1000);
  }

  return <button onClick={start}>Start</button>;
}

export default Timer;

Why Optional Chaining?

This can fail: 

inputRef.current.focus();

Because current might be null.

Use: 

inputRef.current?.focus();
Prevents runtime errors.

Real World Example

File: src/components/Search.tsx
import { useRef } from 'react';

function Search() {
  const inputRef = useRef<HTMLInputElement>(null);

  function clearAndFocus() {
    if (inputRef.current) {
      inputRef.current.value = '';
      inputRef.current.focus();
    }
  }

  return (
    <>
      <input ref={inputRef} />
      <button onClick={clearAndFocus}>
        Clear
      </button>
    </>
  );
}

export default Search;

Interview Answer

useRef is typed using TypeScript generics. For DOM elements, pass the appropriate HTML element type such as HTMLInputElement or HTMLDivElement. For mutable values, pass the value type such as number or string. Example: useRef<HTMLInputElement>(null).

One-Line Interview Answer

Type useRef using generics like useRef<HTMLInputElement>(null) for DOM elements or useRef<number>(0) for mutable values.
