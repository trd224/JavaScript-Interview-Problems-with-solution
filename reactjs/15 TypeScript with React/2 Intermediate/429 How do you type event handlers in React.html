429. How do you type event handlers in React?

One-Line Definition

In React TypeScript, event handlers are typed using React's built-in event types such as ChangeEvent, MouseEvent, KeyboardEvent, FormEvent, and FocusEvent.

Basic Syntax

function handleEvent(
  event: React.EventType<HTMLElement>
) {
  // logic
}

Typing onChange For Input

File: src/components/SearchInput.tsx
import { ChangeEvent } from 'react';

function SearchInput() {

  function handleChange(
    event: ChangeEvent<HTMLInputElement>
  ) {
    console.log(event.target.value);
  }

  return (
    <input
      onChange={handleChange}
    />
  );
}

export default SearchInput;

Typing onClick

File: src/components/Button.tsx

  e: React.MouseEvent<HTMLButtonElement>

Typing Form Submit

File: src/components/LoginForm.tsx
function handleSubmit(
  e: React.FormEvent<HTMLFormElement>
) {
  e.preventDefault();
}

  return (
    <form onSubmit={handleSubmit}>

    </form>
  );

export default LoginForm;

How To Discover Event Type Quickly

Hover over e in VS Code.

React.ChangeEvent<HTMLInputElement>
This is the fastest way to discover event types.

Most Common React Event Types

Event Type
onChange (input) React.ChangeEvent<HTMLInputElement>
onChange (select) React.ChangeEvent<HTMLSelectElement>
onChange (textarea) React.ChangeEvent<HTMLTextAreaElement>
onClick React.MouseEvent<HTMLButtonElement>
onSubmit React.FormEvent<HTMLFormElement>
onKeyDown React.KeyboardEvent<HTMLInputElement>
onFocus React.FocusEvent<HTMLInputElement>
onBlur React.FocusEvent<HTMLInputElement>

Real World Example

File: src/components/LoginForm.tsx
import { useState } from 'react';

function LoginForm() {

  const [email, setEmail] = useState('');

  function handleChange(
    e: React.ChangeEvent<HTMLInputElement>
  ) {
    setEmail(e.target.value);
  }

  function handleSubmit(
    e: React.FormEvent<HTMLFormElement>
  ) {
    e.preventDefault();
    console.log(email);
  }

  return (
    <form onSubmit={handleSubmit}>
      <input
        value={email}
        onChange={handleChange}
      />

      <button>Login</button>
    </form>
  );
}

export default LoginForm;

Interview Answer

Event handlers in React TypeScript are typed using React's event types such as ChangeEvent, MouseEvent, FormEvent, KeyboardEvent, and FocusEvent. The element type is passed as a generic, for example React.ChangeEvent<HTMLInputElement> for input change events.

One-Line Interview Answer

Type React event handlers using built-in event types like React.ChangeEvent<HTMLInputElement> and React.MouseEvent<HTMLButtonElement>.
