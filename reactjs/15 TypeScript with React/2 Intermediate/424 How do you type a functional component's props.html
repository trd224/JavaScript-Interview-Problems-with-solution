424. How do you type a functional component's props?

One-Line Definition

In TypeScript, you type a functional component's props by creating a Props interface or type and using it as the component parameter type.

Project Structure

src/
│
├── App.tsx
└── components/
    └── UserCard.tsx

Recommended Approach (Interface)

File: src/components/UserCard.tsx
interface UserCardProps {
  name: string;
  age: number;
  isAdmin: boolean;
}

function UserCard({ name, age, isAdmin }: UserCardProps) {
  return (
    <div>
      <h2>{name}</h2>
      <p>Age: {age}</p>
      <p>Admin: {isAdmin ? 'Yes' : 'No'}</p>
    </div>
  );
}

export default UserCard;
File: src/App.tsx
import UserCard from './components/UserCard';

function App() {
  return <UserCard name="Rizwan" age={39} isAdmin={true} />;
}

export default App;

Optional Props

File: src/components/UserCard.tsx
interface UserCardProps {
  name: string;
  age?: number;
}

function UserCard({ name, age = 18 }: UserCardProps) {
  return <h2>{name} - {age}</h2>;
}

export default UserCard;

Function Prop

File: src/components/Button.tsx
interface ButtonProps {
  onClick: () => void;
}

function Button({ onClick }: ButtonProps) {
  return <button onClick={onClick}>Click</button>;
}

Function Prop With Parameters

File: src/components/UserList.tsx
interface UserListProps {
  onSelect: (id: number) => void;
}

function UserList({ onSelect }: UserListProps) {
  return (
    <button onClick={() => onSelect(1)}>
      Select User
    </button>
  );
}

export default UserList;

Array Prop

interface UserListProps {
  users: string[];

function UserList({ users }: UserListProps) {

    <ul>
      {users.map(user => <li key={user}>{user}</li>)}
    </ul>

}

export default UserList;

const UserCard: React.FC<UserCardProps> = ({ name }) => {

Most modern React projects prefer normal function components over React.FC.

Modern Recommended Style

File: src/components/UserCard.tsx
interface UserCardProps {
  name: string;
}

function UserCard({ name }: UserCardProps) {
  return <h2>{name}</h2>;
}

export default UserCard;

Easy Way To Remember

Create Interface
↓
Define Props
↓
Apply Interface To Component
↓
Type Safe Component

Interview Answer

A functional component's props are typed by creating an interface or type alias that defines the expected props and then using that type as the component parameter. This provides compile-time checking, IntelliSense, and safer React code.
