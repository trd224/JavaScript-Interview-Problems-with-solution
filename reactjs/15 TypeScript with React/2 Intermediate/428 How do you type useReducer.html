428. How do you type useReducer?

One-Line Definition

To type useReducer in TypeScript, you define types for the State, Action, and Reducer function.

Project Structure

src/
│
├── App.tsx
├── reducers/
│   └── counterReducer.ts
└── types/
    └── counter.ts

Step 1 - Create State Type

File: src/types/counter.ts
export interface CounterState {
  count: number;
}

Step 2 - Create Action Type

File: src/types/counter.ts
export type CounterAction =
  | { type: 'INCREMENT' }
  | { type: 'DECREMENT' }
  | { type: 'RESET' };
Union types are commonly used for reducer actions.

Step 3 - Type Reducer Function

File: src/reducers/counterReducer.ts
import { CounterState, CounterAction } from '../types/counter';

export function counterReducer(state: CounterState, action: CounterAction): CounterState {

  switch (action.type) {

    case 'INCREMENT':
      return { count: state.count + 1 };

    case 'DECREMENT':
      return { count: state.count - 1 };

    case 'RESET':
      return { count: 0 };

    default:
      return state;
  }

}

Step 4 - Use Typed Reducer

File: src/App.tsx
import { useReducer } from 'react';
import { counterReducer } from './reducers/counterReducer';

function App() {

  const [state, dispatch] = useReducer(counterReducer, { count: 0 });

  return (
    <>
      <h2>Count: {state.count}</h2>

      <button onClick={() => dispatch({ type: 'INCREMENT' })}>
        +
      </button>

      <button onClick={() => dispatch({ type: 'DECREMENT' })}>
        -
      </button>

      <button onClick={() => dispatch({ type: 'RESET' })}>
        Reset
      </button>
    </>
  );

}

export default App;

Type Safety Example

✅ Valid 
dispatch({ type: 'INCREMENT' });
❌ Invalid 
dispatch({ type: 'ADD' });

TypeScript Error:

Type '"ADD"' is not assignable
to type '"INCREMENT" | "DECREMENT" | "RESET"'

Reducer With Payload

export type CounterAction =
  | { type: 'INCREMENT'; payload: number }
  | { type: 'DECREMENT'; payload: number }
  | { type: 'RESET' };
File: src/reducers/counterReducer.ts
export function counterReducer(state: CounterState, action: CounterAction): CounterState {

  switch (action.type) {

    case 'INCREMENT':
      return {
        count: state.count + action.payload
      };

    case 'DECREMENT':
      return {
        count: state.count - action.payload
      };

    case 'RESET':
      return { count: 0 };

    default:
      return state;
  }

}

File: src/reducers/loginReducer.ts
import { LoginState, LoginAction } from '../types/login';

export function loginReducer(
  state: LoginState,
  action: LoginAction
): LoginState {

  switch (action.type) {

    case 'SET_EMAIL':
      return { ...state, email: action.payload };

    case 'SET_PASSWORD':
      return { ...state, password: action.payload };

    case 'SET_LOADING':
      return { ...state, loading: action.payload };

    default:
      return state;
  }

}

Interview Answer

To type useReducer in TypeScript, define a State interface, an Action union type, and a reducer function that accepts the typed state and action. TypeScript then automatically infers the dispatch type and provides full type safety.

One-Line Interview Answer

Type useReducer by defining typed State and Action objects and using them in a typed reducer function.
