174. Write a useOnClickOutside Custom Hook

One-Line Definition

useOnClickOutside is a custom hook that detects clicks outside a specific HTML element and executes a callback function.

Why Is It Used?

Many UI components should automatically close when the user clicks outside them.

Examples:

  • Dropdown menus
  • Modal popups
  • Sidebars
  • Profile menus
  • Tooltips
Without this hook, we would need to write document click listeners in every component.

Custom Hook Code

import { useEffect } from 'react';

function useOnClickOutside(ref, handler) {

  useEffect(() => {

    function handleClick(event) {

      if (!ref.current) return;

      if (!ref.current.contains(event.target)) {
        handler();
      }

    }

    document.addEventListener('mousedown', handleClick);

    return () => {
      document.removeEventListener('mousedown', handleClick);
    };

  }, [ref, handler]);

}

export default useOnClickOutside;

Understanding The Parameters

useOnClickOutside(ref, handler);
Parameter Purpose
ref The element we want to monitor.
handler Function to execute when clicked outside.

Most Important Line

if (!ref.current.contains(event.target)) {
  handler();
}

This line checks:

  • Was the click inside the element?
  • Or outside the element?

If the click happened outside: 

handler();

is executed.

Understanding contains()

Suppose: 

<div ref={menuRef}>
  Menu
</div>

User clicks: 

<button>Item</button>

inside the menu. 

menuRef.current.contains(clickedElement)

returns: 

true

User clicks somewhere else: 

<body>

Now: 

menuRef.current.contains(clickedElement)

returns: 

false

Therefore the menu closes.

Complete Example

import { useState, useRef } from 'react';
import useOnClickOutside from './useOnClickOutside';

function App() {

  const [open, setOpen] = useState(false);

  const menuRef = useRef();

  useOnClickOutside(menuRef, () => {
    setOpen(false);
  });

  return (

    <div>

      <button onClick={() => setOpen(true)}>
        Open Menu
      </button>

      {open && (

        <div ref={menuRef}>

          <h3>Dropdown Menu</h3>

          <p>Click outside to close me.</p>

        </div>

      )}

    </div>

  );

}

export default App;

↓

↓
Outside Click?
↓

How ref Works Here

Unlike usePrevious, here the ref IS attached to an HTML element. 

const menuRef = useRef();

Then: 

<div ref={menuRef}>

React stores: 

menuRef.current = HTMLDivElement

Now we can check: 

menuRef.current.contains(event.target)

to determine whether the click happened inside or outside.

This is a DOM ref because it points to a real HTML element.

Real-World Use Cases

  • Close dropdown menu.
  • Close modal popup.
  • Close sidebar.
  • Close user profile menu.
  • Close notification panel.
  • Close tooltip.

Important Interview Points

  • Uses useEffect to register document events.
  • Uses useRef to access a DOM element.
  • Uses contains() to detect outside clicks.
  • Automatically removes event listeners on unmount.
  • Frequently used in dropdowns and modals.

Short Interview Answer

useOnClickOutside is a custom hook that listens for document clicks and runs a callback when the clicked target is outside the referenced element.

One-Line Interview Answer

useOnClickOutside helps close dropdowns, modals, and menus when the user clicks outside them.
