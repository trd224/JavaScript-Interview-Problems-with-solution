462. What is Create React App (CRA)?

One-Line Definition

Create React App (CRA) is a tool that creates and configures a React application with zero manual setup.

What is Create React App?

Before CRA existed, developers had to manually configure:

  • Webpack
  • Babel
  • Development Server
  • Build Configuration
  • Hot Reloading

CRA automatically configured all these tools and provided a ready-to-use React project.

What is Eject?

npm run eject

Eject exposes CRA's hidden Webpack and Babel configuration files.

Eject is irreversible. Once ejected, we cannot go back.

Why CRA Became Less Popular

Current Recommendation

For new React projects, Vite is usually recommended instead of CRA because it provides significantly faster development and build performance.

Create a React Project with Vite

npm create vite@latest my-app -- --template react

cd my-app

npm install

npm run dev

Real Project Example

Older versions of our Aadhaar Authentication application could have been created using CRA. Today, most new React projects use Vite for better performance and developer experience.

Important Interview Points

  • CRA creates React applications with zero configuration.
  • It automatically configures Webpack and Babel.
  • It primarily supports Client-Side Rendering (CSR).
  • Eject exposes hidden configurations.
  • Vite is now the preferred choice for most new React projects.

Short Interview Answer

Create React App is a tool that generates a fully configured React application with Webpack, Babel, and a development server already set up, allowing developers to start building React apps quickly.
