287. What is the register Function in React Hook Form?

One-Line Definition

register() is a React Hook Form function used to connect an input field to the form so RHF can track its value, validation, and submission data.

Why Do We Need register()?

React Hook Form cannot automatically know which inputs belong to the form.

We must register each field.

Input Field
↓
register()
↓
React Hook Form Tracks It
↓
Validation
↓
Form Data

Without register()

File: src/components/LoginForm.jsx
import { useForm } from 'react-hook-form';

function LoginForm() {
  const { handleSubmit } = useForm();

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input placeholder="Email" />
      <button>Submit</button>
    </form>
  );
}

export default LoginForm;

Result: 

{}
React Hook Form does not know about the input because it was never registered.

With register()

File: src/components/LoginForm.jsx
import { useForm } from 'react-hook-form';

function LoginForm() {
  const { register, handleSubmit } = useForm();

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input placeholder="Email" {...register('email')} />
      <button>Submit</button>
    </form>
  );
}

export default LoginForm;

Result: 

{
  email: 'rizwan@gmail.com'
}

What Does register('email') Mean?

{...register('email')}

This tells React Hook Form:

  • Track this field
  • Store its value
  • Validate it
  • Include it in submitted data

How register() Works Internally

register('email')
↓
Adds ref
↓
Adds onChange
↓
Adds onBlur
↓
Tracks Value

register() returns an object similar to: 

{
  name: 'email',
  onChange: fn,
  onBlur: fn,
  ref: fn
}

That object is spread into the input. 

<input {...register('email')} />

Register Multiple Fields

File: src/components/LoginForm.jsx
import { useForm } from 'react-hook-form';

function LoginForm() {
  const { register, handleSubmit } = useForm();

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input placeholder="Name" {...register('name')} />
      <input placeholder="Email" {...register('email')} />
      <input type="password" placeholder="Password" {...register('password')} />
      <button>Submit</button>
    </form>
  );
}

export default LoginForm;

Result: 

{
  name: 'Rizwan',
  email: 'rizwan@gmail.com',
  password: '123456'
}

Validation Using register()

File: src/components/LoginForm.jsx
import { useForm } from 'react-hook-form';

function LoginForm() {
  const { register, handleSubmit, formState: { errors } } = useForm();

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input placeholder="Email" {...register('email', { required: 'Email is required' })} />
      {errors.email && <p>{errors.email.message}</p>}

      <button>Submit</button>
    </form>
  );
}

export default LoginForm;

Common Validation Rules

<input
  {...register('email', {
    required: 'Email is required',
    minLength: {
      value: 5,
      message: 'Minimum 5 characters'
    }
  })}
/>

Compact Style Version

<input {...register('email', { required: 'Email is required' })} />

Interview Answer

The register() function is used to connect an input field to React Hook Form. It allows React Hook Form to track the field's value, perform validation, manage events, and include the field in the submitted form data.

One-Line Interview Answer

register() registers an input with React Hook Form so it can track, validate, and submit that field's value.
