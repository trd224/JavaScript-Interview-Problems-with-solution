297. What is a FieldArray in Formik?

One-Line Definition

FieldArray is a Formik component used to manage dynamic arrays of form fields such as skills, phone numbers, addresses, education records, or work experience.

Why Do We Need FieldArray?

Sometimes we don't know how many fields a user will need.

Examples:

  • Multiple Skills
  • Multiple Phone Numbers
  • Multiple Addresses
  • Work Experience List
  • Education History
Normal Field components work well for fixed inputs. FieldArray is needed for dynamic inputs.

Without FieldArray

Skill 1
Skill 2
Skill 3
↓
Hardcoded Fields

Problem:

  • Cannot add new fields dynamically.
  • Cannot remove fields dynamically.

With FieldArray

Add Skill
↓
FieldArray Adds New Input
↓
Remove Skill
↓
FieldArray Removes Input

Basic FieldArray Example

File: src/components/SkillsForm.jsx
import { Formik, Form, Field, FieldArray } from 'formik';

function SkillsForm() {
  return (
    <Formik
      initialValues={{ skills: [''] }}
      onSubmit={values => console.log(values)}
    >
      {() => (
        <Form>

          <FieldArray name="skills">
            {({ push, remove, form }) => (
              <>

                {form.values.skills.map((skill, index) => (
                  <div key={index}>
                    <Field name={`skills.${index}`} placeholder="Skill" />

                    <button type="button" onClick={() => remove(index)}>
                      Remove
                    </button>
                  </div>
                ))}

                <button type="button" onClick={() => push('')}>
                  Add Skill
                </button>

              </>
            )}
          </FieldArray>

          <button type="submit">
            Save
          </button>

        </Form>
      )}
    </Formik>
  );
}

export default SkillsForm;

How FieldArray Works

skills = ['React']
↓
push('')
↓
skills = ['React', '']
↓
New Input Appears

What Does push() Do?

Adds a new item to the array. 

push('');

Before: 

['React']

After: 

['React', '']

What Does remove() Do?

Removes an item from the array. 

remove(1);

Before: 

['React', 'Node.js']

After: 

['React']

FieldArray Helpers

Method Purpose
push() Add item
remove() Remove item
insert() Insert item at position
replace() Replace item
swap() Swap items
move() Move item

import { Formik, Form, Field, FieldArray } from 'formik';

function PhoneForm() {
  return (
    <Formik
      initialValues={{ phones: [''] }}
      onSubmit={values => console.log(values)}
    >
      {() => (
        <Form>
          <FieldArray name="phones">
            {({ push, remove, form }) => (
              <>

                {form.values.phones.map((phone, index) => (
                  <div key={index}>
                    <Field name={`phones.${index}`} placeholder="Phone Number" />
                    <button type="button" onClick={() => remove(index)}>Remove</button>
                  </div>
                ))}

                <button type="button" onClick={() => push('')}>
                  Add Phone
                </button>

              </>
            )}
          </FieldArray>

          <button type="submit">Save</button>
        </Form>
      )}
    </Formik>
  );
}

export default PhoneForm;

Interview Answer

FieldArray is a Formik component used for managing arrays of dynamic form fields. It provides helper methods such as push(), remove(), insert(), and swap() to add, remove, or modify items in an array. It is commonly used for skills, phone numbers, addresses, education records, and work experience forms.

One-Line Interview Answer

FieldArray is a Formik component used to create and manage dynamic arrays of form fields.
