286. How do you use React Hook Form?

One-Line Definition

React Hook Form is used by calling useForm(), registering inputs with register(), and handling form submission with handleSubmit().

Step-by-Step Process

Install React Hook Form
↓
Call useForm()
↓
Register Inputs
↓
Add Validation
↓
Handle Submit
↓
Get Form Data

Step 1 - Install React Hook Form

npm install react-hook-form

Project Structure

src/
│
├── App.jsx
│
└── components/
    └── LoginForm.jsx

Step 2 - Create Form

File: src/components/LoginForm.jsx
import { useForm } from 'react-hook-form';

function LoginForm() {
  const { register, handleSubmit } = useForm();

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input placeholder="Email" {...register('email')} />
      <input type="password" placeholder="Password" {...register('password')} />
      <button type="submit">Login</button>
    </form>
  );
}

export default LoginForm;

What Happens Here?

Code Purpose
useForm() Creates form controller.
register() Registers inputs.
handleSubmit() Handles form submission.
onSubmit() Receives form data.

Step 3 - Add Validation

File: src/components/LoginForm.jsx
import { useForm } from 'react-hook-form';

function LoginForm() {
  const { register, handleSubmit, formState: { errors } } = useForm();

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>

      <input placeholder="Email" {...register('email', { required: 'Email is required' })} />
      {errors.email && <p>{errors.email.message}</p>}

      <input
        type="password"
        placeholder="Password"
        {...register('password', {
          required: 'Password is required',
          minLength: {
            value: 6,
            message: 'Minimum 6 characters'
          }
        })}
      />

      {errors.password && <p>{errors.password.message}</p>}

      <button type="submit">Login</button>

    </form>
  );
}

export default LoginForm;

Submitted Data

If the user enters: 

Email: rizwan@gmail.com
Password: 123456

React Hook Form returns: 

{
  email: 'rizwan@gmail.com',
  password: '123456'
}

Step 4 - Reset Form

File: src/components/LoginForm.jsx
const { register, handleSubmit, reset } = useForm();

function onSubmit(data) {
  console.log(data);
  reset();
}

After successful submission all fields become empty.

Step 5 - Watch Field Values

File: src/components/LoginForm.jsx
const { register, watch } = useForm();

const email = watch('email');

console.log(email);

watch() tracks field changes in real time.

Step 6 - Set Value Programmatically

File: src/components/LoginForm.jsx
const { register, setValue } = useForm();

setValue('email', 'rizwan@gmail.com');

Useful when loading data from APIs.

Most Common Methods

Method Purpose
register() Register input.
handleSubmit() Submit form.
watch() Watch values.
reset() Reset form.
setValue() Update field value.
getValues() Get form values.

Complete Real-World Example

File: src/components/RegisterForm.jsx
import { useForm } from 'react-hook-form';

function RegisterForm() {
  const { register, handleSubmit, formState: { errors } } = useForm();

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>

      <input placeholder="Name" {...register('name', { required: 'Name is required' })} />
      {errors.name && <p>{errors.name.message}</p>}

      <input placeholder="Email" {...register('email', { required: 'Email is required' })} />
      {errors.email && <p>{errors.email.message}</p>}

      <input type="password" placeholder="Password" {...register('password', { required: 'Password is required' })} />
      {errors.password && <p>{errors.password.message}</p>}

      <button type="submit">Register</button>

    </form>
  );
}

export default RegisterForm;

Why Developers Prefer React Hook Form

  • Less code
  • Built-in validation
  • Better performance
  • Fewer re-renders
  • Easy integration with APIs
  • Works well with Yup and Zod
React Hook Form is currently the most popular form library in React applications.

Interview Answer

To use React Hook Form, first call useForm(), then register inputs using register(), handle form submission with handleSubmit(), and optionally add validation rules. React Hook Form automatically collects values, validates fields, and provides the submitted data object.
