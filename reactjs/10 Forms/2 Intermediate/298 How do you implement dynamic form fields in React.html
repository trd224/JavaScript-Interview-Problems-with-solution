298. How do you implement dynamic form fields in React?

One-Line Definition

Dynamic form fields are implemented by storing fields in an array state and adding, updating, or removing items from that array.

Why Do We Need Dynamic Fields?

  • Multiple phone numbers
  • Multiple skills
  • Multiple addresses
  • Education records
  • Work experience entries

Complete Example

File: src/components/SkillsForm.jsx
import { useState } from 'react';

function SkillsForm() {
  const [skills, setSkills] = useState(['']);

  function addSkill() {
    setSkills([...skills, '']);
  }

  function updateSkill(index, value) {
    const updatedSkills = [...skills];
    updatedSkills[index] = value;
    setSkills(updatedSkills);
  }

  function removeSkill(index) {
    setSkills(skills.filter((skill, i) => i !== index));
  }

  function handleSubmit(e) {
    e.preventDefault();
    console.log(skills);
  }

  return (
    <form onSubmit={handleSubmit}>
      {skills.map((skill, index) => (
        <div key={index}>
          <input value={skill} onChange={e => updateSkill(index, e.target.value)} placeholder="Skill" />
          <button type="button" onClick={() => removeSkill(index)}>Remove</button>
        </div>
      ))}

      <button type="button" onClick={addSkill}>Add Skill</button>
      <button type="submit">Submit</button>
    </form>
  );
}

export default SkillsForm;

Dynamic Object Fields Example

File: src/components/EducationForm.jsx
import { useState } from 'react';

function EducationForm() {
  const [education, setEducation] = useState([{ degree: '', college: '' }]);

  function addEducation() {
    setEducation([...education, { degree: '', college: '' }]);
  }

  function updateEducation(index, field, value) {
    const updatedEducation = [...education];
    updatedEducation[index][field] = value;
    setEducation(updatedEducation);
  }

  function removeEducation(index) {
    setEducation(education.filter((item, i) => i !== index));
  }

  return (
    <>
      {education.map((item, index) => (
        <div key={index}>
          <input value={item.degree} onChange={e => updateEducation(index, 'degree', e.target.value)} placeholder="Degree" />
          <input value={item.college} onChange={e => updateEducation(index, 'college', e.target.value)} placeholder="College" />
          <button type="button" onClick={() => removeEducation(index)}>Remove</button>
        </div>
      ))}

      <button type="button" onClick={addEducation}>Add Education</button>
    </>
  );
}

export default EducationForm;

Important Point

Never directly mutate state like skills[index] = value without creating a new array copy first.

Correct State Update

const updatedSkills = [...skills];
updatedSkills[index] = value;
setSkills(updatedSkills);

Dynamic Fields With Formik

In Formik, dynamic fields are usually handled with FieldArray. 

<FieldArray name="skills">
  {({ push, remove }) => (
    <>
      <button type="button" onClick={() => push('')}>Add</button>
      <button type="button" onClick={() => remove(0)}>Remove</button>
    </>
  )}
</FieldArray>

Dynamic Fields With React Hook Form

In React Hook Form, dynamic fields are usually handled with useFieldArray(). 

const { fields, append, remove } = useFieldArray({
  control,
  name: 'skills'
});

Interview Answer

Dynamic form fields in React are implemented by storing field values in an array state. We render fields using map(), add new fields by pushing new values into the array, update fields by index, and remove fields using filter(). Libraries like Formik provide FieldArray and React Hook Form provides useFieldArray for this purpose.

One-Line Interview Answer

Dynamic form fields are created by storing fields in an array and rendering them with map(), then adding or removing items from that array.
