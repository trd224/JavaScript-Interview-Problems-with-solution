288. What is the watch Function in React Hook Form?

One-Line Definition

watch() is a React Hook Form function used to read and monitor field values in real time as the user types.

Why Do We Need watch()?

Sometimes we need the current value of a field before form submission.

Examples:

  • Show live preview
  • Enable/disable fields
  • Show password strength
  • Confirm password validation
  • Conditional form fields

How watch() Works

User Types
↓
watch()
↓
Gets Current Value
↓
UI Updates

Basic Example

File: src/components/LoginForm.jsx
import { useForm } from 'react-hook-form';

function LoginForm() {
  const { register, watch } = useForm();

  const email = watch('email');

  return (
    <>
      <input placeholder="Email" {...register('email')} />
      <p>Current Value: {email}</p>
    </>
  );
}

export default LoginForm;

As the user types, the paragraph updates automatically.

Watch Multiple Fields

File: src/components/RegisterForm.jsx
import { useForm } from 'react-hook-form';

function RegisterForm() {
  const { register, watch } = useForm();

  const name = watch('name');
  const email = watch('email');

  return (
    <>
      <input placeholder="Name" {...register('name')} />
      <input placeholder="Email" {...register('email')} />

      <p>Name: {name}</p>
      <p>Email: {email}</p>
    </>
  );
}

export default RegisterForm;

Real World Example - Password Confirmation

File: src/components/RegisterForm.jsx
import { useForm } from 'react-hook-form';

function RegisterForm() {
  const { register, watch } = useForm();

  const password = watch('password');

  return (
    <>
      <input type="password" placeholder="Password" {...register('password')} />
      <p>Length: {password?.length || 0}</p>
    </>
  );
}

export default RegisterForm;

Real World Example - Live Preview

File: src/components/ProfileForm.jsx
import { useForm } from 'react-hook-form';

function ProfileForm() {
  const { register, watch } = useForm();

  return (
    <>
      <input placeholder="Name" {...register('name')} />

      <p>{name}</p>
    </>
  );
}

export default ProfileForm;

watch() getValues()

Interview Answer

The watch() function is used to monitor form field values in real time. It allows React Hook Form to provide the current value of one or more fields and is commonly used for live previews, conditional rendering, and dynamic form behavior.

One-Line Interview Answer

watch() is used to observe and retrieve form field values in real time as the user types.
