290.1 How implement dynamic form field using React Hook Form?

One-Line Definition

React Hook Form provides the useFieldArray() hook to create dynamic fields that can be added or removed at runtime.

When Do We Need Dynamic Fields?

  • Multiple Skills
  • Multiple Phone Numbers
  • Multiple Addresses
  • Education Records
  • Work Experience

Flow

↓
useFieldArray()
↓

↓

Complete Example

File: src/components/SkillsForm.jsx
import { useForm, useFieldArray } from 'react-hook-form';

function SkillsForm() {
  const { register, control, handleSubmit } = useForm({
    defaultValues: {
      skills: [{ name: '' }]
    }
  });

  const { fields, append, remove } = useFieldArray({
    control,
    name: 'skills'
  });

  function onSubmit(data) {
    console.log(data);
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>

      {fields.map((field, index) => (
        <div key={field.id}>
          <input placeholder="Skill" {...register(`skills.${index}.name`)} />

          <button type="button" onClick={() => remove(index)}>
            Remove
          </button>
        </div>
      ))}

      <button type="button" onClick={() => append({ name: '' })}>
        Add Skill
      </button>

      <button type="submit">
        Submit
      </button>

    </form>
  );
}

export default SkillsForm;

Understanding useFieldArray()

const { fields, append, remove } = useFieldArray({
  control,
  name: 'skills'
});
Property Purpose
fields Current array items
append() Add new item
remove() Delete item

What Happens When Add Skill Is Clicked?

append({ name: '' });
skills = [ { name: 'React' } ]
↓
append({ name: '' })
↓
skills = [ { name: 'React' }, { name: '' } ]

What Happens When Remove Is Clicked?

remove(index);
[ { name: 'React' }, { name: 'Node.js' } ]
↓
remove(1)
↓
[ { name: 'React' } ]

Submitted Data

{
  skills: [
    { name: 'React' },
    { name: 'Node.js' },
    { name: 'Next.js' }
  ]
}

React Hook Form uses useFieldArray() for dynamic fields, while Formik uses FieldArray.

Interview Answer

Dynamic form fields in React Hook Form are implemented using the useFieldArray() hook. It provides fields, append(), remove(), and other helper methods to dynamically add or remove form fields while keeping them connected to React Hook Form state.

One-Line Interview Answer

Use useFieldArray() to create dynamic add/remove form fields in React Hook Form.
