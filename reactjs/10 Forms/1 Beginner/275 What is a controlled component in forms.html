275. What is a Controlled Component in Forms?

One-Line Definition

A controlled component in forms is a form element whose value is controlled and managed by React state.

What Does "Controlled" Mean?

In a controlled form field, React becomes the single source of truth.

Instead of the browser DOM managing the value, React state manages it.

Basic Controlled Input

import { useState } from 'react';

function Form() {

  const [name, setName] =
  useState('');

  return (

    <input
      value={name}
      onChange={e => setName(e.target.value)}
    />

  );

}

Why Is It Called Controlled?

Because React controls the value shown in the input. 

<input
  value={name}
  onChange={e => setName(e.target.value)}
/>
Property Purpose
value Displays state value inside the input
onChange Updates state when user types

Complete Form Example

import { useState } from 'react';

function RegistrationForm() {

  const [name, setName] =
  useState('');

  return (

    <div>

      <input
        value={name}
        onChange={e => setName(e.target.value)}
        placeholder="Enter Name"
      />

      <p>

        Name: {name}

      </p>

    </div>

  );

}

Controlled Select Example

const [city, setCity] =
useState('Jaipur');

<select
  value={city}
  onChange={e => setCity(e.target.value)}
>

  <option>Jaipur</option>

  <option>Delhi</option>

</select>

Why Are Controlled Components Popular?

  • Easy validation.
  • Easy error handling.
  • Easy form submission.
  • React always knows the current value.
  • Supports dynamic UI updates.

Validation Example

const [email, setEmail] =
useState('');

const isValid =
email.includes('@');
Validation is easy because the current value already exists in React state.

Real Project Example

In our Aadhaar Authentication application, Aadhaar Number, OTP, Department, Purpose, and Consent fields are controlled components because React must validate, mask, and process their values before submitting the request.

Important Interview Points

  • Controlled components use React state.
  • value and onChange are required.
  • React becomes the single source of truth.
  • Typing updates state and triggers re-render.
  • Controlled components are preferred for most React forms.

Short Interview Answer

A controlled component in forms is a form element whose value is managed by React state and updated through onChange handlers.
