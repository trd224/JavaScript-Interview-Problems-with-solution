272. What is a Controlled Component?

One-Line Definition

A controlled component is a form element whose value is controlled by React state using useState.

What Does "Controlled" Mean?

In a controlled component, React controls the value of the form field.

The input value comes from state, and every change updates that state.

Basic Controlled Input Example

import { useState } from 'react';

function UserForm() {

  const [name, setName] = useState('');

  return (

    <input
      value={name}
      onChange={e => setName(e.target.value)}
      />

  );

}

How It Works

const [name, setName] = useState('');

State stores the input value. 

<input value={name} />

The input displays the value stored in state. 

onChange={e => setName(e.target.value)}

Whenever the user types, state gets updated.

Complete Example

import { useState } from 'react';

function UserForm() {

  const [name, setName] = useState('');

  return (

    <div>

      <input
        value={name}
        onChange={e => setName(e.target.value)}
        placeholder="Enter name"
      />

      <p>

        Name: {name}

      </p>

    </div>

  );

}

Output Flow

User Types "R"
↓
State = "R"
↓
Input Shows "R"

User Types "Ri"

State = "Ri"
↓
Input Shows "Ri"

Controlled Textarea

const [message, setMessage] = useState('');

<textarea
  value={message}

></textarea>

Controlled Select

const [city, setCity] = useState('Jaipur');

<select
  value={city}
  onChange={e => setCity(e.target.value)}
>

  <option>Jaipur</option>

  <option>Delhi</option>

</select>

Why Use Controlled Components?

  • Easy form validation.
  • Easy error handling.
  • Easy form submission.
  • React always knows the current value.
  • Values can be modified programmatically.

Example Validation

const [email, setEmail] = useState('');

if (!email.includes('@')) {

  console.log('Invalid Email');

}
Validation becomes easy because the value is already available in React state.

Controlled vs Uncontrolled Component

Controlled Component Uncontrolled Component

Uses useState Uses useRef
React controls input DOM controls input
Easy validation Validation requires reading DOM value

Real Project Example

In our Aadhaar Authentication application, Aadhaar Number, OTP, Department, Purpose, and Consent fields are typically implemented as controlled components so React can validate, mask, and process the values before submission.

Important Interview Points

  • Controlled components store values in React state.
  • They usually use useState.
  • value and onChange are required.
  • React becomes the single source of truth.
  • Controlled components are commonly used in forms.

Short Interview Answer

A controlled component is a form element whose value is managed by React state and updated through event handlers such as onChange.

One-Line Interview Answer

A controlled component is a form field whose value is controlled by React state.
