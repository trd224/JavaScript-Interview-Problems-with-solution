281. How Do You Handle a Checkbox in React?

One-Line Definition

A checkbox in React is typically handled using useState and the checked property instead of the value property.

Why Is a Checkbox Different?

Text inputs use value, but checkboxes use checked.

Input Type Property
Text Input value
Checkbox checked

Checkbox Handling Flow

User Clicks Checkbox
↓
onChange Event Fires
↓
e.target.checked
↓
setState()
↓
React State Updated
↓
Checkbox Updated

Basic Example

import { useState } from 'react';

function TermsForm() {

  const [accepted, setAccepted] = useState(false);

  return (

    <input
      type="checkbox"
      checked={accepted}
      onChange={e => setAccepted(e.target.checked)}
    />

  );

}

How It Works

const [accepted, setAccepted] = useState(false);

State stores whether the checkbox is checked or unchecked. 

<input
  type="checkbox"
  checked={accepted}
  onChange={e => setAccepted(e.target.checked)}
/>

e.target.checked returns true or false.

Displaying Checkbox State

import { useState } from 'react';

function TermsForm() {

  const [accepted, setAccepted] = useState(false);

  return (

    <div>

      <input type="checkbox" checked={accepted} onChange={e => setAccepted(e.target.checked)} />

      <p>

        Accepted: {accepted ? 'Yes' : 'No'}

      </p>

    </div>

  );

}

Using a Separate Handler Function

function handleChange(e) {

  setAccepted(e.target.checked);

}

<input type="checkbox" checked={accepted} onChange={handleChange} />

Checkbox Inside Form Data Object

const [formData, setFormData] = useState({
  accepted: false
});

function handleChange(e) {

  setFormData({
    ...formData,
    accepted: e.target.checked
  });

}

Complete Example

import { useState } from 'react';

function TermsForm() {

  const [accepted, setAccepted] = useState(false);

  function handleSubmit(e) {

    e.preventDefault();

    console.log(accepted);

  }

  return (

    <form onSubmit={handleSubmit}>

      <label>

        <input type="checkbox" checked={accepted} onChange={e => setAccepted(e.target.checked)} />

      </label>

      <button type="submit">

        Submit

      </button>

    </form>

  );

}

export default TermsForm;

Why Use checked Instead of value?

A checkbox does not store text. It stores only two states: checked (true) or unchecked (false). Therefore, we use checked instead of value.

In our Aadhaar Authentication application, the user consent checkbox is handled using React state. The authentication process is allowed only when the consent checkbox is checked.

Important Interview Points

  • Checkboxes use checked instead of value.
  • e.target.checked returns a boolean value.
  • Checkboxes are usually controlled components.
  • useState is commonly used to manage checkbox state.
  • Checkbox values are typically true or false.

Short Interview Answer

A checkbox in React is handled using state and the checked property, where e.target.checked provides the current checked status.

One-Line Interview Answer

React handles checkboxes using the checked property and updates state with e.target.checked.
