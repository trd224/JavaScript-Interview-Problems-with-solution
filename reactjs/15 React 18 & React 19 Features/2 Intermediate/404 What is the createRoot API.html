404. What is the createRoot API?

One-Line Definition

createRoot() is the new React 18 API used to render a React application. It enables React 18 features such as Concurrent Rendering, Automatic Batching, Transitions, and other performance improvements.

Why Was createRoot Introduced?

Before React 18, applications were rendered using ReactDOM.render().

React 17
↓
ReactDOM.render()
↓
Legacy Rendering Engine

React 18 introduced a new rendering engine.

React 18
↓
createRoot()
↓
Concurrent Rendering Engine
Without createRoot(), React 18 features will not work.

React 17 (Old Way)

File: src/index.js
import ReactDOM from 'react-dom';
import App from './App';

ReactDOM.render(
  <App />,
  document.getElementById('root')
);

React 18 (New Way)

File: src/main.jsx
import ReactDOM from 'react-dom/client';
import App from './App';

const root = ReactDOM.createRoot(
  document.getElementById('root')
);

root.render(<App />);

Understanding Step By Step

Step 1

const root = ReactDOM.createRoot(
  document.getElementById('root')
);

Creates a React root container.

Step 2

root.render(<App />);

Mounts the React application into that root.

Where Does root Come From?

  <div id="root"></div>
</body>

Why Not Use ReactDOM.render()?

ReactDOM.render() still works in React 18 but runs the app in legacy mode.

You lose React 18 concurrent features when using ReactDOM.render().

Features Enabled By createRoot()

Feature Available With createRoot?
Automatic Batching ✅ Yes
Concurrent Rendering ✅ Yes

ReactDOM.render() ❌ Legacy Mode

Real World Example

File: src/main.jsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import './index.css';

const root = ReactDOM.createRoot(
  document.getElementById('root')
);

root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

Common Interview Confusion

createRoot() does NOT render the app.

createRoot() only creates the root container.

root.render() actually renders the application.

Interview Answer

createRoot() is the new React 18 API used to create a React root container. It replaces ReactDOM.render() and enables React 18 features such as Concurrent Rendering, Automatic Batching, Transitions, and improved performance. After creating the root, the application is rendered using root.render().

One-Line Interview Answer

createRoot() is the React 18 API that replaces ReactDOM.render() and enables React 18's concurrent features.
