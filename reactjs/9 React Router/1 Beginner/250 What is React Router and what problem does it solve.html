250. What is React Router and What Problem Does It Solve?

One-Line Definition

React Router is a library that enables navigation between different pages (routes) in a React Single Page Application (SPA) without reloading the browser.

What Problem Does It Solve?

A React application normally contains only a single HTML page.

Without React Router, switching between pages such as Home, About, Contact, Dashboard, and Profile becomes difficult because React does not provide routing by default.

React Router solves this problem by mapping URLs to React components.

import { useState } from 'react';

function App() {

  const [page, setPage] = useState('home');

  return (

    <>

      <button onClick={() => setPage('home')}>Home</button>

      <button onClick={() => setPage('about')}>About</button>

      {page === 'home' && <Home />}

      {page === 'about' && <About />}

    </>

  );

}

Problems with This Approach

  • No real URLs.
  • Users cannot bookmark pages.
  • Browser back/forward buttons do not work properly.
  • Application becomes difficult to maintain.

Installation

npm install react-router-dom

Basic Routing Setup

import { BrowserRouter, Routes, Route } from 'react-router-dom';

function App() {

  return (

    <BrowserRouter>

      <Routes>

        <Route path="/" element={<Home />} />

        <Route path="/about" element={<About />} />

        <Route path="/contact" element={<Contact />} />

      </Routes>

    </BrowserRouter>

  );

}

Navigation Using Link

import { Link } from 'react-router-dom';

function Navbar() {

  return (

    <nav>

      <Link to="/">Home</Link>

      <Link to="/about">About</Link>

    </nav>

  );

}

Important Interview Points

  • React Router enables client-side routing.
  • It prevents full page reloads.
  • It maps URLs to React components.
  • Link is preferred over anchor tags.
  • BrowserRouter is the most commonly used router.
  • React Router is provided by react-router-dom.

Short Interview Answer

React Router is a routing library that allows React applications to display different components based on the URL while avoiding full page reloads.

One-Line Interview Answer

React Router enables navigation between pages in a React Single Page Application without reloading the browser.
